Título: Q9 - Fora, repetidos! v2
Topicos do problema: array; repetição; 
Tempo limite de execução: 1 segundo(s)
Descrição: <p>À essa altura, você já deve ter feito a &#34;Q4 - Fora, repetidos!&#34;, certo? Senão, tudo bem! </p><p><br/></p><p>Sua missão é receber n números que serão alocados para uma primeira lista; n2 números alocados para a segunda lista e, por fim, mostrar:</p><p><br/></p><p>1 - Uma lista apenas com os números que se repetiram, sem repetidos (por exemplo, se 6 se repetiu 3 vezes, essa lista de repetidos terá apenas um 6.)</p><p>2 - Uma lista com todos os números sem números repetidos</p><p>3 - Uma lista com todos os números, excetos os números repetidos que estão na lista de repetidos. (Por exemplo, se há 6 na lista com todos os números e há 6 na lista de repetidos, essa lista não deve conter o número 6.)</p>
Formato da entrada: <p>Primeiramente, um número n que representa a quantidade de elementos da primeira lista.</p><p>Segundamente, serão lidos n números que serão alocados para a primeira lista.</p><p>Terceiramente, n2 números que representa a quantidade de elementos da segunda lista.</p><p>Quartamente, serão lidos n2 números que serão alocados para a segunda lista.</p>
Formato da saída: <p>3 LIstas:</p><p>Uma apenas com os números que se repetiram</p><p>Uma com todos os números, incluindo os repetidos, mas sem repetí-los</p><p>Uma com todos os números, exceto os que estão na lista de repetidos.</p>
