Título: Viagem no tempo
Topicos do problema: estrutura de dados; 
Tempo limite de execução: 1 segundo(s)
Descrição: <p>O ano é 2047 e com o advento da tecnologia do reino quântico, a viagem no tempo nunca foi tão possível. Porém, para que isso aconteça é necessário que um cientista com uma inteligência inigualável, como você, monitore a máquina. Com as especificações atuais, é possível viajar para 3 épocas diferentes: </p><ul><li>Egito Antigo (-3100 até -30) </li><li>Japão Feudal (794 até 1185) </li><li>Brasil Futurista (2235 ad infinitum)</li></ul><span>Além dessas épocas o usuário pode ir para o ano de 2019, indo para o Presente. Você deve monitorar a máquina de acordo com os pedidos dos usuários, e após todos os pedidos dizer quantos já voltaram e quantos ainda estão nas outras épocas. Inicialmente todos estão no Presente.</span><div><br/></div><div style="text-align: center;"><img src="https://thumbs.gfycat.com/GiantPreciousGrison-small.gif"/></div><div><div><span><br/></span></div><div><span><br/></span></div><div><span>OBS:</span></div><div><ul><li><span>Anos negativos significam anos antes de Cristo</span></li><li><span>Não há nomes repetidos e não há viagens entre a mesma época (por exemplo, não é possível ir de 895 para 1067 sem passar pelo presente antes)</span></li><li>A questão deve ser feita com <b>listas encadeadas</b></li></ul></div></div>
Formato da entrada: <p>Múltiplas linhas no formato:</p><blockquote><p><b>S X</b></p></blockquote><p><b>S</b> uma string, representando o nome do usuário</p><p><b>X </b>um número inteiro, representando o ano que o usuário deseja ir</p>
Formato da saída: <p>Para cada linha N correta - que o ano esteja catalogado - deve ser exibido: </p><blockquote><p>&#34;<b>nomeDoUsuario </b>foi de <b>E1 </b>para <b>E2</b>.&#34;</p><p><span id="selectionBoundary_1557114646046_8374450888922655" class="rangySelectionBoundary">&#65279;</span></p></blockquote><p>Sendo:</p><ul><li>nomeDoUsuario uma string representando o nome do viajante do tempo</li><li>E1 uma string representando a época em que ele se encontrava</li><li>E2 uma string representando a época para que o usuário viajou</li></ul><p>Senão deve ser exibido:</p><blockquote><p>&#34;Cuidado pra nao ficar preso no reino quantico, NomeDoUsuario.&#34;</p></blockquote><p>nomeDoUsuario uma string representando o nome do viajante do tempo<br/></p><p><br/></p><p>Após todas as viagens, deve-se pular uma linha e exibir onde estão todas as pessoas que viajaram no tempo, sendo necessário apenas exibir a época se houver alguém nela, no seguinte formato:</p><blockquote><p>EpócaX:</p><p>Nome1</p><p>Nome2</p><p>...</p><p>NomeN</p></blockquote><p>Sendo <b>ÉpocaX </b>uma string correspondendo a época em que os usuários a serem listados estão.</p><p><b>NomeX </b>uma string correpondendo a cada nome de usuário presente naquela época</p><p>Obs: Se o usuário não realizou nenhuma viagem, não deve ser listado.</p>
