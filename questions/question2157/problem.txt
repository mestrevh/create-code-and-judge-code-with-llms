Título: Determinante sem cálculo direto
Topicos do problema: decisão; repetição; lógica matemática; array bidimensional; 
Descrição: <p class="inserted">Podemos verificar se o determinante de uma matriz é zero sob algumas condições sem ter que, necessariamente, calcular esse determinante. Dessa forma, faça um programa que verifica as seguintes propriedades:</p><ul class="inserted"><li class="inserted">Ao observar uma matriz e verificar que os elementos de uma linha ou uma coluna são todos iguais a zero, o valor do seu determinante também será zero.</li></ul><img src="https://www.thehuxley.com/api/v1/problems/image/abb47a2a602712385f49b4c41944a7eff682b226o.png"/><br/><ul class="inserted"><li class="inserted">Caso ocorra igualdade de elementos entre duas linhas ou duas colunas, o determinante dessa matriz será nulo.<br/></li></ul><p class="inserted"><img src="https://www.thehuxley.com/api/v1/problems/image/b830783952b84eb7479c8b0083c3b97c2d81bb220.png"/><br/></p><p class="inserted">Seu programa deve também avaliar se a matriz recebida é quadrada, caso contrário deve emitir uma mensagem de erro:</p><p class="inserted">&#34;O determinante de uma matriz só existe para matrizes quadradas.&#34;</p><p class="inserted">Dica: Faça uma função para calcular se duas linhas são iguais. Formato da entrada: <p class="inserted">Uma matriz recebida uma linha por vez.</p>
Formato da saída: <p class="inserted">Uma mensagem que indica se o resultado do determinante é zero ou não.<br/></p><p class="inserted">Ou uma mensagem de erro se a matriz não for quadrada.</p><p class="inserted">O programa deve receber novas linhas da matriz até que receba a palavra &#34;acabou&#34;.</p>
