Título: Função REMOVE - sua versão
Topicos do problema: array; decisão; repetição; 
Descrição: <p style="color: rgb(0, 0, 0);">Quando se usa estrutura de dados homogêneas ( também conhecidas como array, vetores, arranjos) algumas manipulações são necessárias. Dentre elas está a necessidade de remover um elemento da estrutura.</p><p style="color: rgb(0, 0, 0);">Em Python existe uma função denominada REMOVE cujo objetivo é remover <span style="text-align: justify;">o primeiro item encontrado na lista que corresponde ao valor informado pelo usuário. </span><span style="text-align: justify;">Se não existir valor igual, uma exceção </span><span style="text-align: justify;">é apresentada. </span><br/><span><br/></span></p><p style="color: rgb(0, 0, 0);"><span>O objetivo dessa atividade é implementar uma versão adaptada da função REMOVE.</span><br/></p><p style="color: rgb(0, 0, 0);"><span>A sua solução deve receber um número (N) que corresponde ao número de elementos da sua lista e, em seguida, serão lidos N números inteiros.</span></p><p style="color: rgb(0, 0, 0);">Após isso, deve ser lida um valor corresponde ao que você deseja procurar na lista e que deve ser removido caso exista na lista. </p><p style="color: rgb(0, 0, 0);">Na saída devem ser apresentados, na seguinte ordem: a lista criada, o item procurado, a lista sem o item procurado.</p><p style="color: rgb(0, 0, 0);">ATENÇÃO: A sua solução NÃODEVE conter nenhuma função específica de Python para remoção de elementos em listas. Ou seja, <span style="color: rgb(147, 149, 152);">apenas as<b><span class="Apple-converted-space"> </span>funções básicas (append e </b></span><b style="color: rgb(147, 149, 152);">len)</b><span style="color: rgb(147, 149, 152);background-color: rgb(255, 255, 255);float: none;"><span class="Apple-converted-space"> </span>podem ser utilizadas na sua solução.</span></p>
Formato da entrada: <p style="color: rgb(0, 0, 0);">Os dados da entrada são:</p><ul style="color: rgb(0, 0, 0);"><li>número de elementos que se deseja colocar na lista (N)</li><li>N números inteiros (elementos da lista)</li><li>O valor do elemento que se deseja remover da lista </li></ul>
Formato da saída: <p style="color: rgb(0, 0, 0);">A saída deve ter:</p><ul style="color: rgb(0, 0, 0);"><li>A lista com seus elementos </li><li>O item procurado e a ser removido</li><li>A nova lista sem o item procurado</li></ul><p style="color: rgb(0, 0, 0);">Caso não seja possível remover elemento da lista porque a lista está vazia, apresentar a mensagem: &#34;A lista estah vazia - nao eh possivel remover elemento&#34;.</p><p style="color: rgb(0, 0, 0);">Em outro caso, apresentar a mensagem: &#34;Nao eh possivel remover o elemento&#34;.</p>
