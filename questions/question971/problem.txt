Título: Um problema de logística
Topicos do problema: grafo; 
Descrição: <p>Uma empresa de logística possui vários armazéns espalhados por diversos pontos do mapa. Os produtos estão em constante circulação entre esses pontos. Para aumentar o controle e facilitar a gestão da circulação de mercadorias, a empresa decide restringir o transporte a um subconjunto fixo de linhas (bidirecionais) de conexão entre os pontos de distribuição. Naturalmente, as linhas devem ser definidas de forma que seja possível estabelecer uma rota entre quaisquer dois armazéns. Mais ainda, a empresa deseja escolher um conjunto de conexões de forma a minimizar os custos operacionais. O custo de operação de uma linha é estimado como sendo a soma das diferenças absolutas entre as suas respectivas latitudes e longitudes. Mais formalmente, se um ponto A tem coordenadas (Xa, Ya) e um ponto B tem coordenadas (Xb, Yb), então o custo de operar uma linha entre A e B será dado por |Xa - Xb| + |Ya - Yb|.<br/><br/></p>
Formato da entrada: <p>A entrada é constituída de vários casos, cada um correspondente a um cenário/mapa como descrito acima. A primeira linha contém um inteiro <br/><br/>L <br/><br/>representando o número de casos. A seguir, temos a especificação dos diversos casos.<br/>  <br/>Para cada mapa, temos inicialmente uma linha com um inteiro, <br/><br/>N<br/><br/>que indica a quantidade de armazéns.<br/>  <br/>As próximas N linhas contêm um par de inteiros cada<br/><br/>X[i] Y[i]<br/><br/>onde (X[i], Y[i]) é a coordenada do i-ésimo armazém.<br/><br/>Há uma linha em branco após cada caso.<br/><br/>Limites: <br/>1 &lt;= N &lt;= 1500<br/>0 &lt;= X[i], Y[i] &lt;= 10000;<br/></p>
Formato da saída: <p>Para cada caso de teste k = 0,..., L-1, deve ser impressa uma linha na forma <br/><br/>k: C[k]<br/><br/>onde C[k] corresponde ao menor custo possível para interligar os armazéns.<br/></p>
