Título: Os arrays de Kinho
Topicos do problema: array; 
Descrição: <p><span>Kinho recebeu uma importante tarefa que envolve arrays, mas como ele anda muito ocupado votando no paredão da Karol Conká, cabe a você, jovem discípulo de ACM assumir a responsabilidade.  </span><br/></p><p>Dado um array {a(1), a(2), a(3), ..., a(n)} e um número <b>d</b>, você precisará escolher três índices i, j e k (i != j, i != k e j != k) e atribuir a soma de a(j) e a(k) para a(i), fazendo a(i) = a(j) + a(k) de forma a tornar todos os elementos do array menores ou iguais a <b>d.</b> Pode fazer essa operação um número indeterminado de vezes, você consegue? </p>
Formato da entrada: <p>A primeira linha contém um inteiro <b>t</b> (1 &lt;= t &lt;= 2000), o número de casos de teste</p><p>A primeira linha de cada caso teste contém dois inteiros <b>n </b> e <b>d</b>, (3 &lt;= n &lt;= 100; 1 &lt;= d &lt;= 100), o número de elementos do array e o valor <b>d</b>.</p><p>A segunda linha de cada caso teste contém <b>n</b> inteiros <span style="color: rgb(147, 149, 152);background-color: rgb(255, 255, 255);float: none;">{a(1), a(2), a(3), ..., a(n)}, (1 &lt;= a(i) &lt;= 100).</span></p>
Formato da saída: <p>Para cada caso teste, você deverá printar SIM caso seja possível tornar todos os elementos do array menores ou iguais a <b>d</b>, e NAO caso contrário :) </p>
