Título: L4Q3 - Campeonato de batalha naval 3000
Topicos do problema: array bidimensional; 
Tempo limite de execução: 1 segundo(s)
Descrição: <p>Depois de conseguir recriar o jogo Batalha Naval 3000, você resolveu chamar seus colegas alunos de IP para uma competição. Porém, preocupados com o tempo gasto para definir os três primeiros lugares, já que depois do campeonato vocês iriam estudar para a miniprova de AVLC, seus colegas definiram uma condição para participar, que você criasse um programa que definiria os 3 primeiros colocados dessa competição.</p><p>Você, como uma pessoa competitiva, aceitou o desafio e além de criar esse programa, resolveu definir uma regra para deixar as coisas mais divertidas. <b>Se o jogador de número X terminar na posição X no campeonato de número X</b>, esse jogador está automaticamente <b>eliminado</b> do campeonato, ou seja, <b>o jogador não deve ser considerado</b> para o cálculo dos 3 primeiros lugares. Além disso, como a cada partida os jogadores estão mais acostumados com o jogo, a cada competição a pontuação dada para cada jogador aumenta.</p><p><b>OBS1:</b> Se o jogador número 1 ganhar o primeiro torneio, ele não deve ser eliminado.</p><p><b>OBS2:</b> Sempre existirá um possível top 3 nos casos testes, ou seja, nunca a resposta terá menos de 3 jogadores.</p><p><b>OBS3: </b>Deve ser usado array bidimensional para a resolução da questão.</p><p><b>OBS4:</b> A fórmula para o cálculo da pontuação é a seguinte: <img src="https://www.thehuxley.com/api/v1/problems/image/6b1adfc7bb70016097127237cc2c66dd265152a8p.png"/></p><p>Sendo número da competição a quantidade de competições que já ocorreram antes + 1.</p><p><b>OBS5: </b>Os nomes dos jogadores são representados por números, em que o jogador 1 é o número 1, jogador 2 é o número 2 e assim sucessivamente, até o jogador n, que é o número n.</p>
Formato da entrada: <p>É dado um número inteiro <b>N</b>, que representa a quantidade de jogadores do campeonato.</p><p>Um outro número inteiro <b>M</b>, que representa quantos campeonatos houveram.</p><p>Uma matriz MxN:</p><ul><li>cada uma das M linhas representa um campeonato</li><li>as N colunas são usadas para registrar a posição de cada jogador no campeonato, ou seja, se o 3 aparece na 1a coluna é porque o primeiro jogador ficou em 3o colocado.</li><li>portanto, a posição I x J contém o jogador que, na partida I, ficou na J-ésima posição</li></ul>
Formato da saída: <p>Imprimir:</p><blockquote><p>&#34;O primeiro lugar foi: X&#34;</p></blockquote><p>Em que X representa o número do jogador que foi primeiro lugar.</p><blockquote><p>&#34;O segundo lugar foi: Y&#34;</p></blockquote><p>Em que Y representa o número do jogador que foi segundo lugar.</p><blockquote><p>&#34;O terceiro lugar foi: Z&#34;</p></blockquote><p>Em que Z representa o número do jogador que foi terceiro lugar.</p><p><span style="color: rgb(147, 149, 152);background-color: rgb(255, 255, 255);float: none;"><span style="color: rgb(147, 149, 152);background-color: rgb(255, 255, 255);float: none;"><span style="color: rgb(147, 149, 152);background-color: rgb(255, 255, 255);float: none;"><span style="color: rgb(147, 149, 152);background-color: rgb(255, 255, 255);float: none;"><br/></span></span></span></span></p><p><b>OBS: </b>Cada frase é seguida de uma quebra de linha</p>
