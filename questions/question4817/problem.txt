Título: Código de acesso
Topicos do problema: array; caractere; função; string; 
Tempo limite de execução: 1 segundo(s)
Descrição: <p></p><p>O setor de segurança do CIn desenvolveu um sistema de geração de <strong>códigos de acesso</strong>. Para validar cada usuário, o sistema usa uma sequência de dígitos e uma palavra-base, aplicando regras específicas para criar o código final.</p><p><b>Requisitos de implementação:<br/></b></p><title></title><p class="p1">O seu programa deve ser <b>modularizado</b>, dividindo a lógica em diferentes <span>funções: </span></p><p class="p1"><ol><li><span> Uma função para calcular o valor C. </span></li><li><span> Uma função para calcu</span>lar o valor R. </li><li> Uma função para gerar a palavra-base. </li><li> A função main deve orquestrar a chamada para as outras funções e imprimir o resultado final.</li></ol></p><p class="p1">Não é permitido o uso explícito de ponteiros (ou seja, declarar variáveis do tipo <span>* ou usar o operador &amp; para passar o endereço de variáveis simples para as </span><span>funções).</span></p><p><b>O seu programa deve:</b></p><ol><li><p>Ler um número inteiro <code>N</code> (tamanho da sequência), seguido de <code>N</code> dígitos (0–9), que devem ser armazenados em um <strong>array</strong>.</p></li><li><span>Calcular o valor </span><code>C</code><span> como:</span></li><pre
