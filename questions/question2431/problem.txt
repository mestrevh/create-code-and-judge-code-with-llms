Título: Vamos ao Cinema
Topicos do problema: estrutura de dados; 
Descrição: <p><span id="docs-internal-guid-1354d80d-7fff-c1e6-68fd-193f9d9ae99a"><span style="color: rgb(0, 0, 0);background-color: transparent;">Em um cinema existiam dois caixas funcionando, um preferencial e um normal. Também existem duas filas, uma regular e outra preferencial. </span><span style="color: rgb(0, 0, 0);background-color: transparent;"><br/></span><span style="color: rgb(0, 0, 0);background-color: transparent;">Quando o caixa A está disponível, a próxima pessoa da fila regular é chamada. </span><span style="color: rgb(0, 0, 0);background-color: transparent;"><br/></span><span style="color: rgb(0, 0, 0);background-color: transparent;">Quando o caixa B está disponível, a primeira pessoa da fila preferencial é chamada.</span><span style="color: rgb(0, 0, 0);background-color: transparent;"><br/></span><span style="color: rgb(0, 0, 0);background-color: transparent;">Se uma das filas estiver vazia, os caixas atendem pessoas de qualquer uma das filas.</span></span><br/></p>
Formato da entrada: <p><span id="docs-internal-guid-22bc36f3-7fff-e611-c1c3-bd420f8695c8"><span style="color: rgb(0, 0, 0);background-color: transparent;">A entrada começa com um inteiro T, que indica o número de casos de teste.</span><span style="color: rgb(0, 0, 0);background-color: transparent;"><br/></span><span style="color: rgb(0, 0, 0);background-color: transparent;">Cada caso começa com um inteiro N que indica quantos comandos terá o caso.</span><span style="color: rgb(0, 0, 0);background-color: transparent;"><br/></span><span style="color: rgb(0, 0, 0);background-color: transparent;">Cada pessoa que chega na fila contém um número de identificação.</span><span style="color: rgb(0, 0, 0);background-color: transparent;"><br/></span><span style="color: rgb(0, 0, 0);background-color: transparent;">As próximas N linhas contém os comandos:</span><span style="color: rgb(0, 0, 0);background-color: transparent;"><br/></span><span style="color: rgb(0, 0, 0);background-color: transparent;">&#34;f x&#34; = chegou uma pessoa com identificação x na fila regular.</span><span style="color: rgb(0, 0, 0);background-color: transparent;"><br/></span><span style="color: rgb(0, 0, 0);background-color: transparent;">&#34;p x&#34; = chegou uma pessoa com identificação x na fila preferencial.</span><span style="color: rgb(0, 0, 0);background-color: transparent;"><br/></span><span style="color: rgb(0, 0, 0);background-color: transparent;">&#34;A&#34; = o caixa A chama a próxima pessoa.</span><span style="color: rgb(0, 0, 0);background-color: transparent;"><br/></span><span style="color: rgb(0, 0, 0);background-color: transparent;">&#34;B&#34; = o caixa B chama a próxima pessoa.</span><span style="color: rgb(0, 0, 0);background-color: transparent;"><br/></span><span style="color: rgb(0, 0, 0);background-color: transparent;">&#34;I&#34; = imprima, em uma linha, as pessoas que estão na frente das filas f e p, nessa ordem. Se nenhuma pessoa estiver na fila, imprima 0.</span><span style="color: rgb(0, 0, 0);background-color: transparent;"><br/></span><span style="color: rgb(0, 0, 0);background-color: transparent;"><br/></span><span style="color: rgb(0, 0, 0);background-color: transparent;">Para cada caso de teste, as filas começam vazias.</span><span style="color: rgb(0, 0, 0);background-color: transparent;"><br/></span><span style="color: rgb(0, 0, 0);background-color: transparent;">Podem existir pessoas diferentes com identificações iguais.</span></span><br/></p>
Formato da saída: <p><span id="docs-internal-guid-3fed2260-7fff-6d99-067e-3d4b5029d1f8"><span style="color: rgb(0, 0, 0);background-color: transparent;">Para cada caso de teste, imprima &#34;Caso y:&#34;</span><span style="color: rgb(0, 0, 0);background-color: transparent;"><br/></span><span style="color: rgb(0, 0, 0);background-color: transparent;">para cada vez que aparecer o comando &#34;I&#34;, imprima em uma linha as primeiras pessoas das filas f e p, nessa ordem.</span><span style="color: rgb(0, 0, 0);background-color: transparent;"><br/></span><span style="color: rgb(0, 0, 0);background-color: transparent;">Obs: O cursor deve estar na última linha (em branco).</span></span><br/></p>
