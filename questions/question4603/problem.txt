Título: Desempenho do insertion sort
Topicos do problema: ordenação; 
Descrição: <p><span style="color: rgb(0, 0, 0);background-color: rgb(255, 255, 255);float: none;">Modifique a implementação do algoritmo Insertion Sort para contabilizar o número de deslocamentos que ele realiza durante a ordenação, o número de comparações realizadas e a quantidade de variáveis utilizadas. A única coisa que você deve imprimir é o número de deslocamentos feitos, o número de comparações e o número de variáveis pelo algoritmo para ordenar completamente o array. Um deslocamento ocorre quando a posição de um elemento muda no array. Não desloque um elemento se não for necessário.</span><br/></p>
Formato da entrada: <p style="color: rgb(0, 0, 0);">A primeira linha deve conter um inteiro n para o número de elementos a serem ordenados.</p><p style="color: rgb(0, 0, 0);">A próxima linha contem n inteiros do array.</p><p style="color: rgb(0, 0, 0);">Exemplo:</p><pre style="color: rgb(14, 20, 30);background-color: rgb(245, 245, 245);">STDIN       Function&#10;-----       --------&#10;5           arr[] size n =5&#10;2 1 3 1 2   arr = [2, 1, 3, 1, 2]</pre>
Formato da saída: <p><br/></p><p style="color: rgb(0, 0, 0);">Retorna apenas um inteiro correspondendo pela quantidade de trocas realizadas pelo algoritmo.</p><p style="color: rgb(0, 0, 0);">Exemplo:</p><p style="color: rgb(0, 0, 0);">4</p><pre style="color: rgb(14, 20, 30);background-color: rgb(245, 245, 245);">Iteration   Array      Shifts  Comparisons&#10;0           2 1 3 1 2&#10;1           1 2 3 1 2     1        &#10;2           1 2 3 1 2     0&#10;3           1 1 2 3 2     2&#10;4           1 1 2 2 3     1&#10;&#10;Total                     4</pre>
