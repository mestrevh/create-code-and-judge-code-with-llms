Título: Pivot Mediana
Topicos do problema: ordenação; 
Tempo limite de execução: 1 segundo(s)
Descrição: <p>Implemente um algoritmo de pivoteamento baseado em mediana. O algoritmo deve classificar o vetor de entrada de acordo com o pivot selecionado (mediana) e retorna o vetor de entrada classificado da maneira como a figura representa:</p><p style="text-align: center;"><img src="https://www.thehuxley.com/api/v1/problems/image/c8d872f9258a1708aee4e0d1d1d0807af51463f73.png"/><br/></p><p style="text-align: left;">Ou seja, elementos menores ou iguais que o pivot devem estar à &#34;esquerda&#34;(entre a posição inicial do vetor indicada pelo índice p e q-1)  do pivot (indicado pelo índice q representado pela cor verde). Elementos maiores que p pivot devem estar à &#34;direita&#34; (da posição q+1 até o final do array indicado pelo índice r). Finalmente o pivot selecionado deve se localizar na posição indicada pelo índice q separando os dois grupos de valores.</p><p style="text-align: left;">Valor médio para a mediana (m) = ( Maior valor + Menor valor ) / 2</p><p style="text-align: left;">Pivot Mediana =&gt; Encontrar o valor no vetor de entrada cuja a diferença seja a menor para m.</p>
Formato da entrada: <p>Um vetor de valores inteiros [0..n] com os números separados por espaço entre eles.</p>
Formato da saída: <p>Uma linha de dados informando o valor do pivô.<br/>Uma linha de dados <tt style="color: rgb(96, 96, 112);text-align: left;background-color: rgb(255, 255, 255);">z[0..n]</tt> representando os vetores de entrada classificados.<br/></p>
