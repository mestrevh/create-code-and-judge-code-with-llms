Título: Primeiro Dicionário de Andy
Topicos do problema: decisão; entrada formatada; repetição; substrings; função; string; dicionário; 
Descrição: <p>Andy de apenas 8 anos tem um sonho - ele deseja criar o seu próprio dicionário. Isto não é uma tarefa fácil para ele, pois conhece poucas palavras. Bem, ao invés de pensar nas palavras que sabe, ele teve uma ideia brilhante. A partir do seu livro de histórias favorito, ele vai criar um dicionário com todas as palavras distintas que existem nele. Ordenando estas palavras em ordem alfabética, o trabalho estará feito. É claro, isso é uma tarefa que toma um certo tempo e, portanto, a ajuda de um programador de computador como você é muito bem-vinda.</p><p>Você foi convidado a escrever um programa que liste todas as diferentes palavras que existem em um texto. Neste caso, uma palavra é definida como uma sequência de letras, maiúsculas ou minúsculas. Palavras com apenas uma letra também deverão ser consideradas. Portanto, seu programa deverá ser &#34;CaSe InSeNsItIvE&#34;. Por exemplo, palavras como &#34;Apple&#34;, &#34;apple&#34; ou &#34;APPLE&#34; deverão ser consideradas como a mesma palavra.<br/></p>
Formato da entrada: <p>A entrada contém no máximo 10.000 linhas de texto, cada uma delas com no máximo 200 caracteres. O fim de entrada é determinado por -1.<br/></p><p>O conjunto de caracteres especiais que devem ser excluídos é formado apenas por: . &#34; ( * $ # : <br/></p><p>Para remover estes caracteres especiais, deve ser criada uma <b>função</b> chamada <b>remover_especiais</b> com apenas <b>um parâmetro, do tipo string, chamado de palavra</b>. A função deve <b>retornar um valor</b>, também  do tipo <b>string</b>, contendo a <b>palavra sem os caracteres especiais</b>.<br/></p>
Formato da saída: <p>Você deve imprimir uma lista de diferentes palavras que aparecem no texto, uma palavra por linha. Todas as palavras devem ser impressas com letras minúsculas, em ordem alfabética. Deverá haver no máximo 5000 palavras distintas.</p>
