Descrição: <b id="docs-internal-guid-13f2b4f8-f04f-ac6f-828f-a6b43739f8bc"></b><p dir="ltr" style="text-align: left;"><span style="background-color: transparent;color: rgb(0, 0, 0);">Tiago (tfg) está muito feliz pois conseguiu aprender um algoritmo novo conhecido como FFT. Ele basicamente consegue multiplicar dois polinômios de maneira muito rápida. Agora ele pede sua ajuda para verificar se a resposta do seu algoritmo está certa. Você vai receber dois polinômios deve responder a multiplicação destes dois polinômios.</span><br/></p><p dir="ltr"><span style="color: #000000;background-color: transparent;">Embora tfg esteja usando o algoritmo FFT, ele quer que você use o algoritmo convencional para computar a multiplicação dos dois polinômios.</span></p><p dir="ltr"><span style="color: #000000;background-color: transparent;">Um polinômio de grau n é representado por:</span></p><p dir="ltr" style="text-align: center;"><span style="color: #000000;background-color: transparent;">A(x) = a</span><span style="color: #000000;background-color: transparent;">0 </span><span style="color: #000000;background-color: transparent;">x⁰ + a</span><span style="color: #000000;background-color: transparent;">1 </span><span transparent;">x^</span><span style="color: #000000;background-color: transparent;">n</span></p><br/><p dir="ltr"><span style="color: #000000;background-color: transparent;">Exemplo de multiplicação de polinômios: </span></p><p dir="ltr" style="text-align: center;"><span style="color: #000000;background-color: transparent;">A(x) = 5 + 3x + 2x²</span></p><p dir="ltr" style="text-align: center;"><span A(x) * B(x)</span></p><p dir="ltr" style="text-align: center;"><span style="color: #000000;background-color: transparent;">C(x) = (5 + 3x + 2x²) * (3 + 0x + 3x² + x³)</span></p><p dir="ltr" style="text-align: center;"><span style="color: #000000;background-color: transparent;">C(x) = 15 + 9x + 21x² + 14x³ + 9x⁴ + 2x⁵</span></p><p dir="ltr"><span style="color: #000000;background-color: transparent;">Este exemplo é o primeiro caso mostrado abaixo.</span></p>
Formato da entrada: <p dir="ltr" style="color: rgb(147, 149, 152);background-color: rgb(255, 255, 255);"><span style="color: rgb(0, 0, 0);background-color: transparent;">A entrada consiste de vários transparent;">Na primeira linha de cada caso  temos dois inteiros n e m (2 &lt;= n, m &lt;= 1000) que são os graus dos polinômios A e B, respectivamente.</span></p><p dir="ltr" style="color: rgb(147, 149, 152);background-color: rgb(255, 255, 255);"><span style="color: rgb(0, 0, 0);background-color: transparent;">Na segunda linha  de cada caso temos n+1 inteiros que são os coeficientes a</span><span style="color: rgb(0, 0, 0);background-color: transparent;">0</span><span style="color: rgb(0, 0, 0);background-color: transparent;">, a</span><span style="color: rgb(0, 0, 0);background-color: transparent;">1</span><span style="color: rgb(0, 0, 0);background-color: transparent;">, a</span><span style="color: rgb(0, 0, 0);background-color: transparent;">2</span><span style="color: rgb(0, 0, 0);background-color: transparent;">, … , a</span><span style="color: rgb(0, 0, 0);background-color: transparent;">n<span class="Apple-converted-space"> </span></span><span style="color: rgb(0, 0, 0);background-color: transparent;"> </span><span style="color: rgb(0, 0, 0);background-color: transparent;">do primeiro polinômio.</span></p><p dir="ltr" style="color: rgb(147, 149, 152);background-color: rgb(255, 255, 255);"><span style="color: rgb(0, 0, 0);background-color: transparent;">Na terceira linha de cada caso temos m+1 inteiros que são os coeficientes b</span><span style="color: rgb(0, 0, 0);background-color: transparent;">0</span><span style="color: rgb(0, 0, 0);background-color: transparent;">, b</span><span style="color: rgb(0, 0, 0);background-color: transparent;">1</span><span style="color: rgb(0, 0, 0);background-color: transparent;">, b</span><span style="color: rgb(0, 0, 0);background-color: transparent;">2</span><span Formato da saída: <p>A saída deve conter uma linha para cada caso com n+m+1 inteiros que são os coeficientes c0, c1, … , cn+m do polinômio C(x) = A(x) * B(x)  seguindo o formato dado no exemplo.<br/></p>
