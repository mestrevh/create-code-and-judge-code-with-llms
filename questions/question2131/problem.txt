Título: Matriz de votação
Topicos do problema: array; acumulador; array bidimensional; 
Descrição: <b id="docs-internal-guid-7e9367de-f204-ea7f-a041-0b9f664f4ca1" style="color: rgb(0, 0, 0);"></b><p dir="ltr" style="text-align: justify;"><span style="color: rgb(0, 0, 0);background-color: transparent;">Muito tempo atrás, em um lugar muito, muito distante, foi fundado o Reino Das Princesas Unidas. Trata-se de um grande Reino formado pela união dos povos liderados pelas princesas da Disney. Agora este Reino precisa eleger seu representante e é sua tarefa ajudar neste ponto.</span></p><br/><p dir="ltr" style="text-align: justify;"><span style="color: rgb(0, 0, 0);background-color: transparent;">A eleição será democrática, na qual os eleitores deverão votar nas princesas, indicando se concordam ou não que elas sejam suas representantes. Se concordar votam 1, se não concordar votam 0, para cada princesa.</span></p><br/><p dir="ltr" style="text-align: justify;"><span style="color: rgb(0, 0, 0);background-color: transparent;">Seu papel será colher os votos e apresentar um relatório com os votos dado a cada princesa.</span></p>
Formato da entrada: <b id="docs-internal-guid-7e9367de-f206-0717-34cc-9a83deec69ab" style="color: rgb(0, 0, 0);"></b><p dir="ltr" style="text-align: justify;"><span style="color: rgb(0, 0, 0);background-color: transparent;">A primeira linha da entrada é composta por um número natural </span><span style="color: rgb(0, 0, 0);background-color: transparent;">P</span><span style="color: rgb(0, 0, 0);background-color: transparent;">, referente a quantidade de princesas</span></p><br/><p dir="ltr" style="text-align: justify;"><span style="color: rgb(0, 0, 0);background-color: transparent;">A segunda linha tem um número natural </span><span style="color: rgb(0, 0, 0);background-color: transparent;">E</span><span style="color: rgb(0, 0, 0);background-color: transparent;">, representando a quantidade de eleitores.</span></p><p dir="ltr" style="text-align: justify;"><span style="color: rgb(0, 0, 0);background-color: transparent;">Cada uma das próximas </span><span style="color: rgb(0, 0, 0);background-color: transparent;">E</span><span style="color: rgb(0, 0, 0);background-color: transparent;"> linhas contém </span><span style="color: rgb(0, 0, 0);background-color: transparent;">P</span><span style="color: rgb(0, 0, 0);background-color: transparent;"> números que podem ser 1 ou 0, separados por espaço. Essas linhas representam os votos de um eleitor. Em cada uma dessas linhas, os </span><span style="color: rgb(0, 0, 0);background-color: transparent;">P</span><span style="color: rgb(0, 0, 0);background-color: transparent;"> números correspondem, em ordem as princesas 1, 2, 3,...,P.</span></p><br/><span style="color: rgb(0, 0, 0);background-color: transparent;">OBS:</span><span style="color: rgb(0, 0, 0);background-color: transparent;"> Você precisa, obrigatoriamente, armazenar os votos dos eleitores em um array bidimensional. Também é necessário um bloco de repetição para receber os votos e um diferente para apurá-los.</span>
Formato da saída: <b id="docs-internal-guid-7e9367de-f206-4bc1-5bb9-0a36d489caa1" style="color: rgb(0, 0, 0);"></b><p dir="ltr" style="text-align: justify;"><span style="color: rgb(0, 0, 0);background-color: transparent;">A saída deve conter </span><span style="color: rgb(0, 0, 0);background-color: transparent;">P</span><span style="color: rgb(0, 0, 0);background-color: transparent;"> linhas, uma para cada princesa, no seguinte formato:</span></p><br/><p dir="ltr" style="text-align: justify;"><span style="color: rgb(0, 0, 0);background-color: transparent;">Princesa </span><span style="color: rgb(0, 0, 0);background-color: transparent;">K</span><span style="color: rgb(0, 0, 0);background-color: transparent;">: </span><span style="color: rgb(0, 0, 0);background-color: transparent;">X</span><span style="color: rgb(0, 0, 0);background-color: transparent;"> voto(s)</span></p><br/><span style="color: rgb(0, 0, 0);background-color: transparent;">K</span><span style="color: rgb(0, 0, 0);background-color: transparent;"> representa o número da princesa e </span><span style="color: rgb(0, 0, 0);background-color: transparent;">X</span><span style="color: rgb(0, 0, 0);background-color: transparent;"> a quantidade de votos da mesma. As linhas devem estar em ordem da princesa 1 até a princesa </span><span style="color: rgb(0, 0, 0);background-color: transparent;">P</span><span style="color: rgb(0, 0, 0);background-color: transparent;">.</span>
