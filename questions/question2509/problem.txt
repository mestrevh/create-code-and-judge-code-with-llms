Título: LED
Topicos do problema: decisão; repetição; função; formatação; 
Tempo limite de execução: 1 segundo(s)
Descrição: <p>João quer montar um painel de leds contendo diversos números. Ele não possui muitos leds, e não tem certeza se conseguirá montar o número desejado. Considerando a configuração dos leds dos números abaixo, faça um programa que ajude João a descobrir a quantidade de leds necessário para montar o valor.</p><img src="https://www.thehuxley.com/api/v1/problems/image/5576a928da77642d41ed1a5bb7ffffb346a524ea2.png"/><div><u>O programa deve conter uma função chamada</u> <b>calcular_leds</b>, a qual deve receber um dígito e ter como resultado o número de LEDs necessários para exibi-lo.</div><p>Por exemplo, ao chamar a função passando como parâmetro o dígito 0, a mesma deve retornar o inteiro 6. Já para o dígito 1, a função deve retornar o número inteiro 2.</p><p><br/></p>
Formato da entrada: <p>A entrada contém um inteiro <b>N</b>, (1 ≤ <b>N</b> ≤ 1000) correspondente ao número de casos de teste, seguido de <b>N</b> linhas, cada linha contendo um número (1 ≤ <b>V</b> ≤  10<sup>100</sup><strong><sup></sup></strong>) correspondente ao valor que João quer montar com os leds.<br/></p>
Formato da saída: <p>Para cada caso de teste, imprima uma linha contendo o número de leds que João precisa para montar o valor desejado, seguido da palavra &#34;leds&#34;.<br/></p>
