Título: Ambrósio e a diferença
Topicos do problema: números binários; multiplas entradas; 
Tempo limite de execução: 2 segundo(s)
Descrição: <p>Professor Ambrósio continua ensinando binários para sua filha Ambrosinita como na questão : https://www.thehuxley.com/problem/599</p><p>Ele criou um novo jogo, Ambrósio escolherá um inteito A e Ambrosinita escolherá um inteiro B. O objetivo do jogo é que Ambrosinita responda qual a diferença entre esses 2 números. A diferença entre os dois números nesse jogo é igual a quantidade de bits que são diferentes entre os dois números em suas representações binárias.</p><p>Ex:  A = 1, B = 4;</p><p>A = (001) base 2.</p><p>B = (100) base 2.</p><p>a diferença é 2 (primeiro e ultimo bit).</p><p>Ajude a Ambrosinita a responder corretamente.</p><p><br/></p>
Formato da entrada: <p>Primeira linha contém um T indicando a quantidade de casos de teste.</p><p>As próximas T linhas contém A e B, indicando a escolha de cada um.</p><p>1 ≤ <strong style="color: rgb(37, 44, 51);background-color: rgb(255, 255, 255);">T</strong> ≤ 10<sup style="color: rgb(37, 44, 51);background-color: rgb(255, 255, 255);">4</sup><br style="color: rgb(37, 44, 51);background-color: rgb(255, 255, 255);"/>0 ≤ <font color="#252c33"><b>A</b></font>, <font color="#252c33"><b>B</b></font> ≤ 10<sup style="color: rgb(37, 44, 51);background-color: rgb(255, 255, 255);">16</sup><br/></p>
Formato da saída: <p>Para cada caso de teste imprima a quantidade de bits diferentes entre os 2 números em suas representações em binário.</p>
