Título: Nickname
Topicos do problema: formatação; 
Tempo limite de execução: 1 segundo(s)
Descrição: <p style="color: rgb(0, 0, 0);background-color: rgb(255, 255, 255);">No jogo<span> </span><i>Minecraft</i>, o jogador precisa escolher um nome de usuário (nickname) para servir como um identificador, que é muito necessário em ambientes multi-jogador.</p><p style="color: rgb(0, 0, 0);background-color: rgb(255, 255, 255);">Esse nickname precisa atender as seguintes restrições:</p><ul style="color: rgb(0, 0, 0);background-color: rgb(255, 255, 255);"><li>Mínimo de 3 caracteres;</li><li>Máximo de 16 caracteres;</li><li>Apenas caracteres alfanuméricos e underlines são permitidos.</li></ul><p style="color: rgb(0, 0, 0);background-color: rgb(255, 255, 255);"><br/></p><p style="color: rgb(0, 0, 0);background-color: rgb(255, 255, 255);">Quando Tantan foi escolher seu nickname, ele foi informado que não poderia utilizar espaços na composição do nome de usuário.</p><p style="color: rgb(0, 0, 0);background-color: rgb(255, 255, 255);"><br/></p><p style="color: rgb(0, 0, 0);background-color: rgb(255, 255, 255);">Sua tarefa é escrever um programa que gera o nickname baseado no nome da pessoa, para ajudar Tantan com o problema encontrado.</p>
Formato da entrada: <blockquote style="color: rgb(0, 0, 0);background-color: rgb(255, 255, 255);"><p>N</p><p>S</p></blockquote><ul style="color: rgb(0, 0, 0);background-color: rgb(255, 255, 255);"><li>N - string</li><li>S - string</li></ul><p style="color: rgb(0, 0, 0);background-color: rgb(255, 255, 255);"><br/></p><p style="color: rgb(0, 0, 0);background-color: rgb(255, 255, 255);">As linhas de entrada correspondem ao primeiro e último nome do jogador, respectivamente.</p><p style="color: rgb(0, 0, 0);background-color: rgb(255, 255, 255);"><br/></p><p style="color: rgb(0, 0, 0);background-color: rgb(255, 255, 255);">obs:</p><ul style="color: rgb(0, 0, 0);background-color: rgb(255, 255, 255);"><li>A entrada deve ser considerada amigável, ou seja, sempre estará no formato descrito;</li><li>O número total de caracteres sempre vai estar entre 3 e 16.</li></ul>
Formato da saída: <blockquote style="color: rgb(0, 0, 0);background-color: rgb(255, 255, 255);"><p>K</p></blockquote><ul style="color: rgb(0, 0, 0);background-color: rgb(255, 255, 255);"><li>K - string</li></ul><p style="color: rgb(0, 0, 0);background-color: rgb(255, 255, 255);"><br/></p><p style="color: rgb(0, 0, 0);background-color: rgb(255, 255, 255);">A linha única de saída é composta pelo nickname que vai ser usado pelo usuário, que deve ser formado pela junção do primeiro e último nome do jogador, sem ser separado por espaço.</p>
