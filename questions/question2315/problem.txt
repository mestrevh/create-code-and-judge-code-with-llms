Título: Naruto e as fichas
Topicos do problema: acumulador; 
Descrição: <p><b id="docs-internal-guid-1cfb028b-7fff-3beb-5cce-80f5d976c0b7" style="color: rgb(0, 0, 0);"></b></p><p dir="ltr"><span style="color: rgb(0, 0, 0);background-color: transparent;">Naruto decide ir ao Arcade depois de tanto treinamento para espairecer um pouco as suas ideias. Ao chegar lá, descobre que as fichas usadas para jogar em cada máquina possuíam tamanhos diferentes. Naruto teve a ideia de usar suas Shurikens no lugar das fichas. Para isso, ele criou uma codificação, relacionando cada tamanho de ficha com o nome de uma Shuriken. Sendo assim, Naruto pediu para você fazer um programa que realiza três atividades: </span></p><p dir="ltr"><span style="color: rgb(0, 0, 0);background-color: transparent;">1) Associa o tamanho das fichas ao nome das Shurikens; </span></p><p dir="ltr"><span style="color: rgb(0, 0, 0);background-color: transparent;">2) Ordena a tabela de códigos por tamanho de ficha, da menor para a maior; </span></p><p dir="ltr"><span style="color: rgb(0, 0, 0);background-color: transparent;">3) Informa o nome da Shurikens que está na i-ésima posição da tabela de código ordenada.</span></p><br/><p dir="ltr"><span style="color: rgb(0, 0, 0);background-color: transparent;">Segue um exemplo das três atividades:</span></p><br/><p dir="ltr"><span style="color: rgb(0, 0, 0);background-color: transparent;">Atidade 1 - Codificação</span></p><p dir="ltr"><span style="color: rgb(0, 0, 0);background-color: transparent;">Shuriken1 13cm</span></p><p dir="ltr"><span style="color: rgb(0, 0, 0);background-color: transparent;">Shuriken2 14cm</span></p><p dir="ltr"><span style="color: rgb(0, 0, 0);background-color: transparent;">Shuriken3 12cm</span></p><p dir="ltr"><span style="color: rgb(0, 0, 0);background-color: transparent;">Shuriken4 09cm</span></p><br/><p dir="ltr"><span style="color: rgb(0, 0, 0);background-color: transparent;">Atividade 2 - Ordenação</span></p><p dir="ltr"><span style="color: rgb(0, 0, 0);background-color: transparent;">Shuriken4 09cm</span></p><p dir="ltr"><span style="color: rgb(0, 0, 0);background-color: transparent;">Shuriken3 12cm</span></p><p dir="ltr"><span style="color: rgb(0, 0, 0);background-color: transparent;">Shuriken1 13cm</span></p><p dir="ltr"><span style="color: rgb(0, 0, 0);background-color: transparent;">Shuriken2 14cm</span></p><br/><p dir="ltr"><span style="color: rgb(0, 0, 0);background-color: transparent;">Atividade 3 - Retorna o nome da Shurikens na i-ésima posição da tabela ordenada</span></p><br/><p dir="ltr"><span style="color: rgb(0, 0, 0);background-color: transparent;">Assumindo </span><span style="color: rgb(0, 0, 0);background-color: transparent;">i</span><span style="color: rgb(0, 0, 0);background-color: transparent;"> igual a 2, seria retornado o nome Shuriken3.</span><span style="color: rgb(0, 0, 0);background-color: transparent;"><br class="kix-line-break"/></span><span style="color: rgb(0, 0, 0);background-color: transparent;"><br class="kix-line-break"/></span><span style="color: rgb(0, 0, 0);background-color: transparent;">OBS</span><span style="color: rgb(0, 0, 0);background-color: transparent;">: É garantido que não haverão shurikens de tamanhos iguais</span></p><br/><p dir="ltr"><span style="color: rgb(0, 0, 0);background-color: transparent;">DICA</span><span style="color: rgb(0, 0, 0);background-color: transparent;">: Para ordenar arrays de forma crescente, utilizando o bubble sort, como visto em sala, você precisa seguir os passos:</span></p><ul><li dir="ltr" style="color: rgb(0, 0, 0);background-color: transparent;"><p dir="ltr"><span style="color: rgb(0, 0, 0);background-color: transparent;">Para cada posição X do array, é realizada uma comparação com cada uma das outras posições Y do array, caso o valor na posição X seja maior que o valor na posição Y, haverá a troca dos valores nas posições X e Y. </span></p></li><li dir="ltr" style="color: rgb(0, 0, 0);background-color: transparent;"><p dir="ltr"><span style="color: rgb(0, 0, 0);background-color: transparent;">Você deve repetir estes passos N vezes, para um array de tamanho N.</span></p></li></ul>
Formato da entrada: <p><span style="color: rgb(147, 149, 152);background-color: rgb(255, 255, 255);float: none;"><b id="docs-internal-guid-f128445a-7fff-f65c-f882-dc2a00bcdb8c" style="color: rgb(0, 0, 0);"></b></span></p><p dir="ltr"><span style="color: rgb(0, 0, 0);background-color: transparent;">N </span><span style="color: rgb(0, 0, 0);background-color: transparent;">- número de possíveis tamanhos de fichas.</span></p><br/><p dir="ltr"><span style="color: rgb(0, 0, 0);background-color: transparent;">N linhas com a codificação no formato:</span></p><p dir="ltr"><span style="color: rgb(0, 0, 0);background-color: transparent;">NOME-1 T-1</span></p><p dir="ltr"><span style="color: rgb(0, 0, 0);background-color: transparent;">NOME-2 T-2</span></p><p dir="ltr"><span style="color: rgb(0, 0, 0);background-color: transparent;">…</span></p><p dir="ltr"><span style="color: rgb(0, 0, 0);background-color: transparent;">NOME-N T-N</span></p><br/><p dir="ltr"><span style="color: rgb(0, 0, 0);background-color: transparent;">Representando a codificação, em que, </span><span style="color: rgb(0, 0, 0);background-color: transparent;">Nome-i</span><span style="color: rgb(0, 0, 0);background-color: transparent;"> corresponde ao nome da Shuriken e </span><span style="color: rgb(0, 0, 0);background-color: transparent;">T-i</span><span style="color: rgb(0, 0, 0);background-color: transparent;"> ao tamanho da ficha (em centímetros) </span></p><br/><p dir="ltr"><span style="color: rgb(0, 0, 0);background-color: transparent;">Seguido de:</span></p><p dir="ltr"><span style="color: rgb(0, 0, 0);background-color: transparent;">i</span><span style="color: rgb(0, 0, 0);background-color: transparent;"> - índice da tabela de codificação ordenada.</span></p><br/><br/>
Formato da saída: <p><span style="color: rgb(147, 149, 152);background-color: rgb(255, 255, 255);float: none;"></span></p><p dir="ltr" style="color: rgb(147, 149, 152);background-color: rgb(255, 255, 255);"><span style="color: rgb(0, 0, 0);background-color: transparent;">Deverá ser impresso:</span></p><p dir="ltr" style="color: rgb(147, 149, 152);background-color: rgb(255, 255, 255);"><span style="color: rgb(0, 0, 0);background-color: transparent;">“Naruto devera usar a shuriken NOME para esta maquina.”</span></p><p dir="ltr" style="color: rgb(147, 149, 152);background-color: rgb(255, 255, 255);"><span style="color: rgb(0, 0, 0);background-color: transparent;">Substituindo<span class="Apple-converted-space"> </span></span><span style="color: rgb(0, 0, 0);background-color: transparent;">NOME<span class="Apple-converted-space"> </span></span><span style="color: rgb(0, 0, 0);background-color: transparent;">pelo respectivo nome da Shuriken.</span></p><br/>
