Título: Triângulos
Topicos do problema: geometria computacional; lógica matemática; 
Tempo limite de execução: 1 segundo(s)
Descrição: <p>Caio estava brincando de construir triângulos com palitos de diferentes tamanhos. Ele fazia isso juntando as pontas de três palitos sobre uma mesa. Ele notou que podia agrupar os triângulos formados em três grupos:</p>&#13;<p>• Triângulos acutângulos, que são aqueles em que todos os ângulos internos medem menos de 90&#14;;<br/><br/>• Triângulos retângulos, que são aqueles que possuem um ângulo interno que mede exatamente 90&#14;;<br/><br/>• Triângulos obtusângulos, que são aqueles que possuem um ângulo interno que mede mais de 90&#14;.</p>&#13;<p>Ele também percebeu que nem sempre é possível formar um triângulo com três palitos.</p>&#13;<p><a href="https://thehuxley.com/api/v1/problems/image/a0b4762e5795d7cae5caa9371c1c27f0.jpg" target="_blank"><img src="https://thehuxley.com/api/v1/problems/image/a0b4762e5795d7cae5caa9371c1c27f0.jpg" border="0" width="524" height="173"/></a></p>&#13;<p>Sua tarefa é, dados os comprimentos A, B e C de três palitos, dizer se é possível formar um triângulo com esses palitos e, em caso a&#28;rmativo, dizer a qual grupo o triângulo formado pertence.</p>
Formato da entrada: <p>A entrada consiste de uma &uacute;nica linha, contendo tr&ecirc;s inteiros A, B e C separados por espa&ccedil;o.</p>
<p>Considere:</p>
<p>1 &lt;= A &lt;= 10000</p>
<p>1 &lt;= B &lt;= 10000</p>
<p>1 &lt;= C &lt;= 10000</p>
Formato da saída: <p>Imprima uma linha contendo apenas uma letra min&uacute;scula:<br />&bull; 'n' se n&atilde;o for poss&iacute;vel formar um tri&acirc;ngulo;<br />&bull; 'a' se o tri&acirc;ngulo formado for acut&acirc;ngulo;<br />&bull; 'r' se o tri&acirc;ngulo formado for ret&acirc;ngulo;<br />&bull; 'o' se o tri&acirc;ngulo formado for obtus&acirc;ngulo.</p>
