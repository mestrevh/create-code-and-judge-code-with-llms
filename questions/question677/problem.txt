Título: Grandes Atalhos
Topicos do problema: string; 
Tempo limite de execução: 1 segundo(s)
Descrição: <p dir="ltr" style="color: rgb(0, 0, 0);text-align: justify;background-color: rgb(255, 255, 255);"><span style="color: rgb(34, 34, 34);background-color: rgb(255, 255, 255);">Uma grande empresa de tecnologia percebeu que os usuários de seu chat poderiam alcançar uma maior velocidade de digitação se pudessem definir atalhos pessoais para palavras que utilizam bastante. Com o lançamento dessa nova feature, os usuários podem personalizar seu chat de forma que todas vezes que eles escreverem *C (onde C é um caractere escolhido), a palavra A será colocada no lugar.</span></p><br style="color: rgb(0, 0, 0);background-color: rgb(255, 255, 255);"/><p dir="ltr" style="color: rgb(0, 0, 0);text-align: justify;background-color: rgb(255, 255, 255);"><span style="color: rgb(34, 34, 34);background-color: rgb(255, 255, 255);">Dada uma string de entrada S e N atalhos da forma C -&gt; A, onde C é um caractere e A é uma string, escreva um programa que imprime S', uma string onde todas as ocorrências de *C em S são transformadas em A, para cada atalho.</span></p>
Formato da entrada: <p><span style="color: rgb(34, 34, 34);background-color: rgb(255, 255, 255);">A primeira linha da entrada contém um inteiro T (1<span class="Apple-converted-space"> </span></span><span style="color: rgb(51, 51, 51);background-color: rgb(255, 255, 255);">≤</span><span style="color: rgb(34, 34, 34);background-color: rgb(255, 255, 255);"><span class="Apple-converted-space"> </span>T<span class="Apple-converted-space"> </span></span><span style="color: rgb(51, 51, 51);background-color: rgb(255, 255, 255);">≤</span><span style="background-color: rgb(255, 255, 255);"><span class="Apple-converted-space" style="color: rgb(34, 34, 34);"> </span><font color="#222222">100), o número de casos de teste. Cada caso de teste começa com uma linha contendo S (1</font><span class="Apple-converted-space" style="color: rgb(34, 34, 34);"> </span></span><span style="color: rgb(51, 51, 51);background-color: rgb(255, 255, 255);">≤</span><span style="color: rgb(34, 34, 34);background-color: rgb(255, 255, 255);"><span class="Apple-converted-space"> </span>|S|<span class="Apple-converted-space"> </span></span><span style="color: rgb(51, 51, 51);background-color: rgb(255, 255, 255);">≤</span><span style="color: rgb(34, 34, 34);background-color: rgb(255, 255, 255);"><span class="Apple-converted-space"> </span>10^5), representando a string de entrada. A linha seguinte contém um inteiro N (0<span class="Apple-converted-space"> </span></span><span style="color: rgb(51, 51, 51);background-color: rgb(255, 255, 255);">≤</span><span style="color: rgb(34, 34, 34);background-color: rgb(255, 255, 255);"><span class="Apple-converted-space"> </span>N<span class="Apple-converted-space"> </span></span><span style="color: rgb(51, 51, 51);background-color: rgb(255, 255, 255);">≤</span><span style="color: rgb(34, 34, 34);background-color: rgb(255, 255, 255);"><span class="Apple-converted-space"> </span>26), indicando o numero de atalhos definidos. As N linhas seguintes contém  os atalhos, da forma C A, onde C é um caractere e A é uma string (1<span class="Apple-converted-space"> </span></span><span style="color: rgb(51, 51, 51);background-color: rgb(255, 255, 255);">≤</span><span style="color: rgb(34, 34, 34);background-color: rgb(255, 255, 255);"><span class="Apple-converted-space"> </span>|A|<span class="Apple-converted-space"> </span></span><span style="color: rgb(51, 51, 51);background-color: rgb(255, 255, 255);">≤</span><span style="background-color: rgb(255, 255, 255);"><span class="Apple-converted-space" style="color: rgb(34, 34, 34);"> </span><font color="#222222">10). As strings de entrada e dos atalhos são compostas apenas por letras maiúsculas e *, e todos os caracteres dos atalhos são letras maiúsculas. É garantido que cada caractere C aparecerá no máximo uma vez por caso de entrada.</font></span><br/></p>
Formato da saída: <p><span style="color: rgb(34, 34, 34);float: none;background-color: rgb(255, 255, 255);">Para cada caso de teste, imprima uma linha com a string resultante depois que os atalhos são aplicados à string de entrada.</span><br/></p>
