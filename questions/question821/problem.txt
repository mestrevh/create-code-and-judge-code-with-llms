Título: Bitmap
Topicos do problema: array bidimensional; dividir e conquistar; recursão; 
Descrição: <p>O bitmap é uma estrutura de dados que é utilizada em muitas áreas da computação. Na Computação Gráfica, por exemplo, um bitmap pode representar uma imagem, onde 1 seria um pixel preto e 0 um pixel branco.</p><p>Mas as formas de representar um bitmap podem ser bem variadas. Neste problema, o objetivo é transformar de uma forma para outra.</p><p>A primeira forma é simplesmente um array bidimensional composto de 0s e 1s. Já a segunda forma é gerada a partir de uma decomposição dessa matriz.</p><p>Primeiro, o bitmap inteiro é considerado. Se todos os bits forem 1, então 1 é a saída. Se todos forem 0, então 0 é a saída. Caso contrário, a saída deve ser D e o bitmap deve ser dividido em até 4 partes. Cada uma das partes deve ser processada da mesma maneira que o bitmap original.</p><p>Os 4 quadrantes são processados na seguinte ordem: superior esquerdo, superior direito, inferior esquerdo, inferior direito.</p><p>Quando um bitmap possui um número par de linhas e também de colunas, então todos os quadrantes possuirão as mesmas dimensões.</p><p>Quando o número de colunas é ímpar, os quadrantes da esquerda devem ter uma coluna a mais do que os quadrantes da direita.</p><p>Quando o número de linhas é ímpar, os quadrantes superiores devem possuir uma linha a mais do que os quadrantes inferiores.</p><p>Seu objetivo é ler um bitmap no formato de matriz e transformá-lo na forma descrita.<br/></p>
Formato da entrada: <p>A entrada consiste de um número n indicando a quantidade de casos de teste.</p><p>Cada caso de teste consiste de uma linha com os números L e C, indicando o número de linhas e colunas do bitmap. (1&lt;= L, C &lt;= 200)<br/></p><p>Na sequência seguirão N linhas, cada uma com C colunas de caracteres 0 ou 1. <br/></p>
Formato da saída: <p>A saída consiste nos bitmaps transformados. Sendo um bitmap para cada caso de teste.</p><p>Cada linha do bitmap deve ter 50 colunas, exceto a última linha, que pode conter menos.<br/></p>
