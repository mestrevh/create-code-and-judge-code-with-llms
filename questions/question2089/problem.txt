Título: Happy Number
Topicos do problema: repetição; 
Descrição: <p style="color: rgb(0, 0, 0);background-color: rgb(255, 255, 255);">Willy acabou de descobrir o conceito de Happy Number e ficou fascinado com esses números, então ele pediu que você o ajudasse a dizer se um número dado é um Happy Number.</p><p style="color: rgb(0, 0, 0);background-color: rgb(255, 255, 255);">Um número é dito Happy Number quando o substituímos pela soma do quadrado dos seus dígitos repetidas vezes e acabamos chegando ao número 1, exemplo:</p><p style="color: rgb(0, 0, 0);background-color: rgb(255, 255, 255);">7: 7² = 49: 4² + 9² = 97: 9² + 7² = 130: 1² + 3² + 0² = 10: 1² + 0² = 1. Logo 7 é um Happy Number.</p><p style="color: rgb(0, 0, 0);background-color: rgb(255, 255, 255);">Uma forma fácil de descobrir que um número <b>não </b>é Happy Number é quando ao repetirmos o processo de substituição chegamos a um número que já tínhamos chegado antes, exemplo:</p><p style="color: rgb(0, 0, 0);background-color: rgb(255, 255, 255);">37: 3² + 7² = 58: 5² + 8² = 89: 8² + 9² = 145: 1² + 4² + 5²  = 42: 4² + 2² = 20: 2² + 0² = 4: 4² = 16: 1² + 6² = 37. Logo 37 não é um Happy Number</p><p style="color: rgb(0, 0, 0);background-color: rgb(255, 255, 255);"><br/></p><p style="color: rgb(0, 0, 0);background-color: rgb(255, 255, 255);">Agora é sua vez de ajudar Willy, dado um número<span> </span><b>N<span> </span></b>diga se ele é umHappy Number.</p>
Formato da entrada: <p style="color: rgb(0, 0, 0);background-color: rgb(255, 255, 255);">Uma única linha contendo um inteiro<span> </span><b>N</b>.</p><p style="color: rgb(0, 0, 0);background-color: rgb(255, 255, 255);">1 &lt;=<span> </span><b>N</b><span> </span>&lt;= 1000</p>
Formato da saída: <p style="color: rgb(0, 0, 0);background-color: rgb(255, 255, 255);"><span>Uma única linha contendo<span> </span></span><b>S</b>, caso seja um HAPPY Number, ou contendo <b>N</b>, caso não seja um Happy Number<span>.</span><span> </span></p>
