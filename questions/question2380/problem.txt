Título: Busca em profundidade
Topicos do problema: grafo; 
Descrição: <title></title><p><font color="#000000">A busca em profundidade é um algoritmo utilizado para percorrer ou buscar itens dentro das estruturas de dados grafos. Como característica temos que a busca sempre ocorre vértices mais distantes ao vértice pelo qual a busca foi iniciada.</font></p><p><span style="color: rgb(0, 0, 0);">Desenvolva um programa para percorrer um grafo a partir de um vértice inicial e apresentar a ordem em que os vértices foram visitados, considerando a lógica do algoritmo de busca em profundidade.</span><br/></p>
