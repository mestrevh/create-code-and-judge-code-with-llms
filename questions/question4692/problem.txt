Título: Selection Sort passo a passo
Topicos do problema: estrutura de dados; eof; array; alocação dinâmica; ponteiro; ordenação; 
Tempo limite de execução: 2 segundo(s)
Descrição: <p>Crie um programa que recebe um array de tamanho variado e ordena o array de forma crescente. A ordenação deve ser feita utilizando um selection sort. </p><p>Além disso, sempre que um número for movido para sua posição o programa deve exibir o array.</p><p>`Por fim, a ordenação deve ser feita movendo o menor número para a esquerda.<br/></p>
Formato da entrada: <p>A entrada é um array numérico.<br/></p>
Formato da saída: <p>A saída deve exibir o array original e a cada linha mostrar a atualização dos valores do array quando um número for movido de sua posição original para sua nova posição. <br/></p>
