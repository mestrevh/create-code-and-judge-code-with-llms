Título: A Sequência de Josué
Topicos do problema: decisão; caractere; 
Tempo limite de execução: 1 segundo(s)
Descrição: <p>Em um de seus dias de trabalho nos correios, Josué, O Carteiro, teve de ficar o dia inteiro separando as encomendas no estoque. Como ele é um rapaz bastante <strike>esquisito</strike> excêntrico resolveu deixar o seu trabalho um pouco mais divertido. Como lá as encomendas são divididas por etiquetas com letras maiúsculas e minúsculas, ele resolveu encontrar determinadas sequências com apenas letras <b>MAIÚSCULAS</b> a cada trio de encomendas. Quando ordenadas, se as letras formarem uma sequência direta e forem apenas consoantes, como B C e D, temos uma &#34;Sequência perfeita.&#34;, caso tenha vogais, será uma &#34;Sequência quase perfeita.&#34;. Se a distância entre as letras for de 2 letras, teremos uma &#34;Bissequência perfeita.&#34; quando existirem apenas consoantes e &#34;Bissequência quase perfeita&#34; se possuir alguma vogal e se a distância for de 3 letras, teremos uma &#34;Trissequência perfeita&#34; quando possuir apenas consoantes e uma &#34;Trissequência quase perfeita&#34; quando possuir alguma vogal. Seu trabalho é dados 3 caracteres indicar se eles fazem uma sequência do <strike>maluco</strike> grandioso Josué ou se não.</p>
Formato da entrada: <p><b>3 caracteres na mesma linha em qualquer ordem.</b></p>
Formato da saída: <p>&#34;<b>Sequência perfeita</b>.&#34; caso a &#34;distância&#34; entre as letras seja de apenas uma letra e possua apenas consoantes.</p><p><span>&#34;<b>Sequência quase perfeita.</b>&#34; caso a &#34;distância&#34; entre as letras seja de apenas uma letra e possua vogais.</span></p><p><span>&#34;<b>Bissequência perfeita.</b>&#34; caso a &#34;distância&#34; entre as letras seja de duas letra e possua apenas consoantes.</span></p><p><span>&#34;<b>Bissequência quase perfeita.</b>&#34; caso a &#34;distância&#34; entre as letras seja de duas letra e possua vogais.</span></p><p><span>&#34;<b>Trissequência perfeita.</b>&#34; caso a &#34;distância&#34; entre as letras seja de três letra e possua apenas consoantes.</span></p><p><span>&#34;<b>Trissequência quase perfeita.</b>&#34; caso a &#34;distância&#34; entre as letras seja de três letra e possua vogais.</span></p><p><span>&#34;<b>Só umas letras ai...</b>&#34; caso sejam apenas letras maiúsculas com distâncias variadas.</span></p><p><span>&#34;<b>Etiquetas erradas!</b>&#34; caso possua algum caractere que não seja uma letra maiúscula.</span></p><p><span>OBS: A distância de uma letra significa A + 1 = B, duas letras A + 2 = C e três letras A + 3 = D</span></p>
