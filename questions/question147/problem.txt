Título: Soma das casas
Topicos do problema: lógica matemática; 
Descrição: <p>Joãozinho mora em uma rua que tem N casas. Marquinhos é o melhor amigo dele, mas sempre gosta de pregar peças em Joãozinho. Desta vez, ele pegou os dois brinquedos prediletos de Joãozinho e os escondeu em duas casas distintas da rua. Em compensação, Marquinhos deu uma dica importante para Joãozinho: <br/><br/>A soma dos números das casas em que escondi teus brinquedos é igual a K. Além disso, escolhi as casas de tal forma que não existe outro par de casas cuja soma tenha esse mesmo valor.<br/><br/>Sabendo disto, encontre qual é o par de casas em que se encontram os brinquedos de Joãozinho. Para auxiliar seu amigo, Marquinhos entregou a Joãozinho uma lista com o número das casas já em ordem crescente (isto é, do menor para o maior número).</p>&#13;<p> </p>&#13;<p><a href="../../data/images/app/problems/76c3ade49f1333b41222e51d71360835.png" target="_blank"><img src="https://thehuxley.com/api/v1/problems/image/76c3ade49f1333b41222e51d71360835.png" border="0" width="163" height="163"/></a></p><p><br/></p>
Formato da entrada: <p>A primeira primeira linha da entrada cont&eacute;m um n&uacute;mero inteiro N, que representa o n&uacute;mero de casas que existem na rua. Cada uma das N linhas seguintes cont&eacute;m um n&uacute;mero inteiro, representando o n&uacute;mero de uma casa. Note que esses N n&uacute;meros est&atilde;o ordenados, do menor para o maior. A &uacute;ltima linha da entrada cont&eacute;m um inteiro K, que &eacute; a soma dos n&uacute;meros das duas casas onde os brinquedos est&atilde;o escondidos.</p>
<p>Suponha:</p>
<ul>
<li>Em um conjunto de casos que totaliza 30 pontos, N &lt;= 1000.</li>
<li>2 &lt;= N &lt;= 100000</li>
<li>Para cada casa Ci, 0 &lt;= Ci &lt;= 1000000000, i = 1,2, ... ,N</li>
<li>Os n&uacute;meros das casas est&atilde;o em ordem crescente, do menor para o maior n&uacute;mero, e casas distintas t&ecirc;m n&uacute;meros distintos.</li>
</ul>
Formato da saída: <p>Seu programa deve imprimir uma &uacute;nica linha, contendo dois inteiros, A e B, A &lt; B, que representam os n&uacute;meros das casas em que est&atilde;o escondidos os brinquedos. Os dois n&uacute;meros devem ser separados por um espa&ccedil;o em branco.</p>
