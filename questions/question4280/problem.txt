Título: Aeroporto - Lista de checagem
Topicos do problema: multiplas entradas; decisão; repetição; 
Tempo limite de execução: 1 segundo(s)
Descrição: <p>você é um segurança com uma lista checagem, o seu trabalho é checar cada um do passageiros e ver se eles tem</p><ol><li>RG;</li><li>Passagem</li></ol><p>Após isso você deve checar se as datas de nascimento no RG e na passagem são iguais, qual é o assento do passageiro e informa-lo qual é o seu assento</p><p>se o passageiro não tiver um RG ele deve ser direcionado para a saída;</p><p>se o passageiro não tiver uma passagem ele deve ser direcionado para a recepção do aeroporto;</p><p>se a data de nascimento do RG e da Passagem não baterem você deve chamar a policia. </p>
Formato da entrada: <p>um número inteiro positivo N, que representa a quantidade de passageiros;</p><p>uma String com o valor &#34;RG&#34; ou &#34;Nao possui&#34; para indicar se a pessoa possui ou não RG;</p><p>uma String no formato &#34;DD/MM/AAAA&#34; indicando a data de nascimento do RG;</p><p>uma String com o valor &#34;Passagem&#34; ou &#34;Nao possui&#34; para indicar se a pessoa possui ou não Passagem;</p><p><span style="color: rgb(147, 149, 152);background-color: rgb(255, 255, 255);float: none;">uma String no formato &#34;DD/MM/AAAA&#34; indicando a data de nascimento da Passagem;</span><br/></p><p><span style="color: rgb(147, 149, 152);background-color: rgb(255, 255, 255);float: none;">uma String no formato &#34;A12&#34; indicando a cadeira do passageiro.</span></p>
Formato da saída: <p>caso o passageiro não tenha RG a saída deve ser &#34;a saída é nessa direção&#34;;</p><p>caso o passageiro não tenha Passagem a saída deve ser &#34;a recepção é nessa direção&#34;;</p><p>caso as datas de nascimento não sejam iguais a saída deve ser &#34;190&#34;;</p><p>caso todos os anteriores não aconteção a saída deve ser &#34;o seu assento é&#34; seguido do assento do passageiro e depois &#34; tenha um ótimo dia&#34;.</p>
