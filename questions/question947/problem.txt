Título: Remoção de strings
Topicos do problema: programação dinâmica; 
Tempo limite de execução: 1 segundo(s)
Descrição: <p>Dada uma string s de 2 caracteres 'a' e 'b'. Seja um grupo a maior substring de um mesmo caractere consecutivo.</p><p>Qualquer grupo g de tamanho s de tamanho de pelo menos 2 pode ser removido e uma nova string é construída concatenando o que resta do lado esquerdo com o que resta do lado direito de s.</p><p>Esse processo é repetido até a string se tornar vazia ou não existir mais grupos de tamanho pelo menos 2.</p><p>Por exemplo, a string s = babbbaaabb tem 5 grupos: b, a, bbb, aaa e bb. A string s pode ser transformada em uma string vazia removendo os grupos da seguinte sequência:</p>babbbaaabb ----- (removendo 'bbb') ----&gt; baaaabb<br/>baaaabb ----- (removendo 'aaaa') ----&gt; bbb<br/>bbb ----- (removendo 'bbb') ----&gt; string vazia<br/><br/>Entretanto, dependendo da sequência de remoções, você não conseguirá gerar uma string vazia. Lembre-se que você só poderá remover grupos, de acordo com a definição de grupo acima. Por exemplo, se as remoções forem:<br/><br/>babbbaaabb ----- (removendo 'bb') ----&gt; babbbaaa<br/>babbbaaa ----- (removendo 'bbb') ----&gt; baaaa<br/>baaaa ----- (removendo 'aaaa') ----&gt; b<br/><br/>No exemplo acima, 'b' não é um grupo, pois possui somente 1 elemento.<br/><br/>Dada uma string, escreva um programa para decidir se a string pode ser transformada em uma string fazia através de uma sequência de operações de remoções de grupos.<br/><br/><br/><br/><br/>
Formato da entrada: <p>Você receberá uma linha contendo o inteiro T indicando a quantidade de casos de testes.</p><p>Cada caso de teste consiste de uma única linha contendo N caracteres 'a' e 'b' (1&lt;= N &lt;=25).</p>
Formato da saída: <p>Para cada caso de teste imprima uma única linha contendo 0 ou 1. <br/></p><p>1 caso a string pode ser transformada em uma string vazia. 0 caso contrário.<br/></p>
