Título: Palíndromo
Topicos do problema: programação dinâmica; 
Tempo limite de execução: 1 segundo(s)
Descrição: <p>Um palíndromo é uma palavra que lê a mesma coisa para a esquerda que lê para a direita. Por exemplo, I, GAG e MADAM são palíndromos,mas ADAM não é. Aqui, nós consideramos também uma palavra vazia como um palíndromo.Para cada palavra que não é um palíndomo, você pode sempre retirar algumas letras e obter uma subsequência que é um palíndromo. Por exemplo,dada uma palavra ADAM, se você remover a letra M você possui um palíndromo ADA. Escreva um programa para determinar o tamanho do maior palíndromoque você pode conseguir de uma palavra.<br/></p>
Formato da entrada: <p>A primeira linha da entrada contem um inteiro T (≤ 60). Cada uma das T linhas seguintes é uma palavra, cujo tamanho é sempre menor que 1000.Para ≥ 90% dos casos de teste, o tamanho da palavra é ≤ 255.<br/></p>
Formato da saída: <p>Para cada palavra recebida, seu programa deve imprimir o tamanho do maior palindromo que você pode conseguir removendo zero ou mais caracteresdele.<br/></p>
