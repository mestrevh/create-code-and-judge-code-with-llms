Título: Big Problem Brasil
Topicos do problema: eof; acumulador; 
Descrição: <p>Deyene e Kinho amam o BBB (Big Brother Brasil), e na edição 21 já declaram torcida para Gil do Janga e Juliette de Juliette.</p><p>Eles amam tanto esses dois personagens no jogo que decidiram criar um bot de votação para ajudá-los, porém, a Gl0b0 descobriu a malícia por trás da ideia de Deyene e Kinho e decidiu colocar captchas para que o bot não conseguisse acertar. Como Kinho e Dey ainda não sabem mexer com Deep Learning, decidiram que o bot irá escolher captchas aleatórios com a probabilidade de acerto em 25%.</p><p>O que eles não sabiam, é que toda vez que o captcha era selecionado errado, o bot era penalizado no envio de requisitos para o servidor da Globo, essa penalidade era:</p><blockquote><p><span id="selectionBoundary_1612968702056_10558792054794841" class="rangySelectionBoundary">&#65279;</span>Dado uma chave <b>K</b> e uma base <b>B</b>, descubra qual o código correto para enviar os requisitos. Para descobrir o código é simples, você apenas deve somar todos os números de K na base B.<span id="selectionBoundary_1612968702056_13183862831777549" class="rangySelectionBoundary">&#65279;</span></p></blockquote><blockquote><p>Exemplos:</p><p>18 2 = 10010 = 1+0+0+1+0 = 2. O código é 2.</p><p>107 5 = 412 = 4+1+2 = 7. O código é 7.</p></blockquote><p>Veja mais sobre bases N-árias aqui: https://pt.wikipedia.org/wiki/Convers%C3%A3o_de_base_num%C3%A9rica</p><p><br/></p><p>Ajude Kinho e Deyene a resolver este Big Problem. Escreva o algoritmo que descobre qual o código para cada chave e base recebida.</p>
Formato da entrada: <p>A entrada contém vários casos testes.</p><p>Cada caso teste é composto por uma linha contendo a chave K e a base B.</p><p>A entrada termina quando não houver mais linhas para processar.</p><p><br/></p><h3>Restrições</h3><p>0&lt;=K&lt;=1000000000000000000</p><p>2&lt;=B&lt;=1000000000000000000</p>
Formato da saída: <p>Para cada caso teste, seu programa deve imprimir apenas o número correspondente a cada chave e base recebida.</p>
