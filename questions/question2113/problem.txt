Título: Critério de Sassenfeld
Topicos do problema: array; acumulador; métodos numéricos; repetição; array bidimensional; 
Descrição: <p style="color: rgb(0, 0, 0);background-color: rgb(255, 255, 255);">Quando se utiliza um algoritmo iterativo para solução de sistemas lineares, é importante verificar se a convergência será garantida independentemente dos valores adotados no chute inicial. Um dos critérios verificados é o Critério de Sassenfeld.</p><p style="color: rgb(0, 0, 0);background-color: rgb(255, 255, 255);">Este critério consiste em calcular um fator alfa para cada linha da matriz do sistema de equações lineares. Se o máximo valor em módulo, dentre os alfas calculados, for menor que 1 (um) a matriz passou no critério e o sistema linear converge independente do chute inicial.</p><p style="color: rgb(0, 0, 0);background-color: rgb(255, 255, 255);">Para calcular o alfa da primeira linha você deve somar os elementos da linha 1 (<span>em módulo), exceto o elemento da diagonal principal, e dividir a soma assim obtida pelo valor do elemento da diagonal principal (em módulo).</span></p><p style="color: rgb(0, 0, 0);background-color: rgb(255, 255, 255);"><span>Para as próximas linhas, o cálculo dos alfas deverá obedecer o cálculo a seguir:</span></p><p style="color: rgb(0, 0, 0);background-color: rgb(255, 255, 255);"><img class="" src="https://www.thehuxley.com/api/v1/problems/image/ff2e11735731e92f8284c0a1ba7e84fee560d6401.png"/><span><br/></span></p><p style="color: rgb(0, 0, 0);background-color: rgb(255, 255, 255);"><span>Mais informações podem ser obtidas em:</span></p><p style="color: rgb(0, 0, 0);background-color: rgb(255, 255, 255);"><span>https://sites.google.com/a/lccv.ufal.br/william-cn/Apostila_Dezembro_2009.pdf?attredirects=0<br/></span></p><p style="color: rgb(0, 0, 0);background-color: rgb(255, 255, 255);"><span><br/></span></p><p style="color: rgb(0, 0, 0);background-color: rgb(255, 255, 255);"><span>Neste problema, dada uma matriz quadrada, você deve informar se ela satisfaz ou não esse critério e imprimir o conjunto de alfas calculados para a matriz dada.</span></p>
Formato da entrada: <p><span style="color: rgb(0, 0, 0);background-color: rgb(255, 255, 255);float: none;">Um valor inteiro que representa a dimensão da matriz quadrada, seguido pelos elementos da matriz.</span><br/></p>
Formato da saída: <p><span style="color: rgb(0, 0, 0);background-color: rgb(255, 255, 255);float: none;">Você deve imprimir SIM se o critério for satisfeito e NAO caso contrário. Em qualquer um dos casos o string é seguido de um final de linha.</span><br/></p><p><font color="#000000">Logo abaixo do SIM/NAO você deve imprimir os alfas calculados para a matriz dada, com duas casas decimais, separados por espaço. Não incluir espaço após o último alfa, no lugar de espaço, inclua uma nova linha.</font></p>
