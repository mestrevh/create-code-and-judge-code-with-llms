Título: TeleZika
Topicos do problema: 
Descrição: <p>Um vírus de celular espalha-se automaticamente, infectando, a cada segundo, os celulares dos contatos de quem já está infectado.<br/>O objetivo do problema é descrever o processo de infestação a partir de um primeiro celular infectado. Especificamente, queremos saber quantos celulares estão infectados a cada segundo<br/></p>
Formato da entrada: <p>A entrada é constituída por C casos 0..C-1, cada um correspondendo a um cenário de espalhamento do vírus.<br/><br/>Cada caso possui a forma<br/><br/>N<br/>S<br/>K[0] A[0] A[1] ... A[K[0]]<br/>K[1] A[0] A[1] ... A[K[1]]<br/>...<br/>K[N-1] A[0] A[1] ... A[K[N-1]]<br/>//linha em branco<br/><br/>onde <br/><br/>N (0&lt;N&lt;=10000) = Quantidade de celulares.<br/>S (0&lt;=S&lt;N) = O primeiro celular infectado.<br/>K[j] = quantidade de contatos do celular j.<br/>A[0] ... A[K[j]] = os contatos do celular j.<br/><br/></p>
Formato da saída: <p>Para cada caso 0...C-1 na entrada, o programa deve imprimir<br/><br/>caso C: t[0] t[1] t[2] ... t[n]<br/>//linha em branco<br/><br/>onde <br/><br/>t[s] = quantidade de celulares infectados cumulativamente no segundo s, ou seja, tn é a quantidade máxima de celulares infectados.<br/><br/>Note que no segundo 0 apenas o celular S está infectado.<br/></p>
