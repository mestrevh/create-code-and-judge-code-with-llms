Título: Destruindo células
Topicos do problema: repetição; array bidimensional; recursão; 
Descrição: <p>Ambrósio precisa eliminar alguns elementos de uma tabela. Como Ambrósio é preguiçoso, lhe pediu para fazer um programa que dado um array bidimensional e a quantidade de eliminações, destrua as células da tabela. </p><p>As células podem ter valor <b>1 (Viva) ou 0 (Morta)</b>. Se a célula estiver viva, ela pode ser destruída. </p><p>Caso uma célula seja destruída, troque o valor dela de <b>1 para 0.</b><br/></p><p>Note que se o número de linhas for par, você deve eliminar células linha por linha, da esquerda para a direita e de cima para baixo.</p><p>Já se o número de linhas for ímpar você deve eliminar células coluna por coluna, de cima para baixo e da esquerda para a direita.</p><p>Observe também que pode haver mais eliminações do que células <span style="color: rgb(147, 149, 152);background-color: rgb(255, 255, 255);float: none;">disponíveis </span>para destruir, sendo assim, irão restar eliminações.</p>
Formato da entrada: <p>Você irá receber dois inteiros <b>L</b> e <b>C ( 1&lt;= L, C &lt;= 1000)</b>, representando respectivamente linhas e colunas. </p><p>Logo depois você receberá <b>L</b> linhas contendo <b>C</b> elementos cada.</p><p>Após isso, você irá receber um outro inteiro <b>D (0 &lt;= D &lt;= 10^6) </b>que representa a quantidade de elementos que podem ser destruídos.</p><p>Exemplo:</p><p><br/></p><p>2 4</p><p>0 1 0 1</p><p>1 1 0 1</p><p>3</p>
Formato da saída: <p>O array bidimensional impresso, seguido pela quantidade de eliminações que restaram, e a quantidade de células que não foram destruídas, da seguinte maneira:</p><p><br/></p><p>0 0 0 0</p><p>0 1 0 1</p><p>0 2<br/></p>
