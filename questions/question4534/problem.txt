Título: Inversões
Topicos do problema: dividir e conquistar; 
Descrição: <p><span style="color: rgb(0, 0, 0);background-color: rgb(255, 255, 255);float: none;">Dada uma sequência de elementos <i>r</i>, uma inversão é um par (</span><i style="color: rgb(0, 0, 0);background-color: rgb(255, 255, 255);">i</i><span style="color: rgb(0, 0, 0);background-color: rgb(255, 255, 255);float: none;">,<span> </span></span><i style="color: rgb(0, 0, 0);background-color: rgb(255, 255, 255);">j</i><span style="color: rgb(0, 0, 0);background-color: rgb(255, 255, 255);float: none;">) tal que <span> </span></span><i style="color: rgb(0, 0, 0);background-color: rgb(255, 255, 255);">i</i><span style="color: rgb(0, 0, 0);background-color: rgb(255, 255, 255);float: none;">&lt;</span><i style="color: rgb(0, 0, 0);background-color: rgb(255, 255, 255);">j<span> </span></i><span style="color: rgb(0, 0, 0);background-color: rgb(255, 255, 255);float: none;">e<span> </span></span><i style="color: rgb(0, 0, 0);background-color: rgb(255, 255, 255);">r</i><span style="color: rgb(0, 0, 0);background-color: rgb(255, 255, 255);float: none;">[</span><i style="color: rgb(0, 0, 0);background-color: rgb(255, 255, 255);">i</i><span style="color: rgb(0, 0, 0);background-color: rgb(255, 255, 255);float: none;">] &gt;<span> </span></span><i style="color: rgb(0, 0, 0);background-color: rgb(255, 255, 255);">r</i><span style="color: rgb(0, 0, 0);background-color: rgb(255, 255, 255);float: none;">[</span><i style="color: rgb(0, 0, 0);background-color: rgb(255, 255, 255);">j</i><span style="color: rgb(0, 0, 0);background-color: rgb(255, 255, 255);float: none;">]. Ou seja, uma inversão é nada mais do que um par de elementos que está fora da ordem na lista<span> </span></span><i style="color: rgb(0, 0, 0);background-color: rgb(255, 255, 255);">r</i><span style="color: rgb(0, 0, 0);background-color: rgb(255, 255, 255);float: none;">.</span><br/></p><p><span style="color: rgb(0, 0, 0);background-color: rgb(255, 255, 255);float: none;">O objetivo deste problema é projetar um algoritmo com complexidade O(n log n) para computar o número de inversões de uma sequência.</span></p>
Formato da entrada: <p>A quantidade n de elementos da sequência, em uma linha.</p><p>Todos os n elementos, um em cada linha.</p>
Formato da saída: <p>A quantidade de inversões na sequência de entrada.</p>
