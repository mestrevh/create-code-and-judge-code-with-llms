Título: Critério das linhas
Topicos do problema: array; métodos numéricos; repetição; array bidimensional; 
Descrição: <p>Quando se utiliza um algoritmo iterativo para solução de sistemas lineares, é importante verificar se a convergência será garantida independentemente dos valores adotados no chute inicial. Um dos critérios verificados é o Critério das Linhas.</p><p>Este critério consiste em calcular um fator alfa para cada linha da matriz do sistema de equações lineares. Se o máximo valor em módulo, dentre os alfas calculados, for menor que 1 (um) a matriz passou no critério e o sistema linear converge independente do chute inicial.</p><p>Para calcular cada alfa você deve somar os elementos de cada linha (<span>em módulo), exceto o elemento da diagonal principal, e dividir a soma assim obtida pelo valor do elemento da diagonal principal (em módulo).</span></p><p><span>Neste problema, dada uma matriz quadrada, você deve informar se ela satisfaz ou não esse critério.</span></p>
Formato da entrada: <p>Um valor inteiro que representa a dimensão da matriz quadrada, seguido pelos elementos da matriz.</p>
Formato da saída: <p>Você deve imprimir SIM se o critério for satisfeito e NAO caso contrário. Em qualquer um dos casos o string é seguido de um final de linha.</p>
