Título: Matriz escada
Topicos do problema: obi; array bidimensional; 
Descrição: <p>Jo&atilde;ozinho est&aacute; aprendendo sobre matrizes. Hoje ele aprendeu como deixar matrizes na forma escada, e est&aacute; exercitando. Para ajud&aacute;-lo, voc&ecirc; deve escrever um programa que determine se o resultado dele realmente est&aacute; no formato correto.</p>
<p>Uma matriz est&aacute; na forma escada quando, para cada linha, as condi&ccedil;&otilde;es a seguir forem satisfeitas:</p>
<p>i) Se a linha s&oacute; possuir zeros, ent&atilde;o todas as linhas abaixo desta tamb&eacute;m s&oacute; possuem zeros.</p>
<p>ii) Caso contr&aacute;rio, seja X o elemento diferente de zero mais &agrave; esquerda da linha; ent&atilde;o, para todas as linhas abaixo da linha de X, todos os elementos nas colunas &agrave; esquerda de X e na coluna de X s&atilde;o iguais a zero.</p>
<div><br /></div>
Formato da entrada: <p>A primeira linha possui dois inteiros N e M, as dimens&otilde;es da matriz. Cada uma das N linhas seguintes cont&eacute;m M inteiros n&atilde;o-negativos, os elementos da matriz.</p>
<p>Considere:</p>
<p>1 &lt;= N &lt;= 500 e 1&nbsp;&lt;=&nbsp;M&nbsp;&lt;=&nbsp;500.</p>
<p>&nbsp;Cada elemento da matriz est&aacute; entre 0 e 100000</p>
Formato da saída: <p>Seu programa deve produzir uma &uacute;nica linha, contendo o caractere &lsquo;S&rsquo; caso a matriz esteja no formato escada, ou &lsquo;N&rsquo;, caso contr&aacute;rio.</p>
<p>N&atilde;o coloque nenhum final de linha depois do caractere.</p>
