Título: Valor máximo (implementado em lista)
Topicos do problema: repetição; função; 
Descrição: <p style="color: rgb(0, 0, 0);background-color: rgb(255, 255, 255);"><span style="color: rgb(0, 0, 0);"><b style="color: rgb(0, 0, 0);">[DESCRIÇÃO]</b></span></p><p style="color: rgb(0, 0, 0);background-color: rgb(255, 255, 255);"><span style="color: rgb(0, 0, 0);">Crie um programa que tenha apenas uma função, além do programa principal, que receberá como parâmetros </span><span style="color: rgb(0, 0, 0);"><font color="#000000">uma sequência de inteiros <b><i>L</i></b> e um natural<span> </span><b><i>n (n&gt;=1)</i></b>, que representam uma sequência <b><i>L<span> </span></i></b>de tamanho<span> </span><b><i>n</i></b>.</font></span><span style="color: rgb(0, 0, 0);"> A função deverá devolver um valor inteiro indicando o maior valor de<span> </span><b><i>L</i></b>.</span><br/></p><p style="color: rgb(0, 0, 0);background-color: rgb(255, 255, 255);"><span style="color: rgb(0, 0, 0);background-color: rgb(255, 255, 255);float: none;"><b>[ENTRADA]</b></span></p><p style="color: rgb(0, 0, 0);background-color: rgb(255, 255, 255);"><span style="color: rgb(0, 0, 0);background-color: rgb(255, 255, 255);float: none;">(a) a primeira linha da entrada contém o tamanho <b style="color: rgb(0, 0, 0);background-color: rgb(255, 255, 255);"><i>n</i></b><span> </span>da sequência; (b) as próximas <b style="color: rgb(0, 0, 0);background-color: rgb(255, 255, 255);"><i>n</i></b> linhas representam os itens que compõem a sequência.</span></p><p style="color: rgb(0, 0, 0);background-color: rgb(255, 255, 255);"><span style="color: rgb(0, 0, 0);"><b style="color: rgb(0, 0, 0);">[SAÍDA]</b><span style="color: rgb(0, 0, 0);background-color: rgb(255, 255, 255);float: none;"> </span></span></p><p style="color: rgb(0, 0, 0);background-color: rgb(255, 255, 255);"><span style="color: rgb(0, 0, 0);">(a) o programa principal deverá imprimir um valor inteiro indicando o valor máximo de<span style="color: rgb(0, 0, 0);background-color: rgb(255, 255, 255);"> </span><b style="color: rgb(0, 0, 0);background-color: rgb(255, 255, 255);"><i>L</i></b> de acordo com o valor devolvido pela função. Note que existe a possibilidade de uma sequência apenas com valores negativos.</span><br/></p><p style="color: rgb(0, 0, 0);background-color: rgb(255, 255, 255);"><b style="color: rgb(0, 0, 0);background-color: rgb(255, 255, 255);">[RESTRIÇÕES]</b></p><p style="color: rgb(0, 0, 0);background-color: rgb(255, 255, 255);"><span style="color: rgb(0, 0, 0);background-color: rgb(255, 255, 255);float: none;">(a) a leitura da entrada deve ser feita no programa principal; <span style="color: rgb(0, 0, 0);background-color: rgb(255, 255, 255);float: none;">(b) a sequência deve ser armazenada em uma lista; (c) não usar append(), a lista deve ser criada já com o tamanho necessário;</span> (d) não usar variáveis globais; (e) a função deve devolver um inteiro.</span></p>
