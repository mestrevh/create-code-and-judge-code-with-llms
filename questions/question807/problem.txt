Título: Tradução de Texto
Topicos do problema: multiplas entradas; decisão; repetição; caractere; string; dicionário; 
Descrição: <title></title><p lang="pt-PT"><font color="#000000"><font face="Times New Roman, serif"><font size="2">Escreva um programa que leia os dados de um dicionário qualquer e faça a tradução de um texto de uma língua para a outra de acordo com o dicionário fornecido.</font></font></font></p><p lang="pt-PT"><br/></p>
Formato da entrada: <title></title><p lang="pt-PT"><font color="#000000"><font face="Times New Roman, serif"><font size="2">Uma linha contendo um número inteiro (N) de palavras do dicionário. Em seguida, as N linhas que representam o conteúdo do dicionário. Em cada uma destas linhas, há uma palavra seguida dos caracteres ‘ =&gt;’ (espaço, igual, maior e espaço) e da palavra para a qual ela deve ser traduzida.</font></font></font></p><p lang="pt-PT"><font color="#000000"><font face="Times New Roman, serif"><font size="2">Após carregar os dados do dicionário, o programa deve ler uma sequência de linhas de qualquer tamanho. Estas linhas devem ser traduzidas conforme os dados do dicionário. O programa acaba quando fo rencontrada uma linha que contém apenas o caractere ‘*’(asterisco).</font></font></font></p><p lang="pt-PT"><br/></p>
Formato da saída: <title></title><p lang="pt-PT"><font color="#000000"><font face="Times New Roman, serif"><font size="2">Para cada linha (frase a ser traduzida) da sequência de linha(s)fornecida(s) na entrada, deve ser impressa uma linha com a tradução correspondente.</font></font></font></p><p lang="pt-PT"><br/></p>
