Título: Critério das colunas
Topicos do problema: array; métodos numéricos; repetição; array bidimensional; 
Tempo limite de execução: 1 segundo(s)
Descrição: <p style="color: rgb(0, 0, 0);background-color: rgb(255, 255, 255);">Quando se utiliza um algoritmo iterativo para solução de sistemas lineares, é importante verificar se a convergência será garantida independentemente dos valores adotados no chute inicial. Um dos critérios verificados é o Critério das Colunas.</p><p style="color: rgb(0, 0, 0);background-color: rgb(255, 255, 255);">Este critério consiste em calcular um fator alfa para cada coluna da matriz do sistema de equações lineares. Se o máximo valor em módulo, dentre os alfas calculados, for menor que 1 (um) a matriz passou no critério e o sistema linear converge independente do chute inicial.</p><p style="color: rgb(0, 0, 0);background-color: rgb(255, 255, 255);">Para calcular cada alfa você deve somar os elementos de cada coluna (<span>em módulo), exceto o elemento da diagonal principal, e dividir a soma assim obtida pelo valor do elemento da diagonal principal (em módulo).</span></p><p style="color: rgb(0, 0, 0);background-color: rgb(255, 255, 255);"><span>Neste problema, dada uma matriz quadrada, você deve informar se ela satisfaz ou não esse critério.</span></p>
Formato da entrada: <p><span style="color: rgb(0, 0, 0);background-color: rgb(255, 255, 255);float: none;">Um valor inteiro que representa a dimensão da matriz quadrada, seguido pelos elementos da matriz.</span></p>
Formato da saída: <p><span style="color: rgb(0, 0, 0);background-color: rgb(255, 255, 255);float: none;">Você deve imprimir SIM se o critério for satisfeito e NAO caso contrário. Em qualquer um dos casos o string é seguido de um final de linha.</span><br/></p>
