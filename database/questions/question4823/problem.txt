Título: Desenvolvendo o Code Crush
Topicos do problema: array; acumulador; simulação de jogos; array bidimensional; busca; 
Tempo limite de execução: 1 segundo(s)
Descrição: <p>Você foi contratado por uma grande empresa de jogos para ajudar no desenvolvimento do Code Crush, a nova versão do clássico jogo de combinar doces, Candy Crush. Sua missão é programar uma parte essencial: o sistema de pontuação do tabuleiro!</p><p>O jogo representa o tabuleiro como uma matriz quadrada de tamanho <code>a x a</code>, onde cada número indica o tipo de doce. Existem <code>b</code> tipos diferentes de doces, numerados de <code>1</code> até <code>b</code>.</p><p>Durante os testes, o time de design decidiu que o jogo deve reconhecer sequências de exatamente três doces iguais, que podem aparecer nas seguintes direções:</p><p><br/></p><ul><li><p>Linha (horizontal) → vale 1 ponto</p></li><li><p>Coluna (vertical) → vale 2 pontos</p></li><li><p>Diagonal da esquerda para a direita → vale 3 pontos</p></li><li><p>Diagonal da direita para a esquerda → vale 4 pontos</p></li></ul><p><br/></p><p><span>Você deve escrever um programa que leia o tamanho da matriz, a quantidade de tipos de doces e os valores da matriz, e então </span><strong>calcule a pontuação total e o número de sequências encontradas</strong><span>.</span></p><p><span><br/></span></p><p></p><p style="color: rgb(147, 149, 152);background-color: rgb(255, 255, 255);">Observação: Nenhuma sequência será maior do que três doces — o time de balanceamento não deixou!</p><p></p>
Formato da entrada: <p></p><p>Um inteiro <code>a</code>, representando o tamanho da matriz (<code>a x a</code>);</p><p></p><p></p><p>Um inteiro <code>b</code>, representando a quantidade de tipos de doces;</p><p></p><p></p><p>Em seguida, <code>a*a</code> inteiros representando o tabuleiro do jogo.</p><p></p>
Formato da saída: <p>Seu programa deve exibir uma mensagem personalizada, conforme o desempenho do jogador:</p><p><br/></p><p><span>1. Se não houver sequências:</span></p><ul><li>&#34;Poxa, voce nao pontuou :(&#34;</li></ul><p><br/></p><p>2. Se a pontuação total for 1:&#13;&#13;<br/></p><ul><li>&#34;Voce so fez um ponto :/&#34;</li></ul><p><br/></p><p>3. Caso contrário:</p><ul><li>&#34;Voce fez X pontos e foram registradas Y sequencias :)&#34;<br/><br/></li></ul>
