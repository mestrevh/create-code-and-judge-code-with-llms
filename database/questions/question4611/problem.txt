Título: Mais populares da rede
Topicos do problema: array; repetição; 
Tempo limite de execução: 1 segundo(s)
Descrição: <p>Você finalmente conseguiu seu primeiro emprego como programador. Bem, na verdade, o título do cargo é Social Media Analysis Developer, mas você precisa programar, então vamos dizer que é um cargo de programador.<br/></p><p>Neste trabalho, uma de suas funções é determinar estatísticas de usuários das redes sociais. A primeira tarefa é determinar estatísticas de seguidores. Para isso você recebe uma lista de usuários seguidos por cada usuário e então precisa determinar duas coisas: qual é o usuário com o maior número de seguidores e qual o percentual de usuários que seguem cada um dos usuários (o próprio usuário é exlcluído para o cálculo do percentual de modo que se todos os demais seguirem ele, ele terá 100% de seguidores).<br/></p>
Formato da entrada: <p>A primeira linha da entrada contém um número inteiro n (1 &lt; n &lt; 101) indicando o número de usuários da rede. As n linhas seguintes contém um número inteiro k (0 &lt;= k &lt; n) indicando o número de usuários que aquele usuário segue, em seguida são apresentados k números inteiros entre 0 e n-1 indicando os índices da lista correspondentes aos usuários seguidos por ele.<br/></p><p>Assim, por exemplo, se a entrada for:</p><pre>4&#10;<span>2 1 2&#10;1 0&#10;3 0 1 3&#10;1 0</span></pre><p><span><br/></span>Significa que existem quatro usuários na rede, o primeiro deles (de índice 0) segue dois outros usuários, o de índice 1 e índice 2. O segundo usuário (de índice 1) segue apenas o usuário de índice 0. O terceiro usuário (de índice 2) segue todos os três outros usuários e, finalmente, o quarto usuário (de índice 3) segue apenas o de índice 0.</p><p>Você pode assumir que um usuário nunca segue ele mesmo.<br/></p>
Formato da saída: <p>A saída é composta por uma linha com um número inteiro indicando o usuário com mais seguidores (caso haja empate, apenas aquele com menor índice deve ser exibido), seguida por n linhas indicando o percentual de seguidores de cada um dos usuários com duas casas decimais.<br/></p>
