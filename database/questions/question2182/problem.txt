Título: Meu produto escalar
Topicos do problema: repetição; 
Tempo limite de execução: 1 segundo(s)
Descrição: <p><span>Dado dois vetores  v<sub style="color: rgb(0, 0, 0);background-color: rgb(255, 255, 255);">1</sub>=(x<sub style="color: rgb(0, 0, 0);background-color: rgb(255, 255, 255);">1</sub>,x<sub style="color: rgb(0, 0, 0);background-color: rgb(255, 255, 255);">2</sub>,...,x<sub style="color: rgb(0, 0, 0);background-color: rgb(255, 255, 255);">n</sub>) e v<sub style="color: rgb(0, 0, 0);background-color: rgb(255, 255, 255);">2</sub>=(y<sub style="color: rgb(0, 0, 0);background-color: rgb(255, 255, 255);">1</sub>,y<sub style="color: rgb(0, 0, 0);background-color: rgb(255, 255, 255);">2</sub>,...,y<sub style="color: rgb(0, 0, 0);background-color: rgb(255, 255, 255);">n</sub>). </span><span>O produto escalar desses vetores é um número único, calculado como x<sub style="color: rgb(0, 0, 0);background-color: rgb(255, 255, 255);">1</sub>y<sub style="color: rgb(0, 0, 0);background-color: rgb(255, 255, 255);">1</sub>+x<sub style="color: rgb(0, 0, 0);background-color: rgb(255, 255, 255);">2</sub>y<sub style="color: rgb(0, 0, 0);background-color: rgb(255, 255, 255);">2</sub>+...+x<sub style="color: rgb(0, 0, 0);background-color: rgb(255, 255, 255);">n</sub>y<sub style="color: rgb(0, 0, 0);background-color: rgb(255, 255, 255);">n</sub>.</span></p><p><span><br/></span></p><p><span>Suponha que você possa permutar as coordenadas de cada vetor como desejar. Escolha duas permutações de modo que o produto escalar de seus dois novos vetores seja o menor possível e produza esse produto escalar mínimo.</span><br/></p>
Formato da entrada: <p>Na primeira linha, tem-se um número inteiro <b>C</b> que especifica a quantidade casos de teste.</p><p><span>Para cada caso de teste, a primeira linha contém o número inteiro <b>n</b>. As próximas duas linhas contêm <b>n</b> inteiros cada, fornecendo as coordenadas de v<sub style="color: rgb(0, 0, 0);background-color: rgb(255, 255, 255);">1</sub> e v<sub style="color: rgb(0, 0, 0);background-color: rgb(255, 255, 255);">2</sub>, respectivamente.</span><br/></p>
Formato da saída: <p>Para cada caso de teste, imprima uma linha com <span>o produto escalar mínimo de todas as permutações dos dois vetores dados.</span></p>
