Descrição: <p style="text-align: justify;">Há muito tempo a civilização necessita trabalhar com números, especialmente para representar as quantidades. A história nos mostra várias formas, ou sistemas numéricos como: Sistema Egípcio, Sistema Romano, Sistema Maia, Sistema Sumério, Sistema Chinês, Sistema Indo-Arábico, entre outros. </p><p style="text-align: justify;">O sistema adotado atualmente é o Sistema Decimal Posicional, sua origem é o sistema indo-arábico, sendo aperfeiçoado com o tempo. Nele, temos a representação de DEZ símbolos (0 1 2 34 5 6 7 8 9), representando os números e de acordo com a posição desses símbolos o seu valor (peso) pode mudar. </p><p style="text-align: justify;">Contudo, existem outros sistemas numéricos que são trabalhados em diversas áreas, os mais conhecidos são: Sistema de base 10 (decimal), Sistema de base 8 (Octal), Sistema de base 16(Hexadecimal) e o mais conhecido e utilizado na informática que é o Sistema de base 2 (binário). Sendo possível um sistema se comunicar com outro através de algumas regras para conversão de bases como Decimal para Binário, Binário para Octal e assim por diante. </p><p style="text-align: justify;">Entretanto, é possível, pela lógica desses sistemas, serem criados outros como o sistema de base 5, o sistema de base 6, Sistema de Base 12, ... nestes casos, as regras de conversão devem atender a esses sistemas também.</p><p style="text-align: justify;">Uma estratégia usada é seguir a regra onde é possível converter qualquer base para Decimal, e a regra que é possível converter de Decimal para qualquer base.
