Título: Multiplicações de matrizes
Topicos do problema: lógica matemática; repetição; array bidimensional; 
Tempo limite de execução: 2 segundo(s)
Descrição: <p>Quando falamos de multiplicação de matrizes, podemos ter a multiplicação entre duas matrizes ou a multiplicação de uma matriz por um escalar. Faça um programa que possa fazer uma dessas multiplicações de acordo com a escolha do usuário.</p><p>O termo <strong style="color: rgb(33, 36, 44);background-color: rgb(255, 255, 255);"><b>multiplicação escalar</b></strong> refere-se ao produto de um número real com uma matriz. Em multiplicações escalares, cada elemento da matriz é multiplicado pelo escalar determinado como no exemplo abaixo.<br/></p><p><img src="https://www.thehuxley.com/api/v1/problems/image/95296d1962de5b0424b8d381081d54be2d9a1c72r.png"/><br/></p><p>A multiplicação de matrizes é realizada de acordo com a seguinte condição: o número de colunas da 1ª matriz deve ser igual ao número de linhas da 2ª matriz. Observe alguns modelos de matrizes que podem ser multiplicadas, considerando o formato m x n:<br/></p><p>A<sub style="color: rgb(0, 0, 0);text-align: justify;">4x3</sub> * B<sub style="color: rgb(0, 0, 0);text-align: justify;">3x1</sub><br style="color: rgb(0, 0, 0);text-align: justify;"/><br style="color: rgb(0, 0, 0);text-align: justify;"/>A<sub style="color: rgb(0, 0, 0);text-align: justify;">4x2</sub> * B<sub style="color: rgb(0, 0, 0);text-align: justify;">2x3</sub><br style="color: rgb(0, 0, 0);text-align: justify;"/><br style="color: rgb(0, 0, 0);text-align: justify;"/>A<sub style="color: rgb(0, 0, 0);text-align: justify;">1x2</sub> * B<sub style="color: rgb(0, 0, 0);text-align: justify;">2x2</sub><br style="color: rgb(0, 0, 0);text-align: justify;"/><br style="color: rgb(0, 0, 0);text-align: justify;"/>A<sub style="color: rgb(0, 0, 0);text-align: justify;">3x4</sub> * B<sub style="color: rgb(0, 0, 0);text-align: justify;">4x3</sub></p><p> Assim, o produto das matrizes A = ( aij) m x p  e B = ( bij) p x n é a matriz C = (cij) m x n em que cada elemento cij é obtido por meio da soma dos produtos dos elementos correspondentes da i-ésima linha de A pelos elementos da j-ésima coluna B. Exemplos:</p><p><img src="https://www.thehuxley.com/api/v1/problems/image/a9dbe1839901ad71b533369be5abcc28a663db36s.png"/><br/></p><p><br/></p><p><img src="https://www.thehuxley.com/api/v1/problems/image/0d6eb6249e54695daebb24d2132d6cc5dbd6f0482.png"/><br/></p><p><br/></p><p><img src="https://www.thehuxley.com/api/v1/problems/image/3573a7108c32a2a8e35dbeffa7c0c9ee19e51c823.png"/><br/></p>
Formato da entrada: <ul><li>A primeira linha da entrada indica qual multiplicação será feita, se (e)scalar ou (m)atricial.</li><ul><li>Caso a multiplicação seja por escalar:</li><ul><li>A segunda linha recebe esse escalar</li><li>Em seguida, deve ser recebido o tamanho da matriz (que pode não ser quadrada) com dois números inteiros na mesma linha</li><li>Em seguida, será recebida a matriz que será multiplicada pelo escalar em formato tabular</li></ul><li>Caso a multiplicação seja entre matrizes:</li><ul><li>Você deve receber em 2 linhas os tamanhos das matrizes (que podem não ser quadradas) com dois números inteiros em cada linha</li></ul></ul></ul><p>Obs.: Todos os valores internos das matrizes são inteiros</p>
Formato da saída: <ul><li>Caso a multiplicação seja por escalar, a saída deve ser a matriz multiplicada pelo escalar dado</li><li>Caso a multiplicação seja entre matrizes, você deve retornar o resultado da multiplicação entre as matrizes ou um erro, caso não seja possível multiplicar as matrizes.</li></ul>
