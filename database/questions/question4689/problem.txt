Título: Bubble Sort Passo a passo
Topicos do problema: array; grafo; repetição; ordenação; 
Tempo limite de execução: 3 segundo(s)
Descrição: <p>Crie um programa que recebe um vetor de tamanho variável e ordena o vetor de forma crescente. A ordenação deve ser realizada utilizando uma função para com o algoritmo bubble sort alterado. O algoritmo deve parar caso em uma das iterações não ocorra nenhuma troca.<br/></p>
Formato da entrada: <p>A entrada é uma sequencia de números. A leitura deve encerrar no fim do arquivo (EOF);<br/></p>
Formato da saída: <p>A saída deve ser a exibição do passo a passo da ordenação. A cada passagem completa do algoritmo pelo array, deve-se exibir o estado atual do array.<br/></p>
