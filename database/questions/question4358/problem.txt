Título: Sistema de Gerenciamento de Biblioteca
Topicos do problema: decisão; função; 
Tempo limite de execução: 1 segundo(s)
Descrição: <p>Você está desenvolvendo um sistema de gerenciamento de biblioteca que permite aos usuários reservarem livros. Cada livro possui uma quantidade limitada de cópias disponíveis, e o sistema deve controlar as reservas e informar ao cliente se a reserva foi bem-sucedida ou não.</p><p>O sistema deve ser capaz de lidar com múltiplas tentativas de reserva, sem permitir que o mesmo livro seja reservado mais de uma vez para um mesmo cliente. Caso o cliente tente fazer uma reserva para um livro que já está completamente reservado, o sistema deve informar que a reserva não foi bem-sucedida.</p><p>A função que você deve implementar em C é chamada fazerReservaLivro e recebe como parâmetros o identificador do livro e a primeira letra do nome do cliente que está fazendo a reserva. A função deve retornar 1 se a reserva foi bem-sucedida (ou seja, há cópias disponíveis do livro) ou 0 caso contrário.</p><p>Cada livro possui um identificador único (código) para ser identificado no sistema.Estão inicialmente disponíveis 5 livros, com os identificadores 1, 2, 3, 4 e 5.Considere que cada livro possui 2 cópias disponíveis inicialmente.</p><p>Serão feitas 5 reservas.</p><p>Além disso, o sistema deve apresentar a seguinte informação após cada reserva:</p><p>Quantidade total de cópias disponíveis do livro.</p>
Formato da entrada: <p>Serão 5 linhas. <span>Cada linha contém o identificador do livro a ser reservado e o primeiro caractere do nome da pessoa que reservou.</span></p><p><span>2 J</span></p><p><span>3 M</span></p><p><span>1 Z</span></p><p><span>2 R</span></p><p><span>2 Y</span></p>
Formato da saída: <p>Como são sempre 5 reservas, use o exemplo abaixo que foi gerado usando o exemplo dado no formato de entrada para identificar o formato de saída. Perceba que se não houver mais livros disponíveis, a mensagem muda. Veja a reserva 5 abaixo como exemplo.</p><p>Reserva 1 - Cliente: J</p><p>Livro 2.</p><p>Reserva bem-sucedida.</p><p>Copias disponiveis apos a reserva: 1</p><p><br/></p><p>Reserva 2 - Cliente: M</p><p>Livro 3.</p><p>Reserva bem-sucedida.</p><p>Copias disponiveis apos a reserva: 1</p><p><br/></p><p>Reserva 3 - Cliente: Z</p><p>Livro 1.</p><p>Reserva bem-sucedida.</p><p>Copias disponiveis apos a reserva: 1</p><p><br/></p><p>Reserva 4 - Cliente: R</p><p>Livro 2.</p><p>Reserva bem-sucedida.</p><p>Copias disponiveis apos a reserva: 0</p><p><br/></p><p>Reserva 5 - Cliente: Y</p><p>Livro 2.</p><p>Reserva mal-sucedida.</p>
