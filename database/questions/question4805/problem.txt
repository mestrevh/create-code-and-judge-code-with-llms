Formato da entrada: <h5>A primeira linha contém <b>uma letra maiúscula (A, B, C ou D)</b>, indicando a equipe que recebeu a mensagem.<br/></h5><h5>A = Alpha</h5><h5>B = Bravo</h5><h5>C = Charlie</h5><h5>D = Delta</h5><p><br/></p><h5>A segunda linha contém um <b>número N ( 1 &lt;= N &lt;= 100 )</b>, o qual indica a quantidade de letras da mensagem.</h5><p><br/></p><h5>As demais N linhas contêm <b>7 números inteiros separados por espaços, </b>ou seja, 7 x N números. Cada número representa um bit e pode ser 0 ou 1.</h5><h5><span>Os bits chegam na ordem do </span><strong>mais significativo (MSB) para o menos significativo (LSB)</strong><span>.</span></h5>
Formato da saída: <p>&#13;&#13;Para cada caractere decodificado, imprimir uma linha no formato:&#13;&#13;<br/></p><pre>Letra decodificada: X</pre><p>Onde X é o caractere correspondente ao número binário na Tabela ASCII.</p><p><b>Dica: </b><span style="color: rgb(147, 149, 152);">Em C, é possível atribuir um número a um char, por exemplo: </span><span style="color: rgb(147, 149, 152);"><i>printf(&#34;%c&#34;, </i></span><span style="color: rgb(0, 0, 96);text-align: right;"><i>65) = A</i></span></p><p><br/></p><p>Ao final, deve-se imprimir uma mensagem indicando a equipe que recebeu o texto, no formato:</p><pre>Equipe Alpha recebeu a mensagem!</pre><p>&#13;&#13;(ou Bravo, Charlie, Delta, conforme o caso).&#13;&#13;<br/></p>
