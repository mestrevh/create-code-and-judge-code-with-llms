Título: Seis Strings
Topicos do problema: multiplas entradas; decisão; repetição; string; 
Tempo limite de execução: 1 segundo(s)
Descrição: <p>Dada uma string x, um valor de distância k e 5 outras strings, encontre a string com o menor valor de distância de edição (diferença) com relação a x. Se o valor da menor distância de edição encontrada for maior do que k, imprima -1 para indicar que não foi encontrada uma string suficientemente próxima de x.<br/></p>
Formato da entrada: <p>A primeira linha contém uma string x (1 ≤ len(x) ≤ 100).</p><p>A próxima linha contém um inteiro k (1 ≤ k ≤ 100).</p><p>Cada uma das próximas 5 linhas contém uma string y (len(x) = len(y)).</p>
Formato da saída: <p>Imprima o índice da string mais próxima da primeira linha (de 1 a 5) ou -1 se a distância de edição foi maior que k. Se este valor for diferente de -1, imprima o valor de distância de edição da segunda linha.</p><p><b>Observação:</b> É obrigatório definir e utilizar uma função chamada <b>diferenca(x,y)</b>, onde x é a string da primeira linha e y é a string cuja distância de edição (diferença) se deseja calcular. A função deve retornar um valor inteiro. <br/></p><p>Veja alguns exemplos abaixo:</p><p>diferenca(‘ABC’, ‘ABC’) # retorna 0</p><p>diferenca(‘ABC’, ‘ABB’) # retorna 1</p><p>diferenca(‘ABC’, ‘AAA’) # retorna 2</p><p>diferenca(‘ABC’, ‘DDD’) # retorna 3<br/></p>
