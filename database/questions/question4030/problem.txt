Título: Peça no Tabuleiro
Topicos do problema: array; decisão; repetição; array bidimensional; 
Tempo limite de execução: 1 segundo(s)
Descrição: <p>Num tabuleiro NxM, uma peça é colocada na posição (0, 0). Tal peça só pode mover-se para cima (C), baixo (B), esquerda (E), e direita (D). Cada casa do tabuleiro tem um valor inteiro. A peça só pode mover-se para uma casa adjacente se essa casa tiver valor 0. Senão, a casa para onde a peça tentou se mover terá seu valor decrementado por 1. </p><p>Por exemplo, se a casa à esquerda da peça tiver valor 5 e a peça tentar mover-se para a esquerda, a casa à esquerda ficará com valor 4 e a peça não se move.<br/><br/></p><p>- Se a peça tentar mover-se para uma casa com valor 0, ela consegue (o 0 não é decrementado).<br/><span>- A peça não consegue ir para fora dos limites do tabuleiro.</span></p><p><br/>Serão dados os movimentos da peça. Será também dada uma coordenada no tabuleiro. Se a peça conseguiu chegar nessa coordenada, seu código deve printar &#34;Cheguei com N movimentos&#34;, sendo N o número de movimentos que foram feitos para chegar na coordenada. Se a peça não conseguiu chegar na coordenada, deve-se printar &#34;Nao cheguei&#34;.</p><p><br/></p><p>OBS: Pode ser que a peça não necessite usar todos os movimentos dados para chegar na coordenada dada.</p>
Formato da entrada: <p>Primeiro, serão dadas as dimensões do tabuleiro. (Dois inteiros N e M)</p><p><br/>Depois, os valores das casas do tabuleiro. (N*M inteiros)</p><p><br/>Então, será dado o número de movimentos que a peça pode fazer. (Um inteiro K)</p><p>Logo em seguida, serão dados os movimentos da peça. (K movimentos):<br/>    - 'C' movimenta a peça para cima.<br/>    - 'D' movimenta a peça para a direita.<br/>    - 'B' movimenta a peça para baixo.<br/>    - 'E' movimenta a peça para a esquerda.</p><p>Finalmente, serão dadas as coordenadas que a peça vai tentar chegar. (Dois inteiros).</p>
Formato da saída: <p>Caso a peça chegue na coordenada dada:<br/>    - O programa deve printar &#34;<span style="color: rgb(147, 149, 152);background-color: rgb(255, 255, 255);float: none;">Cheguei com N movimentos</span>&#34;, sendo N o número de movimentos que a peça utilizou.<br/>Caso a peça não chegue na coordenada dada:<br/>    - O programa deve printar &#34;Nao cheguei&#34;.</p>
