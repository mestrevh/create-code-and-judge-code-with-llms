Título: Controle de passagens aéreas
Topicos do problema: array bidimensional; função; 
Tempo limite de execução: 1 segundo(s)
Descrição: <p>Para controlar a quantidade de passageiros por vôo, uma companhia aérea quer desenvolver um sistema de cadastro de vôos e de passageiros. Você deve fazer um programa que recebe os dados de cada vôo, seguido dos dados dos passageiros. O sistema deve prever que pode haver <i>overbooking </i>de 5% à 15%, dependendo das características do vôo. Em seguida, o sistema deve receber os dados das pessoas que compraram passagem.</p><p>No final, o sistema deve imprimir um resumo dos vôos, indicando quantas pessoas foram afetadas pelo <i>overbooking </i>em cada um deles.</p><p>Use pelo menos 3 funções para modularizar seu programa.<br/></p>
Formato da entrada: <p>O sistema deve receber os seguintes dados, nessa sequência:</p><ul><li>Dados dos vôos:</li><ul><li>O número do vôo</li><li>Destino</li><li>A quantidade máxima de passageiros</li><li>Percentual de overbooking</li><li>Deve parar de receber vôos quando o número do vôo for negativo</li></ul><li>Dados dos passageiros para check-in</li><ul><li>Nome</li><li>Número do vôo<br/></li><li>Deve parar de receber passagegiros quando o nome for &#34;sair&#34;<br/></li></ul></ul>
Formato da saída: <p><span style="color: rgb(147, 149, 152);background-color: rgb(255, 255, 255);float: none;">No final, o sistema deve imprimir um resumo dos vôos, indicando quantas pessoas foram afetadas pelo<span> </span></span><i style="color: rgb(147, 149, 152);background-color: rgb(255, 255, 255);">overbooking<span> </span></i><span style="color: rgb(147, 149, 152);background-color: rgb(255, 255, 255);float: none;">em cada um deles</span><span style="color: rgb(147, 149, 152);background-color: rgb(255, 255, 255);float: none;">.</span><br/></p><p><span style="color: rgb(147, 149, 152);background-color: rgb(255, 255, 255);float: none;">Se uma pessoa tentar comprar uma passagem para um vôo que está lotado, ou seja, excedeu a capacidade máxima mais o overbooking. Deve ser emitida uma mensagem de erro.</span></p><p><span style="color: rgb(147, 149, 152);background-color: rgb(255, 255, 255);float: none;">Se não tiver ocorrido overbooking, deve ser impressa uma mensagem.
