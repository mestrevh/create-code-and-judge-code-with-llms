Título: Ansa x Kcaj: O desafio dos triângulos 2
Topicos do problema: array; repetição; array bidimensional; recursão; 
Tempo limite de execução: 1 segundo(s)
Descrição: <p>Depois da competição internacional de Geometria Computacional (CIGC), Kcaj e Ansa encontraram um novo desafio vindo do seu amigo DJMS. Eles ainda estão tentando encontrar os triângulos (como o da questão <a href="http://thehuxley.com/problem/3059?quizId=4778" style="background-color: rgb(255, 255, 255);color: rgb(5, 173, 239);">http://thehuxley.com/problem/3059?quizId=4778</a>), mas desta vez eles podem rotacionar.</p><p>O problema é bem simples, você deve indicar quantos triângulos de cada base existe para o inteiro 0 e para o inteiro 1. Entretanto, nessa versão, além de indicar a quantidade de triângulos de cada base, os triângulos <b>(que são isósceles)</b> também podem ser &#34;girados&#34;, ou seja, existe um triângulo para cima, um para baixo, um para a direita e um para a esquerda. Por exemplo, temos um triângulo de base 3 para cada direção:</p><p>Para Cima: </p><p>   1 </p><p>1 1 1</p><p><br/></p><p>Para Baixo:</p><p>1 1 1</p><p>   1 </p><p><br/></p><p>Para Esquerda:</p><p>   1</p><p>1 1</p><p>   1</p><p><br/></p><p>Para Direita:</p><p>1</p><p><span>1 1</span></p><p>1</p><p><br/></p><p><b>OBS: A base de um triângulo é sempre um número ímpar e o triângulo é sempre isósceles.</b></p><p><b>OBS2: Para ser um triângulo, não pode conter inteiros diferentes dentro do seu perímetro.</b></p>
Formato da entrada: <p>Você receberá um inteiro <b>c </b>indicando o número de casos. </p><p>Na primeira linha de cada caso, você receberá um inteiro <b>M. </b>Depois do inteiro <b>M, </b>você receberá MxM inteiros, que representam a matriz do caso.</p>
Formato da saída: <p>Para cada caso você deverá informar a quantidade de triângulos para cada base (que possui pelo menos 1 triângulo). Caso não haja nenhum triângulo para determinada base de um inteiro 0 ou 1, você não deverá informar nada.</p><p>Caso haja pelo menos um triângulo para determinada base, a saída deverá ser a seguinte:</p><p><span style="color: rgb(40, 43, 50);background-color: rgb(255, 255, 255);float: none;">&#34;x triangulos formados por b&#34;, onde x é a quantidade de triângulos da base e b é a base utilizada. Você deverá informar primeiro sobre as bases do inteiro 1 e depois sobre as bases do inteiro 0.</span></p><p><span style="color: rgb(40, 43, 50);background-color: rgb(255, 255, 255);float: none;"><br/></span></p><p><b>OBS: Você deverá incluir uma quebra de linha ao final de toda frase</b></p><p><b>OBS: Existe a possibilidade de não haver nenhum triângulo para o inteiro 0 ou 1 (ou para os dois), então você deverá fazer o que foi especificado e não informar nada.</b></p><p><b><b style="color: rgb(147, 149, 152);background-color: rgb(255, 255, 255);">OBS: As bases informadas devem ser feitas de forma crescente, ou seja, primeiro a base 3 (se existir), depois sobre a base 5 (se existir) e assim em diante.</b><br/></b></p>
