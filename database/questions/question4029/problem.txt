Título: Maze Runner
Topicos do problema: array; decisão; repetição; array bidimensional; 
Tempo limite de execução: 1 segundo(s)
Descrição: <p>Thomas precisa encontrar um objeto escondido num labirinto. Para isso, Teresa irá guiá-lo com as direções que Thomas deve seguir a fim de chegar numa coordenada específica do labirinto.</p><p><span>- O labirinto será dado por valores inteiros, onde os valores 0 são paredes do labirinto, e 1 são locais que Thomas pode percorrer.<br/></span><span>- Em caso do comando dado por Teresa for um comando inválido (se Thomas sai do labirinto ou vai para dentro de uma parede), Thomas não executa o comando.</span></p><p>Dada uma coordenada dentro do labirinto, Thomas deve informar se passou por ela alguma vez.</p><p>OBS: Thomas sempre começa na coordenada (0, 0).</p>
Formato da entrada: <p>A primeira linha será composta de 2 números inteiros, um número X indicando o número de linhas no labirinto e um número Y indicando o número de colunas, respectivamente.</p><p><span>Então, serão dados os X*Y valores do labirinto.</span></p><p><span>Depois, será dado o número N de direções que Teresa dará para Thomas. E, então, serão dadas as N direções:<br/></span><span>- 'C' significa que Thomas deve ir para cima.<br/></span><span>- 'D' significa que Thomas deve ir para direita.<br/></span><span>- 'B' significa que Thomas deve ir para baixo.<br/></span><span>- 'E' significa que Thomas deve ir para esquerda.</span></p><p><span>Então, serão dadas as coordenadas que Thomas deve procurar.</span></p>
Formato da saída: <p>Se Thomas passou pelo menos uma vez na coordenada dada, deve ser printado &#34;ACHEI!&#34;. Caso contrário, deve ser printado &#34;NAO ACHEI!&#34;.</p>
