Título: Gerador de Códigos em Base N
Topicos do problema: backtracking; 
Tempo limite de execução: 1 segundo(s)
Descrição: <p>Implemente um programa que gere <strong>todas as sequências possíveis</strong> de tamanho <code>n</code>, onde cada posição pode assumir valores de <code>0</code> até <code>b - 1</code>, sendo <code>b</code> a base fornecida.</p><p>As combinações devem ser impressas em <strong>ordem crescente</strong>, como em uma contagem na base <code>b</code>.</p><p>Use <strong>backtracking</strong> para gerar as combinações.</p>
Formato da entrada: <p>Dois inteiros:</p><pre class="overflow-visible!"><div class="contain-inline-size rounded-2xl relative bg-token-sidebar-surface-primary"><div class="sticky top-9"><div class="absolute end-0 bottom-0 flex h-9 items-center pe-2"><div class="bg-token-bg-elevated-secondary text-token-text-secondary flex items-center gap-4 rounded-sm px-2 font-sans text-xs"></div></div></div><div class="overflow-y-auto p-4" dir="ltr"><code class="whitespace-pre!"><span><span>n </span><span><span class="hljs-selector-tag">b</span></span><span>&#10;</span></span></code></div></div></pre><ul><li><p><code>n</code>: comprimento da sequência (1 ≤ n ≤ 10)</p></li><li><p><code>b</code>: base (2 ≤ b ≤ 10)</p></li></ul>
Formato da saída: <p>&#13;&#13;Todas as combinações possíveis de comprimento <code>n</code> na base <code>b</code>, uma por linha.&#13;&#13;<br/></p>
