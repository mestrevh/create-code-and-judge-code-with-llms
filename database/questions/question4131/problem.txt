Título: Simplesmente Café
Topicos do problema: maratona; 
Tempo limite de execução: 1 segundo(s)
Descrição: <p>Em uma certa manhã, o Protagonista estava tentando pensar em problemas divertidos para a SECOMP 2022. Depois de algum tempo, ele percebeu que estava com muito sono, e foi na cozinha fazer um bom café para tomar. Magicamente, depois de tomar o seu café, ele despertou e pensou no seguinte problema:</p><p>Dado um array com N inteiros A[0], A[1], ..., A[N - 1]. Conte quantos pares de índices (i, j) satisfazem as seguintes restrições:</p><p>- (i &lt; j)</p><p>- (A[i] + A[j]) é um número ímpar.</p><p>Entretanto, como o Protagonista não sabe programar, ele pediu para que você resolvesse esse problema para ele.</p><p><br/></p><p>Nota: Para o array A = [1, 3, 2, 5, 4] existem 6 pares de índices válidos. São os pares:</p><p>(0, 2) -&gt; 1 + 2 = 3</p><p>(0, 4) -&gt; 1 + 4 = 5</p><p>(1, 2) -&gt; 3 + 2 = 5</p><p>(1, 4) -&gt; 3 + 4 = 7</p><p>(2, 3) -&gt; 2 + 5 = 7</p><p>(3, 4) -&gt; 5 + 4 = 9</p><p>Dica: Como a resposta pode ser muito grande, em linguagens como C e C++ por exemplo, a resposta pode não caber em um int, sendo necessário o uso de long long int.</p>
Formato da entrada: <p>Na primeira linha vai ter um inteiro N.</p><p>Na segunda linha vai ter N inteiros A[0], ..., A[N - 1]. A[i] é o i-ésimo elemento do array.<br/></p><p>1 &lt;= N &lt;= 100000</p><p>0 &lt;= A[i] &lt;= 100000</p>
Formato da saída: <p>Um número inteiro indicando a quantidade de pares (i, j) que satisfazem as restrições.<br/></p>
