Título: Bolhas
Topicos do problema: ordenação; 
Tempo limite de execução: 1 segundo(s)
Descrição: <p style="color: rgb(147, 149, 152);background-color: rgb(255, 255, 255);">O algoritmo de ordenação Bolha (Bubble Sort) é um algoritmo baseado em trocas que é de simples implementação. A ideia básica dele é colocar no topo de uma sequência, a cada passagem pelos seus elementos, o maior elemento de todos. Sendo assim, a cada i passagens, os últimos i elementos da sequência já estão ordenados.</p><p style="color: rgb(147, 149, 152);background-color: rgb(255, 255, 255);">Seu funcionamento consiste do seguinte: a partir do elemento atual (começa pelo primeiro elemento da sequência), comparar este com o próximo e, se o próximo for menor que o elemento atual, trocar suas posições. O algoritmo continua fazendo essas trocas, incrementando o índice do elemento atual, até o final da sequência. Essa é a primeira passagem. O algoritmo fará N-1 passagens por todos os elementos da sequência para garantir que ela está ordenada.</p><p style="color: rgb(147, 149, 152);background-color: rgb(255, 255, 255);">Ao tentar ordenar N elementos, são necessárias N-1 passagens, pois se os N-1 elementos finais já estão ordenados, com certeza o primeiro elemento da sequência também já estará.</p><p style="color: rgb(147, 149, 152);background-color: rgb(255, 255, 255);">Apesar de sua facilidade de implementação, não é um algoritmo performático, já que é O(n2): N-1 do for externo e N-1 do for interno.</p><p style="color: rgb(147, 149, 152);background-color: rgb(255, 255, 255);">Dado esse algoritmo, sua missão é implementá-lo e informar se uma sequência já está previamente ordenada; caso não esteja ordenada, informe quantas passagens e quantas trocas precisam ser feitas para ordená-la.</p>
Formato da entrada: <p>Uma sequência de números inteiros, entre colchetes, e separados por vírgula (sem espaços).</p><p>Ex.1: [1,2,3,4,5]</p><p>Ex.2: [4,8,55,67,1,2]<br/></p>
Formato da saída: <p>Imprimir: ORDENADA, caso a sequência já esteja ordenada desde o princípio. Caso contrário, imprima a quantidade de PASSAGENS e TROCAS necessárias para ordenar a sequência.</p><p>Ex.1: para a entrada [1,2,3,4,5]:</p><p>ORDENADA</p><p>Ex.2: para a entrada [4,8,55,67,1,2]:</p><p>PASSAGENS:5</p><p>TROCAS:8<br/></p>
