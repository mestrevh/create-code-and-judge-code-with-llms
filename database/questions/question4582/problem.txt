Título: Inserir Dígito
Topicos do problema: repetição; 
Tempo limite de execução: 1 segundo(s)
Descrição: <p>Lucas Brenner está trabalhando em um projeto no laboratório Brain na UFCG, que desenvolve vários tipos de aplicações para parceiros do mundo todo. Um desses parceiros é da área de logística, que envia muitos pacotes pelos correios para os seus clientes. Lucas notou que o sistema da empresa está com um problema. As etiquetas contendo o código de barras, que deve ser colado em cada pacote, está com defeito. A etiqueta possui um número n mas tem um dígito d faltando. Lucas sabe que o código de barras correto deve ser o maior número possível que possa ser derivado da inclusão do dígito d que está faltando no número n. Por exemplo, considere o número do código de barras com defeito n for 76543 e o dígito faltante d seja 4. O código de barras correto é 765443, que representa o maior número que podemos criar a partir da inserção do dígito d no número n. Podemos inserir o dígito em qualquer local. Sua tarefa é ajudar Lucas a desenvolver um algoritmo que corrija o código de barras.<br/></p>
Formato da entrada: <p>A primeira linha de cada caso de teste indica um número inteiro n (1 ≤ n &lt; 106) representando o código de barras com defeito. A segunda linha contém um número inteiro d (0 ≤ d ≤ 9) indicando o dígito que está faltando e deve ser inserido.<br/></p>
Formato da saída: <p>A primeira linha de cada caso de teste indica um número inteiro n (1 ≤ n &lt; 106) representando o código de barras com defeito. A segunda linha contém um número inteiro d (0 ≤ d ≤ 9) indicando o dígito que está faltando e deve ser inserido.<br/></p>
