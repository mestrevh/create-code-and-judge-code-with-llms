Título: Missil V-1
Topicos do problema: array bidimensional; 
Tempo limite de execução: 1 segundo(s)
Descrição: <p>Durante as partidas de battlefield V, Zaia cansou de morrer por misseis V-1 e está tentado achar uma forma de saber quando vai morrer. Muito experiente no jogo, ele tenta delimitar uma área segura no mapa para quando o míssil explodir ele saber se consegue sair vivo ou não. Para isso, ele usa uma matriz que corresponde ao mapa e ele delimita uma área, na matriz, em que ele considera segura. Sua função é descobrir se ele acertou ou não a área segura.</p><p>A matriz é feita por numero &#34;0&#34; e a área delimitada é feita por &#34;1&#34;.<br/>Uma matriz 7x7 fica assim:<br/>0 1 1 1 1 1 1<br/>0 1 0 0 0 0 1<br/>0 1 0 0 0 0 1<br/>0 1 0 0 0 0 1<br/>0 1 0 0 0 0 1<br/>0 1 1 1 1 1 1<br/>0 0 0 0 0 0 0<br/><br/>Caso a coordenada do míssil for 2x3, e o raio de explosão for 4:<br/>5 5 5 5 5 5 5<br/>5 1 0 0 0 0 5<br/>5 1 0 X 0 0 5<br/>5 1 0 0 0 0 5<br/>5 1 0 0 0 0 5<br/>5 5 5 5 5 5 5<br/>0 0 0 0 0 0 0<br/>*cuidado com os limites da matriz.<br/></p><p>X representa onde o míssil cai e os números 5 representam o raio de explosão. Como pode ser visto, a área de segurança é ultrapassada.<br/><br/>A contagem do raio é a partir do ponto onde o míssil cai.<br/>A área está errada somente quando o raio de explosão ultrapassar a área segura.<br/></p>
Formato da entrada: <p>Será dado um numero <b>N </b>que corresponde ao tamanho da matriz, e em seguida uma matriz de tamanho <u>NxN</u>. Será dado também uma coordenada na matriz, que é a posição onde a bomba irá cair na matriz e um numero <b>X</b> que é o raio de explosão.<br/></p>
Formato da saída: <p>Caso o raio de explosão ultrapasse a área delimitada na matriz, mostre &#34;Area segura incorreta&#34;. Caso o contrário, mostre &#34;Area segura correta&#34;.<br/></p>
