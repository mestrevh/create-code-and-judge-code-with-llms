Título: Arco de treinamento
Topicos do problema: array; 
Tempo limite de execução: 1 segundo(s)
Descrição: <p>Após ser derrotado no duelo contra Ekul, Aim decide se preparar para a revanche. Ele resolve subir até o dojo do Sr. Miyagi para treinar. No caminho, Aim percebe que existem bandeiras de três cores: amarelas, laranjas e vermelhas, e, entediado, decide contar quantas bandeiras de cada cor aparecem ao longo do percurso.</p><p>Você receberá um número inteiro N que representa o tamanho do caminho até o dojo do Sr. Miyagi, seguido por um array de inteiros que descreve cada trecho desse caminho. Cada valor no array indica a cor de uma bandeira:</p><ul><li>1 para bandeiras vermelhas,</li><li>2 para bandeiras laranjas,</li><li>3 para bandeiras amarelas.</li></ul><p>Sua tarefa é:</p><ol><li>Contar e imprimir quantas bandeiras de cada cor existem no caminho.</li><li>Imprimir o percurso de volta que Aim deverá seguir após terminar seu treino, que será o caminho original percorrido de trás para frente.</li></ol>
Formato da entrada: <p>&#13;&#13;Você receberá um número inteiro <b>N</b>, que representa o tamanho do array. Em seguida, receberá uma sequência de<b> N números inteiros</b> que compõem o array.<br/></p><blockquote><p>Ex.:<br/><span>5<br/></span><span>1 0 1 0 2</span></p><p><span><span id="selectionBoundary_1727629734348_06272635813282923" class="rangySelectionBoundary">&#65279;</span></span></p></blockquote>
Formato da saída: <p></p><p>A saída deverá exibir a quantidade de bandeiras de cada cor, respeitando a seguinte ordem:</p><ul><li style="text-align: left;">Amarelo = 'A'</li><li style="text-align: left;">Laranja = 'L'</li><li style="text-align: left;">Vermelho = 'V'</li></ul><p>Após exibir as quantidades, imprima o array de trás para frente.</p><blockquote><p>Ex.:<br/>Amarelo = 0<br/>Laranja = 1<br/>Vermelho = 2<br/>2 0 1 0 1<br/></p></blockquote>
