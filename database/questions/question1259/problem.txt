Título: Organizando a biblioteca
Topicos do problema: array; 
Tempo limite de execução: 1 segundo(s)
Descrição: <p style="color: rgb(0, 0, 0);background-color: rgb(255, 255, 255);">Você foi pego vagando pela Floresta Proibida pelo Sr. Filch, e agora está de detenção. Seu castigo será organizar os livros da biblioteca com base no código deles. Você, um aprendiz de bruxo muito competente, decidiu que não irá só organizar os livros colocando seus códigos em ordem crescente, mas listará aqueles que tem mesmo código.</p><p style="color: rgb(0, 0, 0);background-color: rgb(255, 255, 255);">Sua tarefa é criar um programa que leia os livros a serem ordenados seguidos de seus respectivos códigos, e os imprima na ordem pedida. Além disso, identificar aqueles de mesmo código.</p><p><h5 style="color: rgb(0, 0, 0);background-color: rgb(255, 255, 255);"><li>Os códigos são todos do tipo : identificador#número, onde identificador é uma sequência de caracteres qualquer(exceto o &#34;#&#34;, e identificador é um inteiro de 4 algarismos.    </li>       </h5><h5 style="color: rgb(0, 0, 0);background-color: rgb(255, 255, 255);"><br/></h5><h5 style="color: rgb(0, 0, 0);background-color: rgb(255, 255, 255);">              <img src="https://www.thehuxley.com/api/v1/problems/image/70ccba7cab532993cd997c25b2e22c768ce2eff5k.jpg" style="color: rgb(147, 149, 152);"/><span>       </span><span>  </span><br/></h5><div><span><br/></span></div></p>
Formato da entrada: <blockquote style="color: rgb(0, 0, 0);background-color: rgb(255, 255, 255);"><p>N</p></blockquote><ul style="color: rgb(0, 0, 0);background-color: rgb(255, 255, 255);"><li>N - número natural, indicando a quantidade de livros a serem ordenados.</li></ul><p style="color: rgb(0, 0, 0);background-color: rgb(255, 255, 255);">2N linhas, tais que :</p><blockquote style="color: rgb(0, 0, 0);background-color: rgb(255, 255, 255);"><p>L</p><p>C</p></blockquote><ul style="color: rgb(0, 0, 0);background-color: rgb(255, 255, 255);"><li>L - string, a primeira linha representará o nome do livro</li><li>C - string, a segunda linha representará o código.</li></ul>
Formato da saída: <blockquote style="color: rgb(0, 0, 0);background-color: rgb(255, 255, 255);"><p>L1</p><p>...</p><p>Ln</p></blockquote><ul style="color: rgb(0, 0, 0);background-color: rgb(255, 255, 255);"><li>Li - string, livros ordenados (identificadores em ordem crescente)</li></ul><blockquote style="color: rgb(0, 0, 0);background-color: rgb(255, 255, 255);"><p>R1</p><p>...</p><p>Rj</p></blockquote><ul style="color: rgb(0, 0, 0);background-color: rgb(255, 255, 255);"><li>Ri - string, representando os livros de mesmo código, caso existam.</li></ul>
