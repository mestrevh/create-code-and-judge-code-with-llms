Título: Lista de Produtos
Topicos do problema: ordenação topológica; 
Tempo limite de execução: 2 segundo(s)

Um produto possui quatro atributos: </span></p><p><span>1) um inteiro com o código do produto; </span></p><p><span>2) um String com o nome do produto; </span></p><p><span>3) um double com o valor do produto; </span></p><p><span>4) um inteiro com a quantidade de produtos em estoque.</span></p><p><span></span></p><p><span>Você deve:</span></p><p><span></span></p><ol><li><span>complementará a implementação do método “procurar”, que recebe o códigodo produto e retorna um String com os dados do produto (resultado do método toString da classe Produto”), caso ele seja encontrado, ou a frase “Produto Inexistente!”, caso o produto não seja encontrado.</span></li></ol><p><span></span></p><ol><li><span>Implementar o método “inserir”, que insere um novo produto no final dalista de produtos.</span></li></ol><p><span></span></p><p><span>Para que você implemente o que falta é essencial estudar o código fornecido ecompreender como a estrutura de dados com a lista de produtos (classe ListaProdutos) está implementada.</span></p><p><span></span></p><p><span>Atenção: você não pode modificar o método main! </span></p>
