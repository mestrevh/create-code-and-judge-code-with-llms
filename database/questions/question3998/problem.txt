Título: Frequência de vogais
Topicos do problema: recursão; 
Tempo limite de execução: 1 segundo(s)
Descrição: <b id="docs-internal-guid-8cefe0d9-7fff-d9a3-b41f-09a7a86daa9a"></b><p dir="ltr"><span style="color: #000000;background-color: transparent;">Define-se a frequência de um caractere dentro de uma string como a quantidade de vezes que este caractere ocorre dentro desta string.</span></p><p dir="ltr"><span style="background-color: transparent;color: rgb(0, 0, 0);">Escreva duas</span><span style="background-color: transparent;color: rgb(0, 0, 0);"> <b>funções recursivas</b></span><span style="background-color: transparent;color: rgb(0, 0, 0);">, uma que verifique se uma string contém ao menos uma vogal e outra que retorne a frequência de cada vogal dentro de uma string. A sua solução não pode utilizar nenhuma função pré-definida da linguagem de programação que você esteja empregando.</span></p>
Formato da entrada: <p><span id="docs-internal-guid-82fdb965-7fff-3ee8-4758-ecdb34f62761"><span style="color: rgb(0, 0, 0);background-color: transparent;">Uma string cujas vogais não estão acentuadas.</span></span><br/></p>
Formato da saída: <p><span id="docs-internal-guid-adc85e41-7fff-7fab-0f90-d84cf1add7d7"><span style="color: rgb(0, 0, 0);background-color: transparent;">Se a string dada na entrada não contém nenhuma vogal imprime-se “nao ha vogais”, caso contrário, imprimem-se as frequências de cada vogal seguindo a formatação dos exemplos.</span></span><br/></p>
