Título: Nomeando Problemas
Topicos do problema: fluxo máximo; grafo; grafo bipartido; 
Tempo limite de execução: 1 segundo(s)
Descrição: <p><b id="docs-internal-guid-321b10d0-f2d5-0a99-6b06-966e72fc14a5"></b></p><p dir="ltr" style="text-align: justify;background-color: #ffffff;"><span style="color: #000000;background-color: #ffffff;">Piton está organizando uma maratona de programação que contém </span><span style="color: #000000;background-color: #ffffff;"><b>N</b></span><span style="color: #000000;background-color: #ffffff;"> problemas, nomeados A, B, C, etc. Ele quer nomear o primeiro problema começando com a letra A, o segundo começando com a letra B, e etc. Porém, ele não pode simplesmente escolher palavras aleatórias para os títulos do problemas; isso não faria sentido. Para cada problema, ele pensou numa lista de nomes aceitáveis. Ajude Piton a ordenar os problemas e escolher os nomes aceitáveis para cada problema, de forma que cada problema comece com a letra correta do alfabeto.</span></p>
Formato da entrada: <p><span id="docs-internal-guid-321b10d0-f2d5-d2e6-e0c4-a1b546751b08"><span style="color: rgb(0, 0, 0);background-color: rgb(255, 255, 255);">A primeira linha da entrada dá o número de casos de teste, </span><span style="color: rgb(0, 0, 0);background-color: rgb(255, 255, 255);"><b>T</b></span><span style="color: rgb(0, 0, 0);background-color: rgb(255, 255, 255);">. </span><span style="color: rgb(0, 0, 0);background-color: rgb(255, 255, 255);"><b>T</b></span><span style="color: rgb(0, 0, 0);background-color: rgb(255, 255, 255);"> casos de teste seguirão. Cada um inicia com um </span><span style="color: rgb(0, 0, 0);background-color: rgb(255, 255, 255);"><b>N</b></span><span style="color: rgb(0, 0, 0);background-color: rgb(255, 255, 255);">, com a quantidade de problemas. As próximas </span><span style="color: rgb(0, 0, 0);background-color: rgb(255, 255, 255);"><b>N</b></span><span style="color: rgb(0, 0, 0);background-color: rgb(255, 255, 255);"> linhas terão a lista de nomes possíveis para cada um dos </span><span style="color: rgb(0, 0, 0);background-color: rgb(255, 255, 255);"><b>N</b></span><span style="color: rgb(0, 0, 0);background-color: rgb(255, 255, 255);"> problemas. Cada linha começa com </span><span style="color: rgb(0, 0, 0);background-color: rgb(255, 255, 255);"><b>K</b></span><span style="color: rgb(0, 0, 0);background-color: rgb(255, 255, 255);"><b>i</b></span><span style="color: rgb(0, 0, 0);background-color: rgb(255, 255, 255);">; o número de nomes aceitáveis para o problema </span><span style="color: rgb(0, 0, 0);background-color: rgb(255, 255, 255);"><b>i</b></span><span style="color: rgb(0, 0, 0);background-color: rgb(255, 255, 255);">, seguida da lista de nomes, separadas por espaços. Cada nome é uma string não vazia de letras. Não vão ter dois nomes iguais, e não vão ter dois nomes começando com a mesma letra para o mesmo problema.</span></span></p>
Formato da saída: <p><span id="docs-internal-guid-321b10d0-f2d7-33c8-58ae-67c84ea8e386"><span style="color: rgb(0, 0, 0);background-color: rgb(255, 255, 255);">Para cada caso de teste, primeiro imprima 'Case #x:', onde </span><span style="color: rgb(0, 0, 0);background-color: rgb(255, 255, 255);">x</span><span style="color: rgb(0, 0, 0);background-color: rgb(255, 255, 255);"> é o número do caso de teste. Depois disso, imprima </span><span style="color: rgb(0, 0, 0);background-color: rgb(255, 255, 255);"><b>N</b></span><span style="color: rgb(0, 0, 0);background-color: rgb(255, 255, 255);"> linhas, listando o nome dos </span><span style="color: rgb(0, 0, 0);background-color: rgb(255, 255, 255);"><b>N</b></span><span style="color: rgb(0, 0, 0);background-color: rgb(255, 255, 255);"> problemas, cada com apenas a primeira letra maiúscula. É garantido que existirá exatamente uma solução.</span></span></p>
