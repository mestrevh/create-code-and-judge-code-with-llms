Título: Diferenças no array
Topicos do problema: array; acumulador; repetição; 
Tempo limite de execução: 1 segundo(s)
Descrição: <p>E cá estamos, chegamos ao final da disciplina de programação I.</p><p>Para provar seu conhecimento, faremos uma simples questão envolvendo o que você já sabe!</p><p>Dado um array de inteiros que servirá como base e em seguida X arrays para serem comparados com esse array base, você deve mostrar 3 coisas:</p><ul><li>Qual dos X arrays é mais parecido com o array base (em caso de empate imprima a diferença do que vier primeiro), deve ser impresso a quantidade da diferença de elementos entre eles </li><li>Qual a linha (de acordo com a ordem de entrada) do array mais parecido com o array base (em caso de empate a menor linha deve ser impressa)</li><li>O array mais parecido substituido por 'x' nas posições diferentes do array base, em caso de empate imprima o que vier primeiro</li></ul>
Formato da entrada: <p>Um tamanho Y, que vai ser a quantidade de elementos do array base, seguido por Y elementos desse array (podem ser números negativos).</p><p>Um valor X, que vai ser a quantidade de arrays para serem comparados com o array base (todos esses arrays vão ser do mesmo tamanho do array base), seguido de X linhas cada uma com um array.</p><p><br/></p>
Formato da saída: <p>Você deve imprimir na seguinte formatação (os valores entre parênteses são seus resultados):<br/><br/>&#34;Menor diferenca: (valor da menor quantidade de mudanças possiveis para tornar um array igual ao base)&#34;</p><p>&#34;Linha: (indice da linha em que se encontra o array com a menor diferenca, as linhas com indice 1)&#34;</p><p>&#34;(array da menor diferenca, com um 'x' nos lugares em que são diferentes do array base)&#34;</p>
