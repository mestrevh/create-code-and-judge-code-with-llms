Título: Jogo da TFG
Topicos do problema: array; decisão; repetição; função; 
Tempo limite de execução: 1 segundo(s)
Descrição: <p>Mike gostaria de desenvolver um jogo novo na TFG. É um jogo para duas pessoas. Inicialmente existem n números a1, a2, a3, ..., an em um quadro. Em cada rodada, um jogador seleciona um número para ser apagado do quadro. O jogador 1 remove o primeiro número, em seguida, o jogador 2 apaga um outro número e, assim, sucessivamente, até que reste um último número no quadro. </p><p>O objetivo do jogador 1 é minimizar o número que for sobrar no quadro enquanto que o do jogador 2 é maximizar o número que ficará no quadro.</p><p>Sua tarefa é ajudar Mike a responder qual o número que restará no quadro, dados os objetivos dos jogadores. Assuma que cada jogador realizará a melhor jogada possível em cada rodada.</p><p>A sua solução deve ser modularizada, ou seja, usar funções.</p><p>Obs.: Apenas as funções básicas da linguagem devem ser usadas. Em caso de dúvida, consultar o professor formador.</p>
Formato da entrada: <p>A primeira linha de cada caso de teste descreve um número inteiro n (1 &lt;= n &lt;= 1000) representando a quantidade de números no quadro.</p><p>A linha seguinte descreve n números inteiros a1, a2, ..., an,  (1 &lt;= ai &lt;= 1000000), separados por um espaço.</p>
Formato da saída: <p>Para cada caso de teste, apresente a lista de números que foi colocada no  quadro inicialmente e o número que restou no quadro.</p>
