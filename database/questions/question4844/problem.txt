Título: Ordenando Alunos
Topicos do problema: ordenação; 
Tempo limite de execução: 1 segundo(s)
Descrição: <p>Você deve implementar um sistema simples para cadastro e ordenação de alunos, utilizando algoritmos clássicos de ordenação.</p><p>Cada aluno possui <b>quatro atributos</b>: nome, matrícula, idade e nota final.<br/>O sistema deve permitir a inserção de alunos e a ordenação do vetor de alunos por <strong>diferentes critérios</strong>, sendo que <strong>cada critério de ordenação deve obrigatoriamente utilizar um algoritmo específico</strong>.</p><p>Os algoritmos de ordenação que devem ser utilizados são:</p><ul><li><p><strong>Bubble Sort</strong> para ordenar por <strong>idade</strong></p></li><li><p><strong>Selection Sort</strong> para ordenar por <strong>nota</strong></p></li><li><p><strong>Insertion Sort</strong> para ordenar por <strong>matrícula</strong></p></li></ul><p>O programa deve processar comandos até que seja informado o comando de encerramento.</p>
Formato da entrada: <p>A entrada é composta por uma sequência de comandos.</p><p>Os comandos possíveis são:</p><ul><li><p><strong>1 – Cadastrar aluno</strong><br/>Após esse comando, serão fornecidos quatro valores:</p><ul><li><p>uma string representando o nome do aluno</p></li><li><p>um inteiro representando a matricula do aluno</p></li><li><p>um inteiro representando a idade do aluno</p></li><li><p>um número real representando a nota final do aluno</p></li></ul></li><li><p><strong>2 – Ordenar por idade</strong><br/>Ordena os alunos em ordem crescente de idade utilizando <strong>Bubble Sort</strong>.</p></li><li><p><strong>3 – Ordenar por nota</strong><br/>Ordena os alunos em ordem crescente de nota utilizando <strong>Selection Sort</strong>.</p></li><li><p><strong>4 – Ordenar por matrícula</strong><br/>Ordena os alunos em ordem crescente de matrícula utilizando <strong>Insertion Sort</strong>.</p></li><li><p><strong>0 – Encerrar o programa</strong></p></li></ul>
Formato da saída: <p>Após cada comando de ordenação (2, 3 ou 4), o programa deve imprimir a lista completa de alunos, <strong>um por linha</strong>, no seguinte formato:</p><p><br/></p><p>ordenado por (idade/nota/matricula):</p><p>nome matricula idade nota</p><p>...</p><p><br/></p><p>A nota deve ser impressa com duas casas decimais.</p><p>Nenhuma saída deve ser impressa para os comandos de cadastro ou encerramento.</p>
