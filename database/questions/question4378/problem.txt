Título: Soma de binário
Topicos do problema: array; recursão; 
Tempo limite de execução: 1 segundo(s)
Descrição: <p>Você encontrou um robô perdido pela rua e percebeu que o dever dele é de fazer somas matemáticas! <span>Entretanto, esse robô funciona apenas com números binários na hora de fazer a soma e também retorna um número binário.</span></p><p><span><br/></span></p><p>Para o robô funcionar você deve dar N que é o tamanho máximo de dígitos do número binário. Então você dará dois números binários de N dígitos e receberá um número binário também de N dígitos que é o resultado da soma. Note que se o resultado da soma ultrapassar um número binário de N dígitos (Exemplo: 1 0 0 + 1 0 0 = 1 0 0 0), o robô exibe a mensagem &#34;OVERFLOW&#34;.</p><p><br/></p><p>A soma de números binários ocorre da seguinte forma:</p><p>Número 1: 0 1 0 1 1</p><p>Número 2: 1 0 0 0 1</p><p>Resultado: 1 1 1 0 0</p><p><br/></p><p>A soma ocorre com o número diretamente acima dele da seguinte forma: Se ambos forem 0 o resultado é 0, se tiver 1 e 0 o resultado é 1, se ambos forem 1 o resultado vai ser 0 mas a soma a esquerda vai receber mais 1.</p><p><br/></p><p>Exemplo: Se N for 5, o número binário pode ser 1 0 1 0 0</p><p><br/></p><p>Dica: Faça a matemática da direita para a esquerda.</p>
Formato da entrada: <p>A primeira linha contém um inteiro N representando o tamanho das arrays.</p><p>As próximas duas linhas são arrays de tamanho N.</p>
Formato da saída: <p>O número binário resultado da soma ou a mensagem &#34;OVERFLOW&#34; caso o número ultrapasse o tamanho original.</p>
