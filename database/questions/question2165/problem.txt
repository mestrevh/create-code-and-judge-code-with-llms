Título: Produtos mais dispendiosos
Topicos do problema: array; estruturas; 
Tempo limite de execução: 1 segundo(s)
Descrição: <p><span style="color: rgb(0, 0, 0);">Dados os preços dos produtos e uma lista de compras queremos saber quais são os produtos <span style="color: rgb(0, 0, 0);background-color: rgb(255, 255, 255);float: none;">mais dispendiosos </span>da lista. Considere que a lista de compras poderá conter vários itens de um mesmo produto.</span><br/></p>
Formato da entrada: <p style="color: rgb(0, 0, 0);background-color: rgb(255, 255, 255);">A entrada consiste de duas sequências de dados.</p><p style="color: rgb(0, 0, 0);background-color: rgb(255, 255, 255);">A primeira é uma sequência de pares: código do produto e preço. O código do produto é um número inteiro e o preço um float. </p><p style="color: rgb(0, 0, 0);background-color: rgb(255, 255, 255);"><span>A segunda sequência é a lista de compras Cada item da lista é um par de inteiros, o primeiro é o código do produto e o segundo a quantidade do produto.</span></p><p style="color: rgb(0, 0, 0);background-color: rgb(255, 255, 255);">Para ambas as sequências, c<span style="color: rgb(0, 0, 0);background-color: rgb(255, 255, 255);float: none;">ada par é digitado numa linha. Ambas sequências</span><span style="color: rgb(0, 0, 0);background-color: rgb(255, 255, 255);float: none;"> terminam com um par com dois zeros que deverá ser ignorado.</span></p>
Formato da saída: <p></p><p style="color: rgb(0, 0, 0);background-color: rgb(255, 255, 255);"><span style="color: rgb(0, 0, 0);background-color: rgb(255, 255, 255);float: none;">Se a lista de compras não tiver itens deve se escrever &#34;nao tem compras&#34;, senão deverá ser escrito o maior dispendio com um único produto e logo depois,</span><span> para </span><span>cada produto mais dispendioso, em ordem crescente pelo código, </span><span>o código, o preço e a quantidade total do produto na lista, </span><span>seguindo o formato do exemplo.</span></p><p style="color: rgb(0, 0, 0);background-color: rgb(255, 255, 255);"><span>Valores float são escritos com duas casas decimais.</span></p>
