Título: Array está ordenado-Recursivo
Topicos do problema: 
Tempo limite de execução: 1 segundo(s)
Descrição: <p>Um array está ordenado quando todos seus elementos estão em ordem crescente, seguindo o critério:</p><p>array[i] &lt;= array[i+1]</p><p>Seu objetivo é construir um algoritmo recursivo que responde SIM se o array está ordenado e NAO se não estiver ordenado.</p><p><br/></p>
Formato da entrada: <p>Uma string com números inteiros separados por vírgula. Use a função split para transformar a entrada num array de inteiros.</p><p><br/></p><p>Ex.1:  3,1,2,4,5</p><p><br/></p><p>Ex.2:  1,4,6,7,8,9</p>
Formato da saída: <p style="color: rgb(147, 149, 152);background-color: rgb(255, 255, 255);">SIM, se o array estiver ordenado. NAO, caso contrário. Para os exemplos abaixo:</p><p style="color: rgb(147, 149, 152);background-color: rgb(255, 255, 255);"></p><p style="color: rgb(147, 149, 152);background-color: rgb(255, 255, 255);">Ex.1:  3,1,2,4,5</p><p style="color: rgb(147, 149, 152);background-color: rgb(255, 255, 255);">NAO</p><p style="color: rgb(147, 149, 152);background-color: rgb(255, 255, 255);"><br/></p><p style="color: rgb(147, 149, 152);background-color: rgb(255, 255, 255);">Ex.2:  1,4,6,7,8,9</p><p style="color: rgb(147, 149, 152);background-color: rgb(255, 255, 255);">SIM</p><p style="color: rgb(147, 149, 152);background-color: rgb(255, 255, 255);"></p>
