Título: Pesquisa sobre violência no trânsito
Topicos do problema: array; ad-hoc; decisão; repetição; 
Tempo limite de execução: 20 segundo(s)
Descrição: <p>Anualmente são divulgadas as estatísticas de trânsito no Brasil. Estas estatísticas costumam apresentar, por município, os seguintes dados: </p><ul><li><b>Nome do município;</b></li><li><b>Quantidade de veículos;</b></li><li><b>Total de acidentes com vítimas fatais;</b></li><li><b>Total de acidentes com vítimas não-fatais;</b></li><li><b>Total de acidentes sem vítimas no trânsito.</b></li></ul><p>Através dessas estatísticas podemos identificar os municípios com o trânsito mais violento. Para isso foi criado o Índice de Violência no Trânsito (IVT), calculado a partir da seguinte regra:</p><pre>(5 * no de acidentes com vítimas fatais + 3 * no de acidentes com vítimas não fatais + 1 * no de acidentes sem vítimas) / no de veículos</pre><p>Após a entrada das informações de um município, seu programa deve verificar se o usuário deseja incluir os dados para um novo município (S para SIM e N para NÃO). Caso o usuário informe N, o sistema deve encerrar a entrada de dados.</p><p>Ao término do programa o sistema deve apresentar:</p><p><b>A quantidade de veículos em todo o país;</b></p><p><b>A quantidade de acidentes (ao todo) no país;</b></p><p><b>A quantidade de acidentes com vítimas fatais no país;</b></p><p><b>Os municípios com o maior e menor número de acidentes (ao todo).</b></p><p><b>Qual a média de veículos por município;</b></p><p><b>Quais as médias de acidentes com vítimas fatais, com vítimas não fatais e sem vítimas por município; </b></p><p><b>Qual o maior e menor IVT ;</b></p><p><b>Para os munícios com o maior e menor IVT, apresentar todos os dados coletados.</b> </p><p>Obs: Utilize listas para armazenar os dados.</p>
Formato da entrada: <p style="color: rgb(0, 0, 0);background-color: rgb(255, 255, 255);"><span style="color: rgb(0, 0, 0);background-color: rgb(255, 255, 255);float: none;">A entrada deverá ser apenas os valores em ordem (nome do município, quantidade de veículos, número de acidentes fatais, não-fatais e sem vítimas, e se desejar incluir novo município). Não coloque strings em seu input (não forneça instruções de preenchimento) pois o corretor automático não os espera.</span></p><p style="color: rgb(0, 0, 0);background-color: rgb(255, 255, 255);"><span style="color: rgb(0, 0, 0);background-color: rgb(255, 255, 255);float: none;">Exemplo de entrada (ignore os comentários):</span></p><pre>Recife # nome do município</pre><pre>10000 # quantidade de veículos</pre><pre>10 # número de acidentes fatais</pre><pre>100 # número de acidentes não fatais</pre><pre>1000 # número de acidentes sem vítimas</pre><pre>S # incluir dados para novo município</pre><pre>Olinda # nome do município</pre><pre>7000 # quantidade de veículos</pre><pre>35 # número de acidentes fatais</pre><pre>70 # número de acidentes não fatais</pre><pre>700 # número de acidentes sem vítimas</pre><pre>N # incluir dados para novo município</pre>

município com menor número de acidentes + número de acidentes do município</pre><pre>8500.00 # média de veículos por municípios</pre><pre>22.50 # média de acidentes com vítimas fatais por municípios</pre><pre>85.00 #
