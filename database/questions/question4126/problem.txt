Título: Mensagem alienígena
Topicos do problema: ad-hoc; 
Tempo limite de execução: 1 segundo(s)
Descrição: <p>RECEBEMOS MENSAGENS ALIENÍGENAS! E elas são... Números?!</p><p>Infelizmente, essas mensagens estão criptografadas. Por isso, a NASA chamou você para traduzir essas mensagens.</p><p>Cada mensagem veio junto com dois inteiros <b><i>x</i></b> e <i><b>y</b></i>. Depois de muito esforço, foi descoberto que é possível mapear cada número para tabela ASCII! Dada um número <b>a</b>, ele é mapeado para tabela ASCII com: (<b><i>a</i></b> * <i><b>x</b></i> + <i><b>y</b></i>) mod 127.</p><p>Observação: mod se refere à operação modular (resto de divisão). Veja <span><a href="https://pt.wikipedia.org/wiki/Aritm%C3%A9tica_modular">https://pt.wikipedia.org/wiki/Aritm%C3%A9tica_modular</a></span><span>.</span><br/>Observação 2: A tabela ASCII é o padrão para strings na maioria das linguagens, como C. Veja <a href="https://pt.wikipedia.org/wiki/ASCII" style="background-color: rgb(255, 255, 255);">https://pt.wikipedia.org/wiki/ASCII</a>.</p><p>.</p>
Formato da entrada: <p>A primeira linha da entrada terá três inteiros <i><b>n</b></i>, <i><b>x </b></i>e <i><b>y</b></i>. Representando respectivamente, o tamanho da mensagem e os valores usados para a tradução.</p><p>Limites:<br/>1 &lt;= <i><b>n</b></i> &lt;= 10⁵<br/>0 &lt;= <i><b>y</b></i> &lt;= 1 &lt;= x &lt;= 10³</p>
Formato da saída: <p>A saída é a mensagem traduzida (como string, seguindo a tabela ASCII).</p>
