Título: Buffer circular de mensagens
Topicos do problema: array; array circular; 
Tempo limite de execução: 1 segundo(s)
Descrição: <p>Um <strong>buffer circular</strong> é uma estrutura de tamanho fixo que funciona como uma “fila”:</p><ul><li><p>quando o buffer enche, os elementos mais novos <strong>sobrescrevem</strong> os mais antigos;</p></li><li><p>o acesso é sempre feito por índices “cíclicos”.</p></li></ul><p>Você deve implementar um programa em C que simule um buffer circular de inteiros.</p>
Formato da entrada: <ol><li><p>Um inteiro <code>n</code> (1 ≤ n ≤ 100), representando o <strong>tamanho do buffer</strong>.</p></li><li><p>Um inteiro <code>m</code> (0 ≤ m ≤ 1000), representando o número de <strong>operações</strong>.</p></li><li><p>Em seguida, <code>m</code> linhas descrevendo operações, que podem ser:</p></li></ol><ul><li><p><code>ADD x</code> → adiciona o inteiro <code>x</code> ao <strong>fim</strong> do buffer.</p><ul><li><p>Se o buffer ainda não está cheio, apenas adiciona.</p></li><li><p>Se já está cheio, o elemento mais antigo é descartado e <code>x</code> entra no lugar dele.</p></li></ul></li><li><p><code>REM</code> → remove o elemento mais antigo do buffer (se estiver vazio, ignore).</p></li><li><p><code>PRINT</code> → imprime o conteúdo do buffer em <strong>ordem de chegada</strong>.</p><ul><li><p>Se estiver vazio, imprimir <code>EMPTY</code>.</p></li><li><p>Caso contrário, imprimir os elementos separados por espaço.</p></li></ul></li></ul>
Formato da saída: <p>&#13;&#13;Deve conter as saídas de todas as operações <code>PRINT</code>, na ordem em que aparecem.&#13;&#13;<br/></p>
