Título: Mensagem secreta de César - função
Topicos do problema: array; repetição; função; string; 
Tempo limite de execução: 1 segundo(s)
Descrição: <p>Os professores César e Daniella costumam trocar mensagens entre eles. Para os alunos não descobrirem o conteúdo das mensagens eles criaram um código simples para &#34;criptografar&#34; as mensagens. A criptografia de César e Dani consiste em duas regras básicas:</p><p>1 - Toda mensagem deve começar com o número 0 ou o número 1.  (chamaremos aqui de valor X)</p><p>2 - Toda mensagem deve terminar com um número inteiro qualquer sendo &gt;0 e &lt;=5.  (chamaremos aqui de valor Y)</p><p>A regra dessa criptografia é: </p><ul><li>Toda frase que começa com 0, a mensagem foi passada pela criptografia_0. (valor X)</li><li>Toda frase que começa com 1, a mensagem foi passada pela criptografia_1. (valor X)</li><li>O número final indica o tamanho do deslocamento de letras no alfabeto. (valor Y)</li></ul><p>A criptografia_<b>0</b> consiste em cada letra minúscula da frase ser substituída por outra letra do alfabeto que encontra-se deslocada para DIREITA, o número de vezes indicada no último número da frase. (o valor Y)</p><p>A c<span>riptografia_1 consiste em cada letra minúscula da frase ser substituída por outra letra do alfabeto que encontra-se deslocada para ESQUERDA, o número de vezes indicada no último número da frase. (o valor Y)</span></p><p><span>obs.: Em amas criptografias, os demais caracteres não sofrerão alterações, nem símbolos, caracteres especiais, números, letras maiúsculas, ...</span></p><p><font face="inherit">Sua missão é fazer um programa que receba a mensagem criptografada e, com função, decifrar as mensagem para exibir na tela a mensagem correta. (lembre-se que a mensagem já foi criptografada pela regra, logo para </font>decifrar<font face="inherit"> use o processo inverso, ou seja, se foi criptografada para direita, para decifrar tem que ser para esquerda, e vice-versa.</font></p><p><font face="inherit"><br/></font></p><p><font face="inherit">obs.: Leia várias frase até que a frase decifrada seja 'fim'. </font></p>
Formato da entrada: <p>uma frase qualquer criptografada</p>
Formato da saída: <p>a mensagem decifrada. (só a mensagem)</p>
