Título: Padaria Pão Bom 2
Topicos do problema: decisão; repetição; função; formatação; 
Tempo limite de execução: 1 segundo(s)
Descrição: <p dir="ltr" style="color: rgb(147, 149, 152);background-color: rgb(255, 255, 255);"><span style="color: rgb(0, 0, 0);background-color: transparent;"><span id="docs-internal-guid-268ff0e2-7fff-0e3e-96ba-ccfb57c9406b"><span style="color: rgb(0, 0, 0);background-color: transparent;">A padaria “Pão Bom” vende certa quantidade de pão francês e de broa a cada dia. Cada pãozinho custa R$0,25 e cada broa custa R$3,10. Caso o cliente pague com cartão, há um acréscimo de 2,5% no valor da compra, caso o cliente pague com dinheiro ou PIX, não há acréscimo. Ao final do dia, o dono quer saber quanto arrecadou com as vendas dos pães e broas (juntos), e quanto deve guardar numa conta investimento (10% do total arrecadado). Você foi contratado para fazer os cálculos para o dono. Com base nestes fatos, faça um programa para ler as quantidades de pães e de broas de cada venda, e depois calcular os dados solicitados.</span></span></span></p><br style="color: rgb(147, 149, 152);background-color: rgb(255, 255, 255);"/><span style="background-color: transparent;color: rgb(0, 0, 0);"><b>Observação:</b></span><span style="background-color: transparent;color: rgb(0, 0, 0);"><span> </span>É necessária a utilização de função (mais de uma se achar necessário) definida pelo programador(a) para a solução.</span>
Formato da entrada: <p><font color="#000000">A primeira linha possui um valor inteiro N que informa a quantidade de vendas do dia.</font></p><p><span style="color: rgb(0, 0, 0);background-color: rgb(255, 255, 255);float: none;">As demais linhas consistem de 2 inteiros (um para quantidade de pães e outra para a quantidade de broas) e uma letra maiúscula para representar pagamento com cartão (C), dinheiro (D) ou PIX (P).</span><br/></p>
Formato da saída: <p><span style="color: rgb(0, 0, 0);background-color: rgb(255, 255, 255);float: none;">Uma mensagem para o usuário informando o total arrecadado (precisão de 2 casas decimais) naquele dia e quanto o dono deve guardar na conta investimento (precisão de 2 casas decimais).</span><br/></p>
