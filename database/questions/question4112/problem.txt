Título: Dado Cabuloso
Topicos do problema: array; lógica matemática; ordenação; 
Tempo limite de execução: 2 segundo(s)
Descrição: <p>Ordep e Kcaj resolveram testar a sorte, eles combinaram de jogar N partidas um jogo que funciona da seguinte forma:</p><p>Cada partida será dado um X, representando o número de vezes que eles vão jogar o dado, esse dado tem 1500 faces, ou seja, temos valores de 1 até 1500 (incluindo ambos), o X será o mesmo para ambos, após isso eles jogam os dados alternadamente até ambos jogarem X vezes.</p><p><span style="color: rgb(147, 149, 152);background-color: rgb(255, 255, 255);float: none;">Caso seja colocado um valor de dado inválido, a partida é anulada e devesse printar &#34;valor fora do limite!&#34;</span><br/></p><p>Com isso, o fator principal dessa disputa é a soma do menor elemento do array + do maior elemento do array + o valor central do arrat (com o array ordenado). Note que cada jogador vai ter seu próprio fator principal.</p><p>Nesse jogo são dadas 3 condições, aquele que conseguir passar em mais condições ganha a partida.</p><p>As regras são:</p><p>1 - O fator principal ser primo.</p><p>2 - O fator principal e o maior elemento do array serem coprimos.</p><p>3 - O somatório dos dígitos do próximo primo do fator principal ser um número primo. </p><p><br/></p><p>Aquele que conseguir satisfazer mais regras vence. Caso tenha um empate, imprima &#34;empate&#34;.</p>
Formato da entrada: <p>um inteiro 1 &lt;= N &lt;= 10, Representando o número de partidas</p><p>para cada partida teremos:</p><p>um inteiro 3 &lt;= X &lt;= 23 ( X sempre ímpar), representando quantas vezes cada jogador irá jogar o dado.</p><p>após isso será lido as X vezes</p><p>um inteiro 0 &lt;= A &lt;= 1502, representando o valor do dado de Ordep</p><p><span style="color: rgb(147, 149, 152);background-color: rgb(255, 255, 255);float: none;">um inteiro 0 &lt;= B &lt;= 1502, representando o valor do dado de Kcaj</span></p>
Formato da saída: <p>Para cada partida, imprima o nome de quem ganhou seguido do fator principal, exemplo:</p><p>Caso Ordep ganhe:</p><p>&#34;Ordep V&#34;, onde V é o fator principal.</p><p>Caso Kcaj ganhe:</p><p>&#34;Kcaj V&#34;<span style="color: rgb(147, 149, 152);background-color: rgb(255, 255, 255);float: none;">, onde V é o fator principal.</span></p><p><span style="color: rgb(147, 149, 152);background-color: rgb(255, 255, 255);float: none;">Em caso de Empate:</span></p><p><span style="color: rgb(147, 149, 152);background-color: rgb(255, 255, 255);float: none;">&#34;empate&#34;</span></p><p><span style="color: rgb(147, 149, 152);background-color: rgb(255, 255, 255);float: none;">Em caso que o valor colocado seja inválido imprima:</span></p><p><span style="color: rgb(147, 149, 152);background-color: rgb(255, 255, 255);float: none;">&#34;valor fora do limite!&#34;</span></p>
