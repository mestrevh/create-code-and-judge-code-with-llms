Título: Médias diferentes com situação do aluno.
Topicos do problema: decisão; lógica matemática; 
Tempo limite de execução: 1 segundo(s)
Descrição: <p dir="ltr" style="color: rgb(0, 0, 0);background-color: rgb(255, 255, 255);"><span style="color: rgb(0, 0, 0);background-color: transparent;">Em uma escola, um professor deve realizar três avaliações por semestre. Para o cálculo da nota final, ele pode usar três diferentes métodos de cálculo de médias:</span></p><ul style="color: rgb(0, 0, 0);background-color: rgb(255, 255, 255);"><li dir="ltr" style="color: rgb(0, 0, 0);background-color: transparent;"><p dir="ltr"><span style="color: rgb(0, 0, 0);background-color: transparent;">Média aritmética (&#34;a&#34;);</span></p></li><li dir="ltr" style="color: rgb(0, 0, 0);background-color: transparent;"><p dir="ltr"><span style="color: rgb(0, 0, 0);background-color: transparent;">Média ponderada (&#34;p&#34;): nesse caso, o programa deve perguntar também os pesos de cada nota;</span></p></li><li dir="ltr" style="color: rgb(0, 0, 0);background-color: transparent;"><p dir="ltr"><span style="color: rgb(0, 0, 0);background-color: transparent;">Média harmônica (&#34;h&#34;): pode ser definida pela quantidade de notas dividida pela soma do inverso de cada nota e informa a situação dos alunos. Ex.: Se as notas forem 10, 7 e 5, a média harmônica será<span class="Apple-converted-space"> </span></span><span style="color: rgb(0, 0, 0);background-color: transparent;">3/(</span><span style="color: rgb(0, 0, 0);background-color: transparent;">1/</span><span style="color: rgb(0, 0, 0);background-color: transparent;">10</span><span style="color: rgb(0, 0, 0);background-color: transparent;">+</span><span style="color: rgb(0, 0, 0);background-color: transparent;">1/</span><span style="color: rgb(0, 0, 0);background-color: transparent;">7</span><span style="color: rgb(0, 0, 0);background-color: transparent;">+</span><span style="color: rgb(0, 0, 0);background-color: transparent;">1/</span><span style="color: rgb(0, 0, 0);background-color: transparent;">5);</span></p></li></ul><div style="color: rgb(0, 0, 0);background-color: rgb(255, 255, 255);"><font color="#000000">Faça um programa que calcula uma das três médias para um conjunto de 3 notas. O usuário deve também informar (por meio de um único caractere) qual média ele gostaria de usar.</font></div>
Formato da entrada: <ul><li>Um caractere que indica qual o tipo de média que será usada (&#34;a&#34;, &#34;p&#34; ou &#34;h&#34;)<br/></li><li>Três notas do tipo inteiro (entre 1 e 100)</li><li>Três pesos (também inteiros entre 1 e 100), caso a média escolhida seja a ponderada</li></ul>
Formato da saída: <ul><li>Se o usuário digitar um caractere inválido para identificar a média emita uma mensagem de erro (sem acentos):</li><ul><li>&#34;Escolha um tipo de media valida.&#34;</li></ul><li>O valor da média</li><li>A situação do aluno de acordo com sua média</li><ul style="color: rgb(0, 0, 0);background-color: rgb(255, 255, 255);"><li>&#34;Aprovado&#34;, se a média estiver no intervalo [70; 100]</li><li>&#34;Reprovado&#34;, se a média estiver no intervalo [0; 40] </li><li>&#34;Final&#34;, se a média estiver no intervalo (40; 70).</li><li>&#34;Média inválida&#34; em qualquer outro caso.</li></ul></ul>
