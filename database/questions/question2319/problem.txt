Título: Jogo de aposta com cartas
Topicos do problema: array; decisão; repetição; string; 
Tempo limite de execução: 1 segundo(s)
Descrição: <div class="page" title="Page 3"><div class="layoutArea"><div class="column"><p style="color: rgb(0, 0, 0);">Cresci jogando baralho!  </p><p style="color: rgb(0, 0, 0);">Além de jogar os jogos mais conhecidos, inventar novos jogos com o baralho era o mais legal. </p><p><font color="#000000">Um dos jogos é o de apostar com base nas cartas colocadas na mesa, melhor dizendo, com os valores das cartas que os jogadores colocam na mesa. Logo, os naipes não são considerados na brincadeira.  </font><font color="#000000" face="inherit">Apenas os valores das cartas são utilizados; isto é, os valores de 1 a 13, considerando as cartas do Valete, D</font><font color="#000000"><span>ama e Rei.</span></font></p><p style="color: rgb(0, 0, 0);">O jogo tem dois participantes e consiste em várias partidas. Em cada partida, cada jogador joga duas cartas. </p><p style="color: rgb(0, 0, 0);"><span>As regras para determinar quem ganha a partida são simples, baseadas nos valores das cartas de cada jogador:</span></p><ul style="color: rgb(0, 0, 0);"><li><p><span>se as duas cartas têm o mesmo valor, o jogador recebe como pontuação na partida duas vezes a soma dos valores das cartas.</span></p></li></ul><ul style="color: rgb(0, 0, 0);"><li>se os valores das duas cartas são números consecutivos (por exemplo, 12 e 13, ou, 23 e 22), o jogador recebe como pontuação na partida três vezes a soma dos valores das cartas.</li></ul><p style="color: rgb(0, 0, 0);"></p><ul style="color: rgb(0, 0, 0);"><li><span>caso contrário, o jogador recebe como pontuação na partida a soma dos valores das cartas. </span><br/></li></ul><p style="color: rgb(0, 0, 0);"></p><p style="color: rgb(0, 0, 0);"><span><br/></span></p><p style="color: rgb(0, 0, 0);"><span>Ganha a partida o jogador que tiver recebido a maior pontuação. Se houver empate, a aposta </span><span>acumula para a próxima partida.</span></p><p style="color: rgb(0, 0, 0);">A sua função é escrever um programa para verificar quem ganha o jogo, lidos os nomes dos jogadores e as cartas de cada jogador em várias partidas. </p></div></div></div>
Formato da entrada: <p style="color: rgb(0, 0, 0);"><span>A entrada é composta por uma ou mais linhas.</span></p><p style="color: rgb(0, 0, 0);"><span>Cada linha corresponde a um jogo e contém os nomes dos dois jogadores e, para cada jogador, pares de valores de cartas, alternados entre os jogadores. </span></p><p style="color: rgb(0, 0, 0);"><span>Por exemplo, para a linha de entrada abaixo:</span></p><blockquote><p style="color: rgb(0, 0, 0);"><span><i>Ianna Henrique 1 2 3 4 5 6 7 8</i></span></p></blockquote><p style="color: rgb(0, 0, 0);"><span><br/></span></p><p style="color: rgb(0, 0, 0);"><span>Tem-se que a jogadora Ianna (jogador 1) tem dois pares de cartas: 1, 2, 5 e 6. O jogador Henrique (jogador 2) tem as cartas 3, 4, 7 e 8.</span></p><p style="color: rgb(0, 0, 0);">Em cada partida, os jogadores usam um par de cartas.</p><p style="color: rgb(0, 0, 0);">Quando a linha de entrada contém a palavra &#34;sair&#34; significa que não há mais jogadores e o jogo acabou.</p><p style="color: rgb(0, 0, 0);">Quando a linha de entrada tem menos que a quantidade mínima de dados para uma partida, deve ser apresentada a mensagem <span style="color: rgb(0, 0, 0);background-color: rgb(255, 255, 255);float: none;">&#34;erro&#34;.</span></p>
Formato da saída: <p style="color: rgb(0, 0, 0);"><span>Seu programa deve produzir uma única linha, para cada entrada de dados, contendo o nome do(a) jogador(a) que venceu o jogo. Se houver empate, a linha deve conter a palavra &#34;empate&#34;.</span></p><p style="color: rgb(0, 0, 0);"><span>No caso da entrada não atender às exigências mínimas, deve-se apresentar a palavra &#34;erro&#34;. </span></p><p style="color: rgb(0, 0, 0);"><span>Ou, quando não há mais jogadores, <span style="color: rgb(0, 0, 0);background-color: rgb(255, 255, 255);float: none;">deve-se apresentar a palavra &#34;fim&#34;. </span></span></p>
