Título: Vetores (GA)
Topicos do problema: multiplas entradas; entrada formatada; lógica matemática; repetição; recursão; 
Tempo limite de execução: 1 segundo(s)
Descrição: <p>Ambrósio começou a cursar exatas, e se deparou com a temida matéria de Geometria Analítica. Seu Professor começou a ensinar vetores, e ele precisa conferir se as respostas dos seus exercícios estão corretas.</p><p>Sabendo que você sabe programar, Ambrósio solicitou que faça um programa onde serão dados vários pares de vetores <i>( Os vetores fornecidos por Ambrósio poderão ter duas ou três componentes ).</i></p><p>Para cada par de vetores, você deve calcular a soma, o produto escalar e indicar se o ângulo entre eles é Agudo, Obtuso, ou Reto.</p><p>O Ângulo é Agudo quando o produto escalar u.v é &gt; 0, é Obtuso quando o escalar é &lt; 0 e é Reto quando o seu escalar é = 0.</p><p>O vetor com duas componentes será dado da seguinte maneira:</p><p><b>u-&gt; = xi + yj</b></p><p>com três componentes será:</p><p><b>u-&gt; = xi + yj + zk</b><br/></p>
Formato da entrada: <p>Será dado um inteiro <b>C</b>, representando as componentes do vetor, seguidos pelo par de vetores <b>u e v</b>, a entrada para quando <b>C</b> for diferente de 2, e diferente de 3. Se o par de vetores tem duas componentes <b>(i, j)</b> a entrada será:<br/></p><p><b></b>2</p><p>u-&gt; = 1i + 3j</p><p>v-&gt; = 7i + -2j<br/></p><p>e se o par tem três componentes <b>(i, j, k)</b>:</p><p>3</p><p>u-&gt; = 2i + 4j + -3k</p><p>v-&gt; = 9i + 6j + 4k</p><p><b>Onde -1000 &lt; C, x, y, z &lt; 1000 </b>(x, y, z são as componentes do vetor).<br/></p><p><br/></p><p></p>
Formato da saída: <p>Usando o exemplo de entrada, a saída para vetores com duas componentes deverá ser apresentada da seguinte maneira:</p><p>u-&gt; + v-&gt; = 8i + 1j</p><p>u-&gt;.v-&gt; = 1</p><p>Agudo</p><p>Para três componentes:</p><p>u-&gt; + v-&gt; = 11i + 10j + 1k</p><p>u-&gt;.v-&gt; = 30<br/></p><p>Agudo</p><p>Onde a primeira linha é o vetor resultante da soma, a segunda linha é o resultado do produto escalar e a terceira linha representa o ângulo (Agudo, Obtuso ou Reto).<br/></p>
