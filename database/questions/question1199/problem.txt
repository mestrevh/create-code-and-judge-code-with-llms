Título: Ordenação Bolhas
Topicos do problema: ordenação; 
Tempo limite de execução: 1 segundo(s)
Descrição: <p lang="pt-BR">O algoritmo de ordenação Bolha (Bubble Sort) é um algoritmo baseado em trocas que é de simples implementação. A ideia básica dele é colocar no topo de uma sequência, a cada passagem pelos seus elementos, o maior elemento de todos.Sendo assim, a cada i passagens, os últimos i elementos da sequência já estão ordenados.</p><p lang="pt-BR">Seu funcionamento consiste do seguinte: a partir do elemento atual (começa pelo primeiro elemento da sequência), comparar este com o próximo e, se o próximo for menor que o elemento atual, trocar suas posições. O algoritmo continua fazendo essas trocas, incrementando o índice do elemento atual, até o final da sequência. Essa é a primeira passagem. O algoritmo fará N-1 passagens por todos os elementos da sequência para garantir que ela está ordenada.<span> </span></p><p lang="pt-BR">Ao tentar ordenar N elementos, são necessárias N-1 passagens, pois se os N-1 elementos finais já estão ordenados, com certeza o primeiro elemento da sequência também já estará.</p><p lang="pt-BR">Apesar de sua facilidade de implementação, não é um algoritmo performático, já que é O(n2): N-1 do for externo e N-1 do for interno.<span> </span></p><p lang="pt-BR">Dado esse algoritmo, sua missão é implementá-lo e informar se uma sequência já está previamente ordenada; caso não esteja ordenada, informe quantas passagens e quantas trocas precisam ser feitas para ordená-la.</p>
Formato da entrada: <p lang="pt-BR">Uma sequência de números inteiros, entre colchetes, e separados por vírgula (sem espaços; máximo de 50 números).</p><p lang="pt-BR"> </p><p lang="pt-BR">Ex.1:[1,2,3,4,5]</p><p lang="pt-BR">Ex.2:[4,8,55,67,1,2]</p>
Formato da saída: <p lang="pt-BR">Imprimir: ORDENADA, caso a sequência já esteja ordenada desde o princípio. Caso contrário, imprima a quantidade de PASSAGENS e TROCAS necessárias para ordenara sequência.</p><p lang="pt-BR"> </p><p lang="pt-BR">Ex.1:para a entrada [1,2,3,4,5]:</p><p lang="pt-BR">ORDENADA</p><p lang="pt-BR"> </p><p lang="pt-BR">Ex.2:para a entrada [4,8,55,67,1,2]:</p><p lang="pt-BR">PASSAGENS:5</p><p lang="pt-BR">TROCAS:8</p>
