Título: Pão Bom 2
Topicos do problema: decisão; repetição; 
Tempo limite de execução: 1 segundo(s)
Descrição: <p>A padaria “Pão Bom” vende certa quantidade de pão francês e de broa a cada dia. Cada pãozinho custa R$0,50 e cada broa custa R$2,30. Caso o cliente pague com cartão, há um acréscimo de 5% no valor da compra, caso o cliente pague com dinheiro ou PIX, não há acréscimo.</p><p>Para cada compra, o dono quer saber quanto arrecadou com a venda dos pães e broas (juntos), e quanto deve guardar numa conta investimento (10% do total arrecadado) no final do dia.</p><p>Você foi contratado para fazer os cálculos para o dono. Com base nestes fatos, faça um programa para ler as quantidades de pães e de broas de cada venda, e depois calcular os dados solicitados. As vendas se encerram quando é informado o caractere *.<br/></p><p><b>Observação:</b> É necessária a utilização de função (mais de uma se achar necessário) definida pelo programador(a) para a solução.<br/></p>
Formato da entrada: <p>Consiste de uma sequência de linhas, sendo que cada linha contém 2 inteiros (um para quantidade de pães e outra para a quantidade de broas) e uma letra maiúscula para representar pagamento com cartão (C), dinheiro (D) ou PIX (P). O sequência se encerra quando é encontrada uma linha contendo apenas o caractere *.</p>
Formato da saída: <p>Uma mensagem para o usuário informando o total arrecadado (precisão de 2 casas decimais) naquele dia e quanto o dono deve guardar na conta investimento (precisão de 2 casas decimais).<br/></p>
