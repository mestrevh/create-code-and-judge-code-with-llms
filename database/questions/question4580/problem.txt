Título: Operação Matemática
Topicos do problema: decisão; 
Tempo limite de execução: 1 segundo(s)
Descrição: <p>Raphael adora resolver problemas de lógica. Recentemente ele está tentando resolver um problemae precisa da sua ajuda.Considere três números inteiros a, b, e c, Raphael precisa identificar qual a operação matemáticaque relaciona os três números. Os números podem ser relacionados pela operação matemática desoma ou subtração de tal forma que exatamente uma das seguintes equações seja verdadeira:• a+b=c• a-b=cPor exemplo, considere os seguintes valores: a = 1, b = 2, e c = 3. Esses três números estãorelacionados pela operação matemática de soma (1 + 2 = 3). Nesse exemplo, imprima na saída ooperador +. Considere outro exemplo: a = 2, b = 9, e c = -7. Nesse caso, os três números estãorelacionados pela operação matemática de subtração (2 - 9 = -7). Nesse exemplo, imprima na saídao operador -.A sua tarefa é ajudar Raphael saber qual operação matemática relaciona os números.<br/></p>
Formato da entrada: <p>Cada caso de teste contém três linhas representando os números a, b, e c (0 ≤ a, b, c &lt; 106),respectivamente.<br/></p>
Formato da saída: <p>Para cada caso de teste, imprima a operação matemática que relaciona os números a, b e c.<br/></p>
