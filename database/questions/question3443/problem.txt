Título: A Expected Value Problem
Topicos do problema: recursão; 
Tempo limite de execução: 1 segundo(s)
Descrição: <p>Em estatística, a definição de valor esperado se refere<b> a soma de todos os valores(X * P(X)) </b>tal que:<b> </b></p><p><b>X é um evento. </b></p><p><b>P(X) é a probabilidade de que esse evento aconteça.</b></p><p><br/></p><p><img src="https://www.thehuxley.com/api/v1/problems/image/3ddb8b97f9957070cef1170fd40b40fb7edaa1539.jpeg"/><br/></p><p><br/></p><p>Considere para esse problema que:</p><p>Todos os valores possíveis de X são inteiros e que estão no intervalo fechado [1, N].</p><p>Para cada valor possível de X, P(X) = (soma de todos os valores inteiros de 1 até X) / (soma de todos os valores dos numeradores das frações).</p><p><br/></p><p>Exemplo: </p><p><span>Para N = 3.</span></p><p>EV = ((1 / 10) * 1) + ((3 / 10) * 2) + ((6 / 10) * 3)</p><p>EV = 2.50</p><p><br/></p><p>É garantido que a soma de todos os P(X) nesse problema, é sempre igual a 1.</p><p><br/></p><p>Sabendo disto, dado o valor de N, calcule o valor esperado da sequencia descrita.</p><p>O valor deve ser impresso com uma precisão de 2 casas decimais.<br/></p>
Formato da entrada: <p>Um inteiro (1 &lt;= N &lt;= 100).</p>
Formato da saída: <p>Um único número de ponto flutuante, indicando o valor esperado da sequencia descrita.</p><p>O valor deve ser impresso com uma precisão de 2 casas decimais.<br/></p>
