Título: Biblioteca Inteligente
Topicos do problema: ordenação; busca binária; 
Tempo limite de execução: 1 segundo(s)
Descrição: <p>Uma biblioteca digital armazena os códigos de identificação de seus livros em um vetor desordenado.</p><p>Seu sistema precisa:</p><ol><li><p>Ordenar os códigos em ordem crescente.</p></li><li><p>Responder consultas informando se um determinado código existe na biblioteca.</p></li></ol><p>Para cada consulta, informe:</p><ul><li><p><b>&#34;SIM&#34;</b> se o código existir.</p></li><li><p><b>&#34;NAO&#34;</b> caso contrário.</p></li></ul><p>A ordenação deve ser feita utilizando <b>Merge Sort ou Quick Sort.</b></p><p>A busca deve ser feita utilizando <strong>Busca Binária</strong>.</p>
Formato da entrada: <p><li><p>A primeira linha contém um inteiro <strong>N</strong> (1 ≤ N ≤ 10⁵), representando a quantidade de livros.</p></li><li><p>A segunda linha contém <strong>N</strong> inteiros representando os códigos dos livros.</p></li><li><p>A terceira linha contém um inteiro <strong>Q</strong> (1 ≤ Q ≤ 10⁵), representando a quantidade de consultas.</p></li><li><p>As próximas <strong>Q</strong> linhas contêm um inteiro cada, representando o código a ser buscado.</p></li></p>
Formato da saída: <p>Para cada consulta, imprima:</p><ul><li><p><b>SIM</b> se o código existir no vetor.</p></li><li><p><b>NAO </b>caso contrário.</p></li></ul><p>Cada resposta deve ser impressa em uma nova linha.</p>
