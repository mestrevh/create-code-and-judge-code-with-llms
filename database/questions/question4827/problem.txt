Título: Perdido no Labirinto
Topicos do problema: array bidimensional; backtracking; estruturas; 
Tempo limite de execução: 5 segundo(s)
Descrição: <p>Omsare foi teleportado e acabou perdido no labirinto de Murici City. Você, usando seus poderes mágicos, consegue enxergar todo o labirinto do alto. Entretanto, este labirinto possui algo especial: pontos de teleporte! Cada ponto de teleporte leva diretamente ao seu par correspondente — e sempre que existir um, há outro formando o par.<br/>A partir da posição inicial (0,0), que é sempre um espaço livre, desenvolva um programa capaz de determinar se Omsare conseguirá escapar sozinho ou se será necessário chamar reforço para resgatá-lo!</p>
Formato da entrada: <p> </p><ul><li><p>Um número inteiro <strong>N</strong>, representando o tamanho de uma matriz <code>N × N</code><br/>com <code>10 &lt; N &lt; 10³</code>;</p></li><li><p>Uma matriz <strong>A</strong> de tamanho <code>N × N</code>, onde:</p><ul><li><p><code>0</code> representa espaços livres pelos quais Omsare pode se locomover;</p></li><li><p><code>1</code> representa paredes;</p></li><li><p><code>2</code> representa pontos de teleporte.</p></li><li><p><code>3</code> representa o fim do labirinto.<br/></p></li></ul></li><li><p>Um número inteiro <strong>T</strong>, representando a quantidade de <strong>pares de teleportes</strong> do labirinto, com <code>0 &lt; T &lt; N / 2</code>.</p></li><li><p>Para cada par de teleporte <code>Ti</code>, são dados <strong>4 inteiros</strong>, representando as coordenadas das duas posições que formam o par de teleporte <code>Ti</code>.<br/><br/><b>DICA</b><b>:</b> Use struct para saber a coordenada que cada ponto de teleporte leva.<br/><b>DICA:</b> É possível fazer um array e até matrizes com struct.</p></li></ul><p></p>
