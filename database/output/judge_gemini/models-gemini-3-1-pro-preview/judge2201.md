# Modelo do judge: models/gemini-3.1-pro-preview

VEREDITO: APROVADO
COMPLEXIDADE: O(1) tempo, O(1) espaço
FEEDBACK: A implementação é excelente. O problema foi resolvido puramente com dedução matemática a partir das dimensões dadas, o que garante complexidade de tempo constante e elimina a necessidade de qualquer iteração ou simulação. O uso de `long long` protege contra qualquer possibilidade de overflow, apesar de os limites máximos (100) caberem em um inteiro normal. O código também lida perfeitamente com os casos extremos (corner cases) como quando a largura ou o comprimento é igual a 1 (L = 1 e/ou C = 1), gerando resultados precisos de acordo com a lógica. Código limpo, exato e altamente eficiente.
