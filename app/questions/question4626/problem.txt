Título: Q7 - Computadores Defeituosos
Topicos do problema: array; decisão; repetição; 
Tempo limite de execução: 1 segundo(s)
Descrição: <p>O IFPB Campus Campina Grande é uma instituição que possui diversos laboratórios e, dentre eles, os de Programação que são numerados de 1 a 4. Esses laboratórios possuem computadores para que os alunos possam utilizar durante as aulas dos professores quando essas ocorrem no laboratório. Contudo, alguns computadores apresentam problemas e/ou defeitos e muitas vezes frustam os alunos. Por isso, você foi encarregado de criar um programa que vai receber o laboratório em específico e uma sequência de 1 e 0 que indicará se o computador funciona bem ou não, junto ao seu id e exibir quantos computadores estão disponíveis para uso e quais os ID's desses computadores em questão. Cada laboratório possui uma quantidade diferente de computadores, sendo:</p><p><br/></p><p>Laboratório 1: 28 máquinas</p><p>Labortaório 2: 26 máquinas</p><p>Laboratório 3: 15 máquinas</p><p>Laboratório 4: 20 máquinas</p><p><br/></p><p>dado o número do laboratório, leia x entradas (onde x é igual ao número de máquinas do laboratório em questão) onde cada máquina irá ter um indentificador de funcionamento dado por 0 indicando que funciona e 1 indicando que não funciona, e o id desse computador.<br/><br/>OBS:</p><p><br/></p><p>1. Caso o identificador de funcionamento seja distinto de 1 ou 0, exiba a mensagem &#34;Identificador inválido&#34; e leia novamente até que seja um identificador de funcionamento válido. </p><p>2. Caso seja dado o ID de uma máquina que já foi cadastrada, exiba a mensagem &#34;ID invalido. Nao pode haver pc's iguais&#34; e leia novamente até que seja um ID válido.</p><p>3. Caso haja 0 máquinas funcionando, exiba a mensagem &#34;Sem computadores disponiveis nesse laboratorio.&#34;</p>
Formato da entrada: <p>Um número i que representa o laboratório, podendo ser 1, 2, 3 ou 4.</p><p>x entradas onde x será a quantidade de máquinas disponíveis no laboratório. Cada leitura receberá o identificador de funcionamento (1 ou 0) e o ID da máquina.</p><p><br/></p>
Formato da saída: <p>Uma mensagem &#34;Quantidade de pessoas que podem utilizar os computadores: y&#34;, onde y é o número de computadores disponíveis para uso.</p><p>uma mensagem &#34;Lista de id's:&#34; seguido de uma lista de números que serão os ID's das máquinas que funcionam. Caso não haja máquinas funcionando, exiba a mensagem &#34;Sem computadores disponiveis nesse laboratório.&#34;</p>
