Título: Sopa de Letrinhas
Topicos do problema: multiplas entradas; array; caractere; string; busca; 
Tempo limite de execução: 1 segundo(s)
Descrição: <p>Kammera Man perdeu sua criatividade e, para passar o tempo (talvez se inspirar?), ele pediu para você fazer um código que se comporta da seguinte forma:</p><p><br/></p><pre>Dado um conjunto de letras, o programa verifica quantas palavras podem ser formadas usando essas letras.&#10;Porém, como você está usando o computador de Kammera Man, você só tem espaço para digitar 5 palavras diferentes.</pre><pre>Toda vez que uma palavra é formada, as letras desta são retiradas do conjunto. Porém, se uma palavra não consegue&#10;ser formada, nenhuma das letras são retiradas do conjunto, ou seja: &#10;&#10;Somente as palavras que podem ser formadas <b>completamente</b>, retiram letras do conjunto.</pre><p><span style="color: rgb(147, 149, 152);background-color: rgb(255, 255, 255);float: none;">OBS: As strings não tem espaço incluído.</span></p><p><span style="color: rgb(147, 149, 152);background-color: rgb(255, 255, 255);float: none;">OBS2: A string <b>não precisa estar contida </b>no conjunto.</span></p><p><span style="color: rgb(147, 149, 152);background-color: rgb(255, 255, 255);float: none;"><br/></span></p><p><span style="color: rgb(147, 149, 152);background-color: rgb(255, 255, 255);float: none;">Por que será que o computador de Kammera Man só tem espaço para verificar 5 palavras?</span><br/></p>
Formato da entrada: <p>Uma string de até 29 caracteres.</p><p>Um inteiro <b>N</b> menor ou igual a 5, indicando a quantidade de palavras a serem checadas.<br/><br/><b>N</b> strings de até 29 caracteres.</p>
Formato da saída: <div><p>&#34;E possivel formar <b>x</b> palavras com esse conjunto&#34;</p><p>Sem as aspas, seguida de uma quebra de linha.</p><p>Onde <b>x</b> é o número de palavras que podem ser formadas. </p><p><br/></p></div>
