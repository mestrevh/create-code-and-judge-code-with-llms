Título: Operador ternário
Topicos do problema: lógica matemática; função; 
Tempo limite de execução: 1 segundo(s)
Descrição: <p>O operador condicional ?: permite avaliar uma expressão condicional e retornar um entre dois valores. Observe:                                        <i><b>int a = 2; int b = 5; x = a &lt; b ? a : b</b></i>. Neste caso, x vale a (2), pois a é menor que b. Já em: <span style="color: rgb(147, 149, 152);background-color: rgb(255, 255, 255);float: none;"> </span><i style="color: rgb(147, 149, 152);background-color: rgb(255, 255, 255);"><b>int a = 2 int b = 5; x = a &gt; b ? a : b</b>, x vale 5.</i>  Com base neste conceito, crie uma função que recebe três inteiros (a, b, c) e retorna <b>2 * a, se c &gt; a;</b> caso contrário, retorna <b>3 * b</b>.<br/></p>
Formato da entrada: <p>3 4 5</p>
Formato da saída: <p>6</p>
