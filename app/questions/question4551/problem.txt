Título: Contagem do insertion sort
Topicos do problema: ordenação; 
Tempo limite de execução: 1 segundo(s)
Descrição: <p>Modifique a implementação do algoritmo Insertion Sort para contabilizar o número de deslocamentos que ele realiza durante a ordenação. A única coisa que você deve imprimir é o número de deslocamentos feitos pelo algoritmo para ordenar completamente o array. Um deslocamento ocorre quando a posição de um elemento muda no array. Não desloque um elemento se não for necessário.<br/></p><p><br/></p><p><br/></p>
Formato da entrada: <p>A primeira linha deve conter um inteiro n para o número de elementos a serem ordenados.</p><p>A próxima linha contem n inteiros do array.</p><p>Exemplo:</p><pre style="color: rgb(14, 20, 30);">STDIN       Function&#10;-----       --------&#10;5           arr[] size n =5&#10;2 1 3 1 2   arr = [2, 1, 3, 1, 2]</pre>
Formato da saída: <p>Retorna apenas um inteiro correspondendo pela quantidade de trocas realizadas pelo algoritmo.</p><p>Exemplo:</p><p>4</p><pre style="color: rgb(14, 20, 30);">Iteration   Array      Shifts&#10;0           2 1 3 1 2&#10;1           1 2 3 1 2     1&#10;2           1 2 3 1 2     0&#10;3           1 1 2 3 2     2&#10;4           1 1 2 2 3     1&#10;&#10;Total                     4</pre>
