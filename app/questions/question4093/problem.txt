Título: Busca Binária Detalhada
Topicos do problema: recursão; busca; busca binária; 
Tempo limite de execução: 1 segundo(s)
Descrição: <p>O algoritmo de busca binária visa diminuir a quantidade de comparações para encontrar um dados valor k num array. No entanto, o array precisa estar ordenado para que a busca binária seja feita. O objetivo é sempre consultar o valor que está na metade do array e, caso, não tenha sucesso, procurar o valor k na metade do array que ele provavelmente estará.</p><p>Você deve implementar o algoritmo de busca binária recursiva usando Java. Apresentar a sequencia de consultas realizadas até chegar ao número procurado. Caso o número procurado não exista no array, deve ser impresso o -1 como sinal de que não existe o valor de k n array de entrada.</p>
Formato da entrada: <p>A entrada consiste de 2 linhas de valores:</p><p>k - um inteiro entre 0 e 99 consistindo do número a ser localizado no array.</p><p>n - o tamanho do array de dados a ser informado na segunda linha.</p><p>A - Array de entrada para ser realizada a busca binária. Os valores inteiros deverão estar separados poro vírgula.</p>
Formato da saída: <p>Deve ser impresso a sequencia de passos para localizar número k no array A. <span style="color: rgb(147, 149, 152);background-color: rgb(255, 255, 255);float: none;">A saída consiste do seguinte formato:</span></p><p><span style="color: rgb(147, 149, 152);background-color: rgb(255, 255, 255);float: none;">Array A entre colchetes [a1, a2, a3, ..., an ] ----&gt; a[j]</span></p><p><span style="color: rgb(147, 149, 152);background-color: rgb(255, 255, 255);float: none;">onde aj é o j-ésimo elemento do array consultado. J é a posição do elemento consultado.
