Título: Cadastro de Alunos
Topicos do problema: array; função; string; ponteiro; formatação; 
Tempo limite de execução: 2 segundo(s)
Descrição: <p><strong></strong>Implemente um sistema que gerencie funcionários de uma empresa, oferecendo as seguintes funcionalidades:</p><p><strong> 1. Adicionar funcionário</strong>: Cadastrar um funcionário com nome e salário.</p><p><strong>2. Exibir funcionários</strong>: Mostrar a lista completa de funcionários com seus respectivos salários.</p><p><strong>3. Remover funcionário</strong>: Permitir a exclusão de um funcionário pelo nome.</p><p><strong>4. Alterar salário</strong>: Permitir que o salário de um funcionário seja atualizado.</p><p><strong>0. Encerrar programa</strong>: Finalizar o sistema liberando todos os recursos alocados dinamicamente.</p><p><br/></p><p><b><i>Importante</i></b><i>: Analise com cuidado o formato das entradas e saídas para evitar erros de formatação. </i><br/></p>
Formato da entrada: <p>A operação do programa se dá por entradas numéricas que selecionam uma das opções do menu, seguidas pelos valores necessários para cada operação<strong></strong>. As entradas são no seguinte formato</p><p><b>Adicionar funcionários</b>: Recebe o nome do funcionário, seguido pelo salário.</p><p><b>Exibir funcionários</b>: Exibe uma lista com todos os funcionários como a seguir</p><blockquote><p>1. NomeFuncionario1</p><p>Salário</p><p>2. NomeFuncionario2</p><p>Salário</p><p>[Linha em branco]</p></blockquote><p><br/></p><p><b>Remover Funcionário</b>: Recebe o nome do funcionário e o mesmo é removido da lista.</p><p><b>Alterar Salário</b>: Recebe o nome do funcionário e o novo salário</p><p><br/></p>
Formato da saída: <p>A funcionalidade de exibir funcionários deve mostrar uma lista completa com todos os funcionários, ordenados a partir do 1 e com uma linha em branco no fim, caso nenhum funcionário esteja cadastrado deve ser exibida uma linha em branco.</p><p>As funcionalidade de remover e alterar o salário só exibem saída caso seja passado um nome inexistente no sistema, então deve ser exibida a mensagem</p><pre>Funcionario nao encontrado.</pre><p>Caso uma opção invalida seja solicitada no menu, o sistema deve retornar a mensagem</p><pre><p>Opcao invalida</p></pre><p><br/></p>
