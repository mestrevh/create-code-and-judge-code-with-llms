Título: Os novos prédios
Topicos do problema: array; 
Tempo limite de execução: 1 segundo(s)
Descrição: <span style="color: rgb(0, 0, 0);float: none;">Em uma rua serão construídos N prédios. Cada prédio tem uma altura Ai. </span><div style="color: rgb(0, 0, 0);"><br/></div><div style="color: rgb(0, 0, 0);">Você é uma das pessoas responsáveis pela supervisão das obras e por isso precisa saber informações dos projetos.<div><br/></div><div>Você recebe um vetor contendo a altura de cada um dos N prédios que serão construídos e a partir dele vai extrais as seguintes informações:</div><div><br/></div><div>- Quantas alturas distintas de prédios existem no total.</div><div>- Quantos prédios são altos, médios e baixos.</div><div>- A diferença de altura do maior e do menor prédio.</div><div><br/></div><div>OBS: Um prédio é considerado alto se tem altura &gt;= 100, médio se tem altura &gt;= 50 e pequeno se tem altura &lt; 50.</div></div><p></p>
Formato da entrada: <div style="color: rgb(0, 0, 0);"><div>Um inteiro N, representando quantos prédios serão construídos.</div></div><div style="color: rgb(0, 0, 0);">1 &lt;= N &lt;= 100</div><div style="color: rgb(0, 0, 0);">Na próxima linha N inteiros, representando a altura de cada prédio Ai.</div>
Formato da saída: <div style="color: rgb(0, 0, 0);">Na primeira linha quantas alturas distintas de prédios existem.</div><div style="color: rgb(0, 0, 0);">Na segunda linha quantos prédios são altos.</div><div style="color: rgb(0, 0, 0);">Na terceira  linha quantos prédios são médios.</div><div style="color: rgb(0, 0, 0);">Na quarta linha quantos prédios são baixos.</div><div style="color: rgb(0, 0, 0);">Na quinta linha a diferença de altura entre o maior e o menor prédio.</div>
