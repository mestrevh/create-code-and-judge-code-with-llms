Título: Chuva
Topicos do problema: array bidimensional; 
Tempo limite de execução: 1 segundo(s)
Descrição: <p>Bob trabalha no OBM (&Oacute;rg&atilde;o Brasileiro de Metereologia), que &eacute; a organiza&ccedil;&atilde;o respons&aacute;vel<br />pela medi&ccedil;&atilde;o dos &iacute;ndices pluviom&eacute;tricos (quantidade de chuva acumulada) em todo o pa&iacute;s.<br />Eles s&atilde;o muito eficientes no que fazem,mas est&atilde;o com um problema: eles n&atilde;o sabem<br />como proceder para calcular a quantidade acumulada de chuva que caiu em cada regi&atilde;o<br />em dois per&iacute;odos consecutivos, muito embora eles saibam os dados de cada per&iacute;odo<br />separadamente.</p>
<p>Como a chefia do &Oacute;rg&atilde;o estava muito ocupada, acabou ficando a cargo de Bob, o estagi&aacute;rio,<br />a tarefa de implementar um programa que some, para cada regi&atilde;o, a quantidade de chuva<br />acumulada em dois per&iacute;odos consecutivos.</p>
<p>O mapa que o OBM usa &eacute; dividido em N&times;N regi&otilde;es, sendo que para cada regi&atilde;o, a cada <br />per&iacute;odo, &eacute; determinado um n&uacute;mero inteiro indicando a quantidade de chuva acumulada.<br />A quantidade de chuva acumulada total em cada regi&atilde;o em dois per&iacute;odos consecutivos<br />&eacute; a soma das quantidades de chuva em cada um dos per&iacute;odos.</p>
<p>Mas como Bob &eacute; s&oacute; um estagi&aacute;rio e n&atilde;o est&aacute; acostumado a fazer nada mais do que tirar<br />c&oacute;pias de documentos, ele pediu sua ajuda para implementar o programa que calcula a<br />quantidade de chuva acumulada total nos dois per&iacute;odos para cada uma das regi&otilde;es, dadas<br />as quantidades de chuva acumulada em cada per&iacute;odo para cada regi&atilde;o.</p>
Formato da entrada: <p>A primeira linha da entrada cont&eacute;m um inteiro N indicando a dimens&atilde;o dos dois mapas que<br />devem ser lidos. Nas pr&oacute;ximas 2N linhas s&atilde;o dados os dois mapas, cada mapa indicando a<br />quantidade de chuva acumulada nas regi&otilde;es em um per&iacute;odo. Cada mapa &eacute; descrito em N<br />linhas consecutivas, cada linha contendo N inteiros, sendo que cada inteiro indica a quantidade<br />de chuva acumulada, no per&iacute;odo, em uma regi&atilde;o.</p>
<p>Considere:</p>
<p>&bull; 1 &lt;= N &lt;= 100.<br />&bull; 0 &lt;= quantidade de chuva acumulada em cada regi&atilde;o de cada mapa &lt;= 100.</p>
Formato da saída: <p>A sa&iacute;da dever&aacute; conter N linhas, com N inteiros em cada linha, indicando a quantidade de<br />chuva acumulada total em cada uma das regi&otilde;es nos dois per&iacute;odos considerados.</p>
