Multiplicação de Matrizes de Forma Ótima

Tempo limite: 1 segundo(s)

Dadas duas matrizes A e B, é possível determinar a matriz C = A*B usando a definição padrão de multiplicação de matrizes:

Como pode ser visto no exemplo abaixo:

O número de colunas da A precisa ser o mesmo número de linhas de B. Para efeitos de notação, dizemos então que rows(A) e columns(A) representam, respectivamente, o número de linhas e colunas de A.

O número de multiplicações individuais necessárias para computar a matriz C (que terá o mesmo número de linhas de A e o mesmo número de colunas de B) é então: rows(A) * columns(B) * columns (A). Por exemplo, se A é uma matriz 10X20 e B é 20X15, então serão necessárias 10 X 15 * 20, ou 3000 multiplicações.

Para realizar a multiplicação em mais de duas matrizes, nós temos que escolher como proceder. Por exemplo, se X, Y e Z são matrizes, então para computar  XYZ, podemos computar:

(XY)Z

ou

X(YZ)

Suponha que X é 5X10, Y é 10X20 e Z é 20X35. Vamos observar o número de múltiplicações necessárias usando as duas formas acima:

(XY)Z

  • 5 * 20 * 10 = 1000 multiplicações para determinar o produto (XY), uma matriz 5X20
  • Então 5 * 35 * 20 = 3500 multiplicações para calcular o resultado final.
  • Total de multiplicações: 4500

X(YZ)

  • 10 * 35 * 20 = 7000 multiplicações para determinar o produto (YZ), uma matriz 10X35
  • 5 * 35 * 10 = 1750 multiplicações para calcular o resultado final.
  • Total de multiplicações: 8750

Claramente, é possível calcular o resultado final com muito menos multiplicações se optarmos por fazer (XY)Z.

Dado o tamanho de cada matriz em uma sequência de matrizes que devem ser multiplicadas, sua missão é determinar a sequência ótima de multiplicações. Ou seja, a sequência que representa o menor número de multiplicações necessárias.

 

 

 

Formato da Entrada

Você receberá um número inteiro n correspondendo a uma sequência de matrizes.

Cada matriz da sequência é determinada apenas pelas suas dimensões. As dimensões são pares de inteiros, um por linha. Os pares representam o número de linhas e colunas de cada matriz.

O valor zero para n indica fim da entrada.

0 <=n <= 10.

Considere também o caso onde apenas uma matriz é dada, ou seja, n=1.

Formato da Saída

Assuma que as matrizes são denominadas A1, A2, ..., An.

Sua saída para cada sequência dada é uma linha contendo uma expressão "parentisada" indicando a ordem na qual as matrizes devem ser multiplicadas.

Utilize como prefixo de cada linha o número do caso de teste.
