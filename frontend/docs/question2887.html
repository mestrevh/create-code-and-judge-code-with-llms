Evolucao das especies

Tempo limite: 1 segundo(s)

Uma forma que os cientistas possuem para medir como uma espécie evoluiu para outra é encontrar como o genoma do ancestral mudou até a espécie atual. Espécies próximas possuem vários genes em comum e, logo, uma boa maneira de compará-las é comparar como seus genes trocaram de lugar.

Uma das mutações mais comuns que mudam a ordem dos genes é a "reversa". Se modelarmos um genoma como uma sequência de N genes, onde cada um é um inteiro de 1 a N, então a "reversa" é uma mutação que muda o genoma revertendo a ordem de um bloco consecutivo de genes.

Uma reversa pode ser descrita por dois índices (i, j), onde: 1 <=i <=j <=N. Os índices indicam que ele reverte a ordem do gene i até o gene j.  Como um exemplo, a reversa (3,6) aplicada ao genoma [1, 2, 3, 4, 5, 6, 7] tem como resultado: [1, 2, 6, 5, 4, 3, 7]. Se após essa primeira reversa, nós aplicarmos novamente uma reversa (1,3), então o genoma será: [6, 2, 1, 5, 4, 3, 7].

Um cientista estudando a evolução das espécies quer simular uma série de reversas em um genoma.  Depois ele vai querer saber a posição final de vários genes. Você deve elaborar um programa para auxiliar na resolução desse problema.

Formato da Entrada

São vários casos de testes.

A primeira linha de um caso de teste contém um inteiro N indicando o número de genes de um genoma (1 <= N <= 500). Você pode assumir que a ordem inicial dos genes é uma sequência de inteiros de 1 a N em ordem crescente. Deve ser mostrada a mensagem "Digite o numero de genes do genoma entre 1 e 500:"

A segunda linha contém um inteiro R (0<=R<=100) indicando o número de reversas. Então seguirão R linhas, cada uma contendo dois inteiros i,j, um em cada linha, indicando os índices da reversa. Lembre-se que nesse caso, i <=1 <= j <= N. Deve ser mostrada a mensagem "Digite o numero de reversas entre 1 e 100:", "Digite a posicao inicial da reversa:"e "Digite a posicao final da reversa:"

Depois das reversas, existe uma linha contendo um inteiro Q indicando o número de consultas que serão feitas para os genes. Seguirão Q linhas, onde cada linha contém um inteiro representando um gene cuja posição final você deve determinar. Deve ser mostrada a mensagem "Digite o número de consultas:" e "Digite a posicao a ser consultada entre 1 e N:" (mostre o valor numérico de N nesta frase)

A entrada só termina quando N for 0.

Formato da Saída

Para cada caso de teste, o seu programa produzirá Q+1 linhas de saída. A primeira linha contém o texto "Genome" seguida pelo número do caso de teste. As Q linhas seguintes contém a mensagem "Posicao n: ", seguidas de um inteiro cada, representando a resposta das consultas. O programa deve exibir as seguintes mensagens:

Digite o numero de genes do genoma entre 1 e 500:

Digite o numero de reversas entre 1 e 100:

Digite a posicao inicial da reversa:

Digite a posicao final da reversa:

Digite o número de consultas:

Genome n

Digite a posicao a ser consultada entre 1 e N:

Posicao n: n

Digite a posicao a ser consultada entre 1 e N:

Posicao n: n
