Tópicos do problema

decisão

Jogo dos numeros

Tempo limite: 1 segundo(s)

Em um determinado jogo, Biu tem dois dados para jogar em três rodadas consecutivas. Para Biu ganhar o jogo, deve existir após as três rodadas de dados

  • Um número maior ou igual a dez ou menor ou igual a 4.
  • Um número par. 
Caso essas duas condições não forem satisfeitas, Biu irá perder o jogo. Outra observação é que se Biu tirar 2 em qualquer rodada, independente se as condições de vitória forem satisfeitas, ele também irá perder o jogo.

Caso o usuário digite um valor fora da faixa válida (2 a 12), o programa deve exibir uma mensagem de erro.

Formato da Entrada

O usuário deve digitar a soma dos dois dados em cada uma das três rodadas, lembrando que esse valor pode estar entre 2 e 12.

Formato da Saída

O programa deve exibir se Biu ganhou ou perdeu o jogo, imprimindo na tela os valores abaixo:


Digite a soma dos dados em cada rodada separados por ENTER:

Biu ganhou / perdeu / Valor invalido
