Tópicos do problema

BOING

Tempo limite: 1 segundo(s)

O jogo de BOING é uma variação do jogo de BINGO conforme descrito a seguir.

Inicialmente, são colocadas N bolas numa urna, cada bola com um valor diferente sorteado no intervalo 0,...,M-1 (M>N).

Em seguida, cada um dos P jogadores sorteia exatamente K bolas da urna "com reposição". Ou seja, cada jogador J=0,...,P-1 terá K números,

V[J][0],...,V[J][K]

correspondentes aos valores das bolas sorteadas, alguns dos quais podem ser repetidos.

Após todos os jogadores terem sorteados seus números, as bolas são retiradas da urna e ordenadas por valor, originando a sequência

0 <= B[0] < B[1] < ... < B[N-1] <= M-1

Cada jogador J tem, em seguida, a sua pontuação T(J) computada como a soma das posições das suas bolas na sequência ordenada, ou seja,

T(J) = R(V[J][0])+...+R(V[J][K]),

onde B[R(X)]=X.

Finalmente, um último número alvo A é sortedo no intervalo 0,...,K*(M-1). O jogador vence (faz BOING), se a sua pontuação for igual ao alvo. Repare que uma partida pode ter zero, um ou mais de um vencedor.
 

Formato da Entrada

A entrada é constituída de C casos, cada um correspondendo a uma partida de BOING.

Cada caso inicia-se com uma linha com cinco inteiros

N M K P A

onde

N = número de bolas (1<N<=50000)
M = limite máximo do valor da bola (não-inclusivo) (1<N<=500000000)
K = número de bolas sorteadas por jogador (1<K<=100)
P = número de jogadores (1<P<=50000)
A = número alvo (0<=A<=K*(M-1))

seguem-se N linhas com os valores "não ordenados" das bolas e, posteriormente, K*P linhas, correspondentes aos valores das bolas sorteadas pelos jogadores (V[0][0],..., V[0][K-1], V[1][0],..., V[1][K-1],..., V[P-1][0],..., V[P-1][K-1]).

Após cada caso, há uma linha em branco.

Formato da Saída

Para cada caso i=0,...,C-1 na entrada, deve ser impresso na saída

caso i:[lista dos vencedores da partida precedidos por espaço]
[linha em branco]
