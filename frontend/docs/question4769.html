Presença na maratona de tecnologia do IFPB

Tempo limite: 1 segundo(s)

Durante uma maratona de tecnologia promovida pelo IFPB, foi disponibilizada uma lista com os nomes de todos os participantes esperados por dia. No entanto, alguns nomes estavam escritos de forma incorreta (com letras repetidas ou trocadas). Você foi encarregado de verificar quantas vezes o nome de um participante específico aparece de forma exata em cada dia. Seu programa deve:

  • Ler o nome do participante de referência (sem espaços).

  • Ler um número inteiro D, indicando a quantidade de dias do evento.

  • Para cada um dos D dias, o programa deve:

    • Ler um número N (quantidade de nomes registrados no dia).

    • Ler N nomes (um por linha).

    • Contar quantas vezes o nome informado aparece exatamente igual.

Ao final, o programa deve imprimir, para cada dia, a quantidade de presenças exatas daquele nome.

Formato da Entrada

  • O nome do participante de referência (sem espaços).

  • Um número inteiro D, indicando a quantidade de dias do evento.

  • Para cada um dos D dias:

    • Um número N (quantidade de nomes registrados no dia).

    • N nomes (um por linha).

Formato da Saída

Ao final, o programa deve imprimir, para cada dia, a quantidade de presenças exatas daquele nome.
