Simplesmente Café

Tempo limite: 1 segundo(s)

Em uma certa manhã, o Protagonista estava tentando pensar em problemas divertidos para a SECOMP 2022. Depois de algum tempo, ele percebeu que estava com muito sono, e foi na cozinha fazer um bom café para tomar. Magicamente, depois de tomar o seu café, ele despertou e pensou no seguinte problema:

Dado um array com N inteiros A[0], A[1], ..., A[N - 1]. Conte quantos pares de índices (i, j) satisfazem as seguintes restrições:

- (i < j)

- (A[i] + A[j]) é um número ímpar.

Entretanto, como o Protagonista não sabe programar, ele pediu para que você resolvesse esse problema para ele.


Nota: Para o array A = [1, 3, 2, 5, 4] existem 6 pares de índices válidos. São os pares:

(0, 2) -> 1 + 2 = 3

(0, 4) -> 1 + 4 = 5

(1, 2) -> 3 + 2 = 5

(1, 4) -> 3 + 4 = 7

(2, 3) -> 2 + 5 = 7

(3, 4) -> 5 + 4 = 9

Dica: Como a resposta pode ser muito grande, em linguagens como C e C++ por exemplo, a resposta pode não caber em um int, sendo necessário o uso de long long int.

Formato da Entrada

Na primeira linha vai ter um inteiro N.

Na segunda linha vai ter N inteiros A[0], ..., A[N - 1]. A[i] é o i-ésimo elemento do array.

1 <= N <= 100000

0 <= A[i] <= 100000

Formato da Saída

Um número inteiro indicando a quantidade de pares (i, j) que satisfazem as restrições.
