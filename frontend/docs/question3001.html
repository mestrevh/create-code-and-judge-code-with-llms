Criando matriz quadrada um pouco diferente

Tempo limite: 1 segundo(s)

Este é um problema que envolve o conceito de matriz.

Apresente um programa que leia a ordem de uma matriz bidimensional. A ordem da matriz é um número inteiro N (0< N <= 100).

O objetivo é construir a matriz correspondente de acordo com o exemplo apresentado no caso de teste.

Observação: Não devem ser usadas na solução funções embutidas como map, sort, reverte, etc.

Formato da Entrada

A entrada consiste de diversos números inteiros, um por linha, que correspondem à ordem das matrizes a serem construídas (veja o modelo no caso de teste). 

O seu programa deve finalizar quando a ordem da matriz for 0 (N = 0) ou um número negativo.

Formato da Saída

Para cada inteiro fornecido (ordem da matriz), apresentar a matriz correspondente. 

Deve existir um espaço em branco após cada elemento de uma dada matriz.

Entre uma matriz e outra deve haver uma linha em branco.

Quando não for possível apresentar a matriz, deve ser apresentada a mensagem "Fim".
