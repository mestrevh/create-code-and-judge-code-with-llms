Subconjuntos de Soma Única

Tempo limite: 3 segundo(s)

Para um qualquer conjunto de números, temos soma (A) como a soma dos elementos de A.
Considere o conjunto B = {1,3,6,8,10,11}.
Há 20 subconjuntos de B contendo três elementos, com seus respectivos cálculos são:

soma ({1,3,6}) = 10;
soma ({1,3,8}) = 12,
soma ({1,3,10}) = 14,
soma ({1,3,11}) = 15,
soma ({1,6,8}) = 15,
soma ({1,6,10}) = 17,
soma ({1,6,11}) = 18,
soma ({1,8,10}) = 19,
soma ({1,8,11}) = 20,
soma ({1,10,11}) = 22,
soma ({3,6,8}) = 17,
soma ({3,6,10}) = 19,
soma ({3,6,11}) = 20,
soma ({3,8,10}) = 21,
soma ({3,8,11}) = 22,
soma ({3,10,11}) = 24,
soma ({6,8,10}) = 24,
soma ({6,8,11}) = 25,
soma ({6,10,11}) = 27,
soma ({8,10,11}) = 29.

Algumas destas somas ocorrem mais de uma vez, outras são únicas.

Determine os subconjuntos que possuem somas únicas.

 

Formato da Entrada

A entrada é composta de um número N, tal que 0 < N < 10, que será o número de elementos do conjunto.


Então você vai receber N elementos, ordenados de forma crescente, separados por um espaço.

Obs: Só aceitar números positivos, caso exista algum negativo, a saída será: "-1"(sem aspas e com uma quebra de linha).

 

 

Formato da Saída

Os subconjuntos, formados de 3 elementos,que possuem somas únicas, e suas somas, separados por uma quebra de linha, no seguinte formato:

{1,2,3} = 6
{1,2,9} = 12

 

OBS:

A ordem da impressão dos conjuntos deve ser com o seguinte exemplo:

 

Suponhamos que os subconjuntos da resposta sejam:

 

{6,8,11} = 25,

{1,3,8} = 12,

{1,6,11} = 18,

{3,8,10} = 21,

{1,3,6} = 10;

 

deve ser impresso na seguinte ordem:

{1,3,6} = 10;

{1,3,8} = 12,

{1,6,11} = 18,

{3,8,10} = 21,

{6,8,11} = 25,

 

(O que é levado em conta para ordenação é a ordem dos elementos, não o valor do somatório).

 Que de forma abstrata, seria:

{menor primeiro elemento, menor segundo elemento, menor terceiro elemento}.

 

 

Obs2: O ultimo caso também tem uma quebra de linha.
