Tópicos do problema

estrutura de dados

Quem sou eu?

Tempo limite: 1 segundo(s)

Bob implementou uma estrutura de dados, mas ele não sabe o nome dessa estrutura.



Bob implementou duas operações:

i) Inserir

Essa operação recebe um número inteiro X e o insere na estrutura.



ii) Remover

Essa operação remove, e retorna, um número inteiro Y da estrutura.



Sua tarefa nesta questão é descobrir que estrutura de dados Bob implementou.

Formato da Entrada

A primeira linha da entrada contém T, o número de casos de teste.

Cada caso de teste será no seguinte formato:



N

O1 X1

O2 X2

...

ON XN



onde

N indica a quantidade de operações que Bob executou na estrutura.

Oi indica a operação que Bob executou na estrutura. (1 <= i <= N)

   Caso igual a 1, então a operação de Inserir foi executada.

   Caso igual a 2, então a operação de Remover foi executada.

Xi indica qual o número que foi inserido/removido da estrutura. (1 <= i <= N)



Exemplo:

5

1 5

1 7

2 7

2 5

1 -5



5 operações estão sendo executadas:

-> A primeira indica que a operação de Inserir está inserindo o número 5.

-> A segunda  indica que a operação de Inserir está inserindo o número 7.

-> A terceira indica que a operação de Remover retornou o número 7.

-> A quarta   indica que a operação de Remover retornou o número 5.

-> A quinta   indica que a operação de Inserir está inserindo o número -5.



Limites:

1 <= T <= 100

1 <= N <= 10000

1 <= Oi <= 2, para todo i entre 1 e N.

-10^9 <= Xi <= 10^9, para todo i entre 1 e N.



É garantido que a operação de Remover só é executada se a estrutura não estiver vazia.

Formato da Saída

Para cada caso de teste imprima



caso C: S



onde

C representa o número do caso atual, começando em 1.

S representa uma das 4 strings, em ordem de prioridade:

-> "ambas"

  caso a estrutura que Bob implementou possa ser tanto uma fila quanto uma pilha.

-> "pilha"

  caso a estrutura que Bob implementou possa ser uma pilha.

-> "fila"

  caso a estrutura que Bob implementou possa ser uma fila.

-> "nenhuma"

  caso a estrutura que Bob implementou não possa nem ser uma pilha e nem uma fila.
