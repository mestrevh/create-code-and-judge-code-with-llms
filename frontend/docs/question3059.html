Ansa vs Kcaj: O desafio dos triângulos

Tempo limite: 1 segundo(s)

Kcaj é um jovem estudante de computação, e se tem algo que ele ama é programar. Ele consegue resolver qualquer problema e, sabendo disso, decidiu participar da Competição Internacional de Geometria Computacional (CIGC) que é um evento bastante relevante na computação e matemática. A competição é realizada em duplas, em que são dados problemas de Geometria e vence quem conseguir resolver mais problemas em menos tempo. Ansa é a dupla de Kcaj que ama Geometria Computacional, porém nunca programou na vida, então o entrosamento deles é : Ansa tem a ideia geométrica, e Kcaj implementa a ideia em código. No entanto, Kcaj em seus códigos utiliza métodos não eficientes de programação, vulgarmente denominados de gambiarras. Tal fato, faz com que seus códigos não passem no tempo limite dos problemas da Competição. Sabendo disso Kcaj e Ansa pediram a sua ajuda para resolver o problema Master do evento.

O problema consiste em : 

Dada uma matriz NxN formada apenas pelos inteiros 0 e 1, você deverá encontrar a quantidade máxima de triângulos correspondentes, de base B, formados por 0  e 1. 

Observações:

OBS1 : Os triângulos buscados devem ser apenas isósceles.

OBS2 : Os triângulos serão sempre verticais (nunca inclinados, nem invertidos).

OBS3 : O valor da base do triângulo será quantos símbolos iguais consecutivos existem na base para aquele respectivo triângulo.



Formato da Entrada

Primeiro será dado um inteiro T ( 1 <= T <= 20) que corresponde a quantidade de casos de teste.

Para cada caso de teste T, virá um inteiro B (1 <= B < 100) onde B é sempre um inteiro ímpar que corresponde a base dos triângulos desejados, após isso virá um N (1 <= N <= 100) que indica as dimensões da matriz NxN e, por fim, virá a matriz.

Formato da Saída

Para cada caso de teste deve ser impresso:

"Caso X: 

P triangulos formados por 1

A triangulos formados por 0 ".

Onde : X é o respectivo caso, P é a quantidade máxima de triângulos formados pelo inteiro 1 e A é a quantidade máxima de triângulos formados pelo inteiro 0, sempre seguidos de quebra de linha.
