Busca em largura

Tempo limite: 1 segundo(s)

A busca em largura é um algoritmo utilizado para percorrer ou buscar itens dentro das estruturas de dados grafos.Como característica temos que a busca sempre ocorre vértices mais próximos ao vértice pelo qual a busca foi iniciada.

Desenvolva um programa para percorrer um grafo a partir de um vértice inicial e apresentar a ordem em que os vértices foram visitados.

Formato da Entrada

  • A primeira linha está a quantidade de vértices do grafo de entrada (inteiro).
  • A segunda linha está o vértice de origem da busca (inteiro).
  • As demais linhas representam as arestas do grafo. Em cada linha, tem-se dois vértices para identificar cada aresta do grafo. 

Por exemplo:


6

0

0 1

0 4

1 0

1 2

1 4

2 1

2 3

3 2

3 4

3 5

4 0

4 1

4 3

5 3

Formato da Saída

  • A saída deve mostrar a sequência em que os vértices foram visitados.

Por exemplo:


0

4

1

3

2

5
