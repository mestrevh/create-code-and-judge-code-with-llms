Consideramos um labirinto de tamanho N como uma grade de NxN células, numeradas em sequência de cima para baixo e da esquerda para a direita de 0 até N-1.



Por exemplo, para N=3, temos:





         _________________

  |     |     |     |

  |  0  |  1  |  2  |

  |     |     |     |

  |-----+-----+-----|

  |     |     |     |

  |  3  |  4  |  5  |

  |     |     |     |

  |-----+-----+-----|

  |     |     |     |

  |  6  |  7  |  8  |

  |     |     |     |

   -----------------





Cada célula do labirinto é delimitada por, no máximo, quatro paredes. As paredes interiores são numeradas também da esquerda para direita, de cima para baixo de 0 até 2*(N^2-N)-1.



Por exemplo, para N=3, temos:





   _________________

  |     |     |     |

  |     0     1     |

  |     |     |     |

  |     |     |     |

  |     |     |     |

  |--7--+--8--+--9--|

  |     |     |     |

  |     10    11    |

  |     |     |     |

   -----------------

Neste exercício, seu trabalho é determinar se células de labirintos são conectadas.

Formato da Entrada

O arquivo de entrada é composto de vários casos, cada caso correspondente a um labirinto.



Cada caso possui a seguinte forma:



N M Q

W1

W1

...

WM

A1 B1

A2 B2

...

AQ BQ

// Linha em branco



onde



N = Tamanho do labirinto

M = Quantidade de paredes internas removidas

Wi (i=1,...,M) = Paredes removidas  

Aj Bj (j=1,...,Q) = Pares de células consultadas 





LIMITES:

1 <= N <= 100

0 <= M <= 2*(N^2-N)

1 <= Q <= N^4/2

Formato da Saída

Para cada caso no arquivo de entrada (com Q consultas), o programa deve imprimir uma lista 



1.C1

2.C2

...

3.CQ

// Linha em branco



onde, para cada j=1,...,Q,



Cj=1, se Aj e Bj são conectados no labirinto correspondente ou

Cj=0, se Aj e Bj não são conectados.
