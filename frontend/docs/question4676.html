Gerenciador de fila

Tempo limite: 1 segundo(s)

Imagine que você está criando um sistema de gerenciamento de tarefas onde as tarefas são processadas em ordem. Uma estrutura de dados muito útil para esse tipo de sistema é a fila (queue), onde as tarefas são adicionadas ao final e retiradas do início, sempre seguindo a ordem em que foram inseridas.

Agora, vamos complicar um pouco! Em vez de usar uma fila comum, você deve implementar uma fila circular utilizando um array. Uma fila circular permite que, quando a última posição do array é alcançada, a próxima tarefa seja inserida na primeira posição disponível, reutilizando as posições do array de maneira eficiente.


Requisitos do sistema:

  1. Estrutura da Fila:

    • O tamanho da fila deve ser determinado por uma entrada fornecida pelo usuário.
    • A fila deve ser representada por um array de números não negativos.

  2. Operações:

    • Inserção (comando 'E'): Insira um valor no final da fila. Se a fila estiver cheia, exiba a mensagem "Fila cheia".
    • Remoção (comando 'D'): Remova o valor do início da fila e exiba-o. Se a fila estiver vazia, exiba a mensagem "Fila vazia".
    • Impressão da fila: A fila deve ser impressa após cada operação, mostrando os valores presentes no array. Elementos vazios devem ser representados por "-".

  3. Encerramento (comando 'F'): O programa deve terminar quando o comando "F" for lido, após exibir "Saindo...".

Dicas:

- Utilize variáveis para guardar os indices da frente e do final da fila!

- O índice do final vai ser o da casa vazia após o último elemento (observe a imagem).

- A fila terá um espaço vazio reservado, ou seja, caso tenha tamanho 5, a fila estará cheia quando tiver 4 números.

- Observe atentamente o exemplo para entender melhor!

Formato da Entrada

  • O programa inicia lendo um número inteiro SIZE, o qual é de forma garantida maior que 0, representando o tamanho da fila.
  • O programa então aceita uma sequência de comandos (E, D ou F), onde:
  • E <valor>: Adiciona o valor lido à fila.
  • D: Remove o valor da fila.
  • F: Encerra o programa.

    • Formato da Saída

      Se o comando for de inserção (enqueue), caso a fila esteja cheia (lembre da dica mencionada anteriormente) exiba "Fila cheia", caso contrário exiba a fila após a operação.

      Se o comando for de deleção (dequeue), caso a fila esteja vazia exiba "Fila vazia", caso contrário exiba o item removido e, na linha seguinte, a fila após a operação.

      Se o comando for de encerrar o programa, exiba "Saindo..."
