Pesquisa sobre violência no trânsito

Tempo limite: 20 segundo(s)

Anualmente são divulgadas as estatísticas de trânsito no Brasil. Estas estatísticas costumam apresentar, por município, os seguintes dados:

  • Nome do município;
  • Quantidade de veículos;
  • Total de acidentes com vítimas fatais;
  • Total de acidentes com vítimas não-fatais;
  • Total de acidentes sem vítimas no trânsito.

Através dessas estatísticas podemos identificar os municípios com o trânsito mais violento. Para isso foi criado o Índice de Violência no Trânsito (IVT), calculado a partir da seguinte regra:

(5 * no de acidentes com vítimas fatais + 3 * no de acidentes com vítimas não fatais + 1 * no de acidentes sem vítimas) / no de veículos

Após a entrada das informações de um município, seu programa deve verificar se o usuário deseja incluir os dados para um novo município (S para SIM e N para NÃO). Caso o usuário informe N, o sistema deve encerrar a entrada de dados.

Ao término do programa o sistema deve apresentar:

A quantidade de veículos em todo o país;

A quantidade de acidentes (ao todo) no país;

A quantidade de acidentes com vítimas fatais no país;

Os municípios com o maior e menor número de acidentes (ao todo).

Qual a média de veículos por município;

Quais as médias de acidentes com vítimas fatais, com vítimas não fatais e sem vítimas por município;

Qual o maior e menor IVT ;

Para os munícios com o maior e menor IVT, apresentar todos os dados coletados.

Obs: Utilize listas para armazenar os dados.

Formato da Entrada

A entrada deverá ser apenas os valores em ordem (nome do município, quantidade de veículos, número de acidentes fatais, não-fatais e sem vítimas, e se desejar incluir novo município). Não coloque strings em seu input (não forneça instruções de preenchimento) pois o corretor automático não os espera.

Exemplo de entrada (ignore os comentários):

Recife # nome do município
10000 # quantidade de veículos
10 # número de acidentes fatais
100 # número de acidentes não fatais
1000 # número de acidentes sem vítimas
S # incluir dados para novo município
Olinda # nome do município
7000 # quantidade de veículos
35 # número de acidentes fatais
70 # número de acidentes não fatais
700 # número de acidentes sem vítimas
N # incluir dados para novo município

Formato da Saída

A saída deve ser formatada exatamente como o exemplo abaixo (ignore os comentários), onde os números de pontos flutuante devem usar duas casas decimais:

17000 # total de veículos no país
1915 # total de acidentes no país
45 # total de acidentes com vítimas fatais no país
Recife 1110 # município com maior número de acidentes + número de acidentes do município
Olinda 805 # município com menor número de acidentes + número de acidentes do município
8500.00 # média de veículos por municípios
22.50 # média de acidentes com vítimas fatais por municípios
85.00 # média de acidentes com vítimas não-fatais por municípios
850.00 # média de acidentes sem vítimas por municípios
0.15 # maior IVT
Olinda # Nome do município com maior IVT
7000 # Quantidade de veículos do município com maior IVT
35 # Quantidade de acidentes com vítimas fatais no município com maior IVT
70 # Quantidade de acidentes com vítimas não-fatais no município com maior IVT
700 # Quantidade de acidentes sem vítimas no município com maior IVT
0.14 # menor IVT
Recife # Nome do município com menor IVT
10000 # Quantidade de veículos do município com menor IVT
10 # Quantidade de acidentes com vítimas fatais no município com menor IVT
100 # Quantidade de acidentes com vítimas não-fatais no município com menor IVT
1000 # Quantidade de acidentes sem vítimas no município com menor IVT
