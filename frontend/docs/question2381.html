Ambrósio não gosta de como o a estrutura de diretórios é exibida no bash.


Como você, amigo de Ambrósio, foi um bom aluno na disciplina de Estrutura de Dados, ele pediu sua ajuda para implementar esse novo módulo do bash (batizado de Nubash) que reconhece os comandos de criar e remover pastas e, além disso, consegue listar os diretórios na forma hierárquica.

O Nubash reconhece quatro comandos: mkdir, rm, ls e quit. A sintaxe de cada comando é como se segue:

  • mkdir <pasta_destino> <nova_pasta>: cria uma nova pasta dentro de uma que já existe. Ex.: mkdir videos filmes cria a pasta "filmes" dentro da pasta "videos".
  • rm <pasta_a_ser_deletada>: apaga a pasta e todas as suas subpastas. Ex.: rm videos remove a pasta "videos" e todas as pastas que estão dentro dela.
  • quit: encerra o Nubash.
  • ls: mostra a estrutura de pastas no formato de hierarquia. Ex.: se dentro do diretório raíz (root) tivéssemos as pastas "videos" e "musicas", dentro da pasta "videos" tivéssemos as pastas "filmes" e "series" e dentro da pasta "musicas" tivéssemos a pasta "rock", o comando ls exibiria o seguinte:
Obs:
  • O Nubash deve ser implementado usando a estrutura de dados árvore;
  • A pasta raíz sempre será a pasta root;
  • Cada pasta pode conter, no máximo, três subpastas;
  • A inserção de novas pastas deve ser feita preenchendo os espaços da esquerda para a direita.


  • É garantido que não existirão pastas com o mesmo nome em toda a estrutura de pastas.

Formato da Entrada

A entrada consiste de várias linhas com o seguinte formato:

Comando mkdir:

  • mkdir D N, onde D é uma string que representa o nome da pasta onde deve ser criada a pasta N, em que N é uma string representando o nome da nova pasta.
Comando rm:
  • rm R, onde R é uma string que representa o nome da pasta a ser excluída.
Comando ls:
  • ls, lista a estrutura de pastas no formato de hierarquia.
Comando quit:
  • quit, encerra o Nubash.

Formato da Saída

Cada comando deve exibir uma das seguintes mensagens:

Comando mkdir:

  • A mensagem "Pasta 'P' criada", em caso de sucesso ao criar a pasta P;
  • A mensagem "Erro ao criar a pasta 'P'", em caso de fracasso ao criar a pasta P;
  • A mensagem "A pasta 'P' esta cheia!\nErro ao criar a pasta 'N'", caso a pasta P já contenha 3 pastas. N é o nome da nova pasta que seria criada.

Comando rm:

  • A mensagem "A pasta 'P' foi removida", em caso de sucesso ao remover a pasta P;
  • A mensagem "Erro ao remover a pasta 'P'", em caso de fracasso ao remover a pasta P;

Comando ls:

  • Observe que a cada nível abaixo na estrutura de pastas, deve-se inserir um recuo de 4 caracteres de espaço em branco.
