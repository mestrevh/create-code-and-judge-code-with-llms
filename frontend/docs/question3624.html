Compra de smartphone

Tempo limite: 1 segundo(s)

Um site de avaliação de dispositivos móveis deseja fazer uma aplicação para permitir que seus  visitantes encontrem os smartphones que desejam com mais facilidade. A aplicação permite aos usuários  cadastrar novos smartphones e pesquisar aqueles já cadastrados até o limite de 10 dispositivos. Faça um programa em C para  auxiliar nesta tarefa, de acordo com os itens abaixo:

  • Definir uma estrutura Smartphone com os campos:
﻿
  • Implementar a função com assinatura int  cadastraSmartphone(int qnt_cadastrada, Smartphone phone[10]), a qual deverá acrescentar um smartphone ao cadastro no final da lista, solicitar do usuário os dados para preenchimento da estrutura e retornar a quantidade de smartphones na lista.
  • Implementar a função com assinatura int pesquisaSmartphones(int qnt_cadastrada, Smartphone reqMin, Smartphone phone[10]), a qual deverá exibir na tela os dados de todos os smartphones que se adequem aos requisitos mínimos e retornar a quantidade de smartphones encontrados utilizando esse critério.
O programa principal main() deverá realizar as seguintes operações:
  1. Em um laço, o programa receberá do usuário a resposta relacionada ao cadastro de um smartphone, enquanto for ‘s’ (para sim), a função cadastraSmartphone deve ser chamada, recebendo como parâmetros a quantidade cadastrada e a vetor de estrutura Smarphone. Caso contrário, o programa deverá parar de cadastrar novos smartphones.
  2. Após o final do laço de cadastro, o programa deverá solicitar uma estrutura Smartphone com os requisitos mínimos a serem procurados na função pesquisaSmartphones, a qual receberá como parâmetros o número de dispositivos cadastrados, uma variável do tipo Smartphone e a lista de dispositivos cadastrados, para que esta procure no arquivo de cadastro os smartphones compatíveis. 
  3. Por fim, o programa deverá imprimir a quantidade de smartphones encontrados pela função pesquisaSmartphones.
Observe que, para a  estrutura Smartphone com os requisitos mínimos, o campo modelo (nome do aparelho) não deverá ser preenchido, já que não é um critério avaliativo.

Formato da Entrada

O formato de entrada será dado por:
  • Um caractere informando se o usuário deseja adicionar um smartphone a lista, até que um valor diferente de ‘s’ seja informado.
  • Uma estrutura Smartphone para cada cadastro.
  • Os requisitos mínimos para o aparelho que servirá como base de comparação, seguindo os campos de memoriaMB, processadorMHz, cameraMPix e bateria da estrutura Smartphone.

O formato de saída será dado pelas informaçoes dos aparelhos que satisfazem os requisitos mínimos e o número de aparelhos encontrados, com duas quebras de linha, segundo o modelo abaixo:
