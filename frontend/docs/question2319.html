Jogo de aposta com cartas

Tempo limite: 1 segundo(s)

Cresci jogando baralho!  

Além de jogar os jogos mais conhecidos, inventar novos jogos com o baralho era o mais legal. 

Um dos jogos é o de apostar com base nas cartas colocadas na mesa, melhor dizendo, com os valores das cartas que os jogadores colocam na mesa. Logo, os naipes não são considerados na brincadeira.  Apenas os valores das cartas são utilizados; isto é, os valores de 1 a 13, considerando as cartas do Valete, Dama e Rei.

O jogo tem dois participantes e consiste em várias partidas. Em cada partida, cada jogador joga duas cartas. 

As regras para determinar quem ganha a partida são simples, baseadas nos valores das cartas de cada jogador:

  • se as duas cartas têm o mesmo valor, o jogador recebe como pontuação na partida duas vezes a soma dos valores das cartas.

  • se os valores das duas cartas são números consecutivos (por exemplo, 12 e 13, ou, 23 e 22), o jogador recebe como pontuação na partida três vezes a soma dos valores das cartas.

  • caso contrário, o jogador recebe como pontuação na partida a soma dos valores das cartas. 


Ganha a partida o jogador que tiver recebido a maior pontuação. Se houver empate, a aposta acumula para a próxima partida.

A sua função é escrever um programa para verificar quem ganha o jogo, lidos os nomes dos jogadores e as cartas de cada jogador em várias partidas. 

Formato da Entrada

A entrada é composta por uma ou mais linhas.

Cada linha corresponde a um jogo e contém os nomes dos dois jogadores e, para cada jogador, pares de valores de cartas, alternados entre os jogadores. 

Por exemplo, para a linha de entrada abaixo:

Ianna Henrique 1 2 3 4 5 6 7 8


Tem-se que a jogadora Ianna (jogador 1) tem dois pares de cartas: 1, 2, 5 e 6. O jogador Henrique (jogador 2) tem as cartas 3, 4, 7 e 8.

Em cada partida, os jogadores usam um par de cartas.

Quando a linha de entrada contém a palavra "sair" significa que não há mais jogadores e o jogo acabou.

Quando a linha de entrada tem menos que a quantidade mínima de dados para uma partida, deve ser apresentada a mensagem "erro".

Formato da Saída

Seu programa deve produzir uma única linha, para cada entrada de dados, contendo o nome do(a) jogador(a) que venceu o jogo. Se houver empate, a linha deve conter a palavra "empate".

No caso da entrada não atender às exigências mínimas, deve-se apresentar a palavra "erro". 

Ou, quando não há mais jogadores, deve-se apresentar a palavra "fim".
