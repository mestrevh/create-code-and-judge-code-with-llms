A Lista de Compras

Tempo limite: 1 segundo(s)

Os monitores de Introdução a Programação tentando fazer a comemoração de fim de ano, começaram a fazer a lista de itens que eles comprariam pra a festa pra ter uma ideia de quanto iriam gastar.

O problema é que como o grupo era muito grande, ficava muito difícil de operar a lista porque várias pessoas estavam constantemente inserindo novos itens e removendo, então eles decidiram pedir sua ajuda para que conseguissem computar direitinho o preço total da festa.

Você precisa criar um programa que aceite os seguintes comandos:

  • INSERIR NOME VALOR QUANTIDADE - insere um novo produto com o Nome, Valor de compra e Quantidade de itens
  • REMOVER NOME QUANTIDADE - remove uma determinada quantidade de itens (ou todos) de um produto pelo Nome
  • REMOVERGRUPO VALOR -  remove todos os produtos que tem seu valor total (por item) maiior que VALOR
  • CONSULTAR - consulta o preço total da Lista até o momento
  • PROCURAR NOME - procura se o produto de Nome está na lista e retorna seu Nome, Valor e Quantidade de itens

Formato da Entrada

A entrada é composta de várias linhas que podem ser no formato:


INSERIR N X Q

N sendo o nome do produto, X o valor em reais, Q a quantidade de itens.

REMOVER N Q

N sendo o nome do produto e Q a quantidade de itens a ser removidos (é garantido que há aquela quantidade de itens).

REMOVERGRUPO X

X sendo o valor que deseja-se remover todos os produtos daquele determinado valor.

CONSULTAR

Sendo o comando de consulta de itens na lista.

PROCURAR N

N sendo o nome do produto a ser procurado na lista.

Formato da Saída

Após cada comando CONSULTAR e PROCURAR deverá ser impresso:

CONSULTAR:

Para cada item da lista deve ser impresso suas características seguindo o padrão:

Atualmente a lista esta em R$X

ou

PROCURAR:

Caso o item esteja presente na lista deve ser impresso suas características seguindo o padrão:

Nome

- Valor

- Quantidade

ou caso não esteja:

Nome nao foi encontrado.

Seguidos sempre de uma quebra de linha
