Variáveis e expressões pós fixa

Tempo limite: 1 segundo(s)

Você deve avaliar um programa composto por múltiplos comandos em notação pós-fixa. Cada comando é uma atribuição escrita no formato:

= var expr

onde var é o nome de uma variável e expr é uma expressão em notação pós-fixa. Os comandos são separados pelo caractere ;.

Na notação pós-fixa (também chamada de notação polonesa reversa), os operadores aparecem após os operandos. Por exemplo, a expressão infixa (3 + 4) * 2 equivale à expressão pós-fixa:

3 4 + 2 *

A avaliação de expressões pós-fixas é realizada com uma pilha:

  • quando um operando é lido (um inteiro ou uma variável já definida), ele é empilhado;

  • quando um operador binário (+, -, *) é lido, dois valores são retirados da pilha, a operação é aplicada e o resultado é empilhado de volta.

Cada comando de atribuição avalia sua expressão pós-fixa e armazena o resultado na variável indicada. Uma variável pode aparecer como operando somente depois que já tiver recebido algum valor em um comando anterior.

O programa termina quando a entrada contém a string EOF.

Ao final da execução do programa, devem ser exibidas todas as variáveis definidas, na ordem em que apareceram, acompanhadas de seus últimos valores, no formato:

nome = valor

O tipo inteiro de 32 bits é suficiente para armazenar todos os resultados.

Formato da Entrada

A entrada consiste em um conjunto de comandos que formam um pequeno programa.
Cada comando representa uma atribuição de variável usando uma expressão em notação pós-fixa.

O formato geral de cada comando é:

= nomeVariavel expr

onde:

  • = indica que um novo comando de atribuição está começando;

  • nomeVariavel é o identificador da variável que receberá o valor calculado;

  • expr é uma expressão pós-fixa composta por:

    • números inteiros;

    • nomes de variáveis já definidas;

    • operadores binários: +, -, *.

Os comandos são sempre encerrados com um ponto e vírgula (;).
A entrada pode conter vários comandos ao longo de uma ou mais linhas.
O programa termina quando a leitura encontra a palavra EOF.

Formato da Saída

Cada linha da saída segue o formato:

nomeVariavel = valor

O valor mostrado é sempre o último atribuído à variável, caso ela tenha sido atualizada ao longo do programa.
