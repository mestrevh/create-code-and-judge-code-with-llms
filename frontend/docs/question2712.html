Viagem no tempo

Tempo limite: 1 segundo(s)

O ano é 2047 e com o advento da tecnologia do reino quântico, a viagem no tempo nunca foi tão possível. Porém, para que isso aconteça é necessário que um cientista com uma inteligência inigualável, como você, monitore a máquina. Com as especificações atuais, é possível viajar para 3 épocas diferentes: 

  • Egito Antigo (-3100 até -30) 
  • Japão Feudal (794 até 1185) 
  • Brasil Futurista (2235 ad infinitum)
Além dessas épocas o usuário pode ir para o ano de 2019, indo para o Presente. Você deve monitorar a máquina de acordo com os pedidos dos usuários, e após todos os pedidos dizer quantos já voltaram e quantos ainda estão nas outras épocas. Inicialmente todos estão no Presente.



OBS:
  • Anos negativos significam anos antes de Cristo
  • Não há nomes repetidos e não há viagens entre a mesma época (por exemplo, não é possível ir de 895 para 1067 sem passar pelo presente antes)
  • A questão deve ser feita com listas encadeadas

Formato da Entrada

Múltiplas linhas no formato:

S X

S uma string, representando o nome do usuário

X um número inteiro, representando o ano que o usuário deseja ir

Formato da Saída

Para cada linha N correta - que o ano esteja catalogado - deve ser exibido: 

"nomeDoUsuario foi de E1 para E2."

﻿

Sendo:

  • nomeDoUsuario uma string representando o nome do viajante do tempo
  • E1 uma string representando a época em que ele se encontrava
  • E2 uma string representando a época para que o usuário viajou

Senão deve ser exibido:

"Cuidado pra nao ficar preso no reino quantico, NomeDoUsuario."

nomeDoUsuario uma string representando o nome do viajante do tempo


Após todas as viagens, deve-se pular uma linha e exibir onde estão todas as pessoas que viajaram no tempo, sendo necessário apenas exibir a época se houver alguém nela, no seguinte formato:

EpócaX:

Nome1

Nome2

...

NomeN

Sendo ÉpocaX uma string correspondendo a época em que os usuários a serem listados estão.

NomeX uma string correpondendo a cada nome de usuário presente naquela época

Obs: Se o usuário não realizou nenhuma viagem, não deve ser listado.
