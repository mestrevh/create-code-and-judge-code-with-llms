Recanto Lenhoso em Perigo v2 (Ainda dificil)

Tempo limite: 1 segundo(s)

Em certo dia no planeta Recanto Lenhoso, Corneana captou 4 sinais intrigantes com seu telescópio. Os sinais estão em forma de inteiros, que fornecem dados de possíveis catástrofes acontecendo pelo sistema solar. Porém, além de dados de catástrofes, os sinais podem ser erros de leitura do equipamento ou pegadinhas do Gabro. Para cada mensagem, Corneana deve analisar a mensagem e informar a Feldspato o que fazer.

(Imagem ilustrativa, não precisa se preocupar com ela pra fazer a questão!)


O sinal é um valor inteiro não negativo. É captado de algarismo por algarismo, ou seja: se a entrada for 9 7 3 2 1, então o valor do sinal é 97321.

Caso o valor do sinal seja um palíndromo, o sinal é uma pegadinha do Gabro. Caso contrário, se o valor é divisível por 2 mas não é divisível por 7, o sinal é perigoso e está vindo do Gêmeo Cálido. Caso contrário, se o valor é divisível por 7 mas não é divisível por 2, o sinal é perigoso e está vindo do Gêmeo Cinzento. Caso contrário, se o valor estiver dentro do intervalo fechado [50000, 99999], o sinal é perigoso e está vindo do Vale Incerto. Caso contrário, se o valor estiver dentro do intervalo fechado [100,49999], o sinal é perigoso e está vindo do Profundezas do Gigante. Caso contrário, se o valor estiver dentro do intervalo aberto (0, 3), o sinal é perigoso e está vindo do Abrolho Sombrio. Caso contrário, o sinal foi um erro de leitura.

OBS1: um palíndromo é um número que é igual de trás pra frente e de frente pra trás. Ex.: 84548, 989, 2, 11, 10001.
OBS2: cuidado com zeros à esquerda. Para a entrada 0 0 5 1 5, o valor do sinal é 515, que é um palíndromo (zeros à esquerda não são considerados para julgar se um número é um palíndromo ou não).
OBS3: o operador % retorna o resto de uma divisão. Por exemplo, 10 % 3 é 1, pois 10/3 deixa resto 1. 20 % 2 é 0, pois 20/2 deixa resto 0.
DICA: ABCDE = A*10^4 + B*10^3 + C*10^2 + B*10 + A, sendo A B C D e E algarismos que formam o número ABCDE.

Formato da Entrada

Serão 4 linhas da seguinte maneira: 5 números inteiros (algarismos), todos entre 0 e 9 (inclusivo).

Formato da Saída

Para cada sinal: se o sinal for perigoso e vier do lugar X, deve ser printada a frase "Feldspato, va para X". Se o sinal for um erro de leitura, deve ser printada a frase "Feldspato, foi um erro de leitura". Se o sinal for uma pegadinha do Gabro, deve ser printada a frase "Feldspato, de um esporro no Gabro por mim".

Após printadas as 4 linhas, se todos os sinais tenham sido pegadinhas do Gabro, deve ser printada a frase "Espere, nao sao pegadinhas, esta vindo da Lua Quantica!".

OBS: Note que não há acentos nas frases printadas.
OBS2: X pode ser os seguintes lugares: Gemeo Calido, Gemeo Cinzento, Vale Incerto, Profundezas do Gigante, Abrolho Sombrio.
