decisão

Doe sangue, doe vida!

Tempo limite: 1 segundo(s)

Doar sangue é um processo fácil, rápido, seguro e solidário. A doação de sangue ocorre quando uma pessoa voluntariamente vai a um centro especializado e disponibiliza seu sangue para ser usado em transfusões ou outras situações clínicas.

Uma única doação é possível salvar até quatro vidas, uma vez que o material é separado em diferentes hemocomponentes.

No entanto, a fim de proteger tanto o doador como a pessoa que vai receber a doação, existem critérios estabelecidos por normas técnicas do Ministério da Saúde. 

O doador tem que atender, além de ter os critérios de impedimento definitivos e temporários verificados,  aos seguintes requisitos básicos:

  • Ter idade entre 16 e 69 anos, inclusive, desde que a primeira doação tenha sido feita até 60 anos;
  • Pesar no mínimo 50 kg;
  • A frequência máxima é de quatro doações anuais para o homem e de três doações anuais para a mulher. O intervalo mínimo deve ser de dois meses para os homens e de três meses para as mulheres.

Você deve fazer um programa que verifique se um possível doador atende aos requisitos básicos acima com base em dados lidos. Existem as mensagens possíveis no item saída e nos exemplos.

Formato da Entrada

O programa deve ler um valor inteiro para o mês do ano corrente.

Além disso, ler os dados do possível doador:

  • Idade

  • Sexo (F para feminino e M para masculino)

  • peso (float)

  • informação se já doou sangue antes (s ou S para sim, n ou N para não)

Caso o usuário já tenha doado sangue antes, deve-se ler também:

  • Ano da primeira doação 

  • Quantas doações já fez no ano corrente (2021)

  • Mês da última doação no ano corrente (use um inteiro para representar o mês) - digite 0 se não fez doação. 

Na solução do problema considere o ano atual (2021).

Considerando que o último mês de doação pode ter sido no ano anterior, você pode usar a função abs para calcular o valor absoluto de uma expressão. Por exemplo, abs(2-8) = 6. 


Formato da Saída

O resultado esperado na saída é uma mensagem indicando se o doador pode ou não doar sangue.

Quando os requisitos básicos forem atendidos, deve ser apresentada a mensagem "Pode ser doador.", se a pessoa for do sexo masculino e, "Pode ser doadora.", para a pessoa do sexo feminino. 

Para o caso negativo, deve ser apresentada a mensagem "Nao pode doar sangue!".

Se for por possível doar e for a primeira vez, a mensagem deve ser: “Parabens! Que bom que veio doar pela primeira vez!“.

Caso o usuário digite uma letra diferente de s ou n para a informação de já ter doado sangue, a mensagem deve ser 

"Erro! Era para ter digitado s ou n!".
