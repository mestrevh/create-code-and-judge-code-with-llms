Questão de String - Falando sobre verbos

Tempo limite: 2 segundo(s)

O estudo sobre verbos é algo bem interessante. Conforme o site da profa. Flávia Neves podemos observar que:


Os verbos, quando conjugados, são compostos por um radical, uma vogal temática, uma desinência modo-temporal e uma desinência número-pessoal, que se encontram estruturados na seguinte ordem:

radical + vogal temática + desinência modo-temporal + desinência número-pessoal

Radical

O radical é a parte menos variável do verbo, apresentando o seu significado lexical. 

  • And- é o radical do verbo andar.
  • Escrev- é o radical do verbo escrever.
  • Part- é o radical do verbo partir. 

Vogal temática

Através da vogal temática é possível distinguir as três conjugações verbais.

  • Vogal temática -a- indica a 1.ª conjugação: amar, brincar, ensaiar, trabalhar, falar,…
  • Vogal temática -e- indica a 2.ª conjugação: saber, viver, caber, entreter, concorrer,…
  • Vogal temática -i- indica a 3.ª conjugação: sorrir, dormir, dividir, concernir, assistir,…
Desinências modo-temporais das formas nominais:
Gerúndio: -ndo
Particípio: -do
Infinitivo: -r


Algumas observações são importantes: 
  • Todo verbo é uma ação.
  • Todo verbo tem sua terminação por r. Mas nem toda palavra que termina com r é um verbo. 
  • Os verbos que terminam com or, são da segunda conjugação também, mas seu particípio termina com sto, já se for er, troca o 'e' por 'i' para colocar o 'do'.
  • Existe alguns verbos irregulares, o que não iremos abordar aqui.
  • Não existe verbo que termina com u, apesar de existir palavras que terminem com ur.

Diante dessas explicações todas, será que somos capazes de ler uma palavra e informar algumas informações sobre ela? O professor de programação de vocês, acredita que sim. Por isso faça: 

1) Leia uma sequencia de palavras na mesma linha, separadas por vírgula. 
2) Informe quantas delas, estão em sua forma modo-temporal do infinitivo.
3) Para cada uma dessas do infinitivo apresente:
            a) Essas palavras nas outra duas formas nominais do modo-temporal. 
            b) Qual vogal temática do verbo.
            c) Qual o radical.
            d) Qual a conjugação verbal dele, se é a primeira, segunda ou terceira. 
   

Formato da Entrada

Várias palavras separadas por vírgula.

Formato da Saída

as frases: 

temos x palavra(s) no infitivo. (onde x é a quantidade de palavras no infinitivo).

o verbo xx no Gerúndio é: xxxxx (onde xx é o verbo no infinitivo e xxxxx é o verbo xx no gerúndio)

o verbo xx no Particípio é: xxx (onde xx é o verbo no infitivo e xxx é o verbo xx no particípio)

a vogal temática do verbo xx é w (onde xx é o verbo no infinitivo e w é a vogal temática)

o radical do verbo xx é zzz (onde xx é o verbo no infinitivo e zzz é o radical desse verbo)


Isso tantas vezes quanto tiver verbos no infinitivo, cada bloco por verbo deve ser separado por uma linha em branco.
