Viagem para Marte

Tempo limite: 1 segundo(s)

Natasha vai fazer uma viagem, num foguete, para Marte e voltar para Terra. Além disso, durante a viagem, ela vai pousar em n - 2 planetas intermediários. Os planetas são numerados de 1 (Terra) até n (Marte). Natasha vai fazer exatamente n viagens: 1 -> 2 -> ... -> n -> 1.

Uma viagem de x para y tem duas fases: decolagem do planeta x e pouso no planeta y. Dessa forma, o itinerário de voo será: decolagem do planeta 1 -> pouso no planeta 2 -> decolagem do planeta 2 -> ... -> pouso no planeta n -> decolagem do planeta n -> pouso no planeta 1.

A massa do foguete (sem considerar o combustível) é m toneladas. Porém, Natasha não sabe quanto de combustível ela vai precisar realizar sua viagem. E infelizmente, só é possível abastecer na Terra; então, se o foguete ficar sem combustível em algum outro planeta, a Natasha não poderá voltar para casa. É necessário combustível para decolar e pousar em cada planeta. Sabe-se que 1 tonelada de combustível pode decolar a_i toneladas de foguete do planeta i ou b_i toneladas para pousar no planeta i.

Por exemplo, se o peso do foguete é 9 toneladas, o peso de combustível é 3 toneladas e o coeficiente de decolagem é 8 (a_i = 8), então: 1.5 toneladas de combustível serão gastos na decolagem (visto que 1.5 * 8 = 9 + 3). O novo peso de combustível será 1.5 toneladas, após essa decolagem.

Note que o peso do combustível é um número não inteiro (real).

Ajude Natasha a calcular o peso de mínimo de combustível para abastecer o foguete. Note que o foguete deve gastar combustível tanto para carregar a si mesmo quanto o combustível. Porém, não é necessário carregar o combustível que já foi gasto. Assuma que o foguete decola e pousa instantaneamente.

Formato da Entrada

A primeira linha contém um inteiro n (2 <= n <= 1000), representando o número de planetas.

A segunda linha contém um inteiro m (1 <= m <= 1000), representando o peso do foguete.

A terceira linha contém n inteiros a_1, a_2, ..., a_n (1 <= a_i <= 1000), onde a_i é a quantidade de toneladas que podem ser "decoladas" com uma tonelada de combustível.

A quarta linha contém n inteiros b_1, b_2, ..., b_n (1 <= b_i <= 1000), onde b_i é a quantidade de toneladas que podem ser "pousadas" com uma tonelada de combustível.

É garantido que se a Natasha pode fazer essa viagem, será com no máximo 10^9 toneladas de gasolina.


Formato da Saída

Se a Natasha conseguir fazer sua viagem, imprima, com precisão de duas casas decimais, a menor quantidade de combustível (em toneladas) que a Natasha deve usar. Do contrário, imprima -1.

É garantido que se a Natasha pode fazer essa viagem, será com no máximo 10^9 toneladas de gasolina.
