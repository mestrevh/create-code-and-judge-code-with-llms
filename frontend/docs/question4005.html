Recanto Lenhoso em Perigo!

Tempo limite: 1 segundo(s)

Em certo dia no planeta Recanto Lenhoso, Corneana captou um sinal intrigante com seu telescópio. O sinal é um número inteiro não negativo, que fornece dados de possíveis catástrofes acontecendo pelo sistema solar. Porém, além de dados de catástrofes, o sinal pode ser erro de leitura do equipamento ou pegadinha do Gabro. Corneana deve analisar o sinal e informar a Feldspato o que fazer.

(Imagem ilustrativa, não precisa se preocupar com ela pra fazer a questão!)


O sinal é um valor inteiro não negativo. É captado de algarismo por algarismo, ou seja: se a entrada for 9 7 3 2 1, então o valor do sinal é 97321.


Caso o valor do sinal seja um palíndromo, o sinal é uma pegadinha do Gabro. Caso contrário, se o valor é divisível por 2 e não é divisível por 7, o sinal é perigoso e está vindo do Gêmeo Cálido. Caso contrário, se o valor é divisível por 7 e não é divisível por 2, o sinal é perigoso e está vindo do Gêmeo Cinzento. Caso contrário, se o valor estiver dentro do intervalo fechado [50000, 99999], o sinal é perigoso e está vindo do Vale Incerto. Caso contrário, se o valor estiver dentro do intervalo fechado [100, 49999], o sinal é perigoso e está vindo do Profundezas do Gigante. Caso contrário, se o valor estiver dentro do intervalo aberto (0, 3), o sinal é perigoso e está vindo do Abrolho Sombrio. Caso contrário, o sinal foi um erro de leitura.


OBS1: um palíndromo é um número que é igual de trás pra frente e de frente pra trás. Ex.: 84548, 989, 2, 11, 10001.

OBS3: o operador % retorna o resto de uma divisão. Por exemplo, 10 % 3 é 1, pois 10/3 deixa resto 1. 20 % 2 é 0, pois 20/2 deixa resto 0. Então, se A % B == 0, significa que A é divisível por B.

DICA: ABCDE = A*10^4 + B*10^3 + C*10^2 + D*10 + E, sendo A B C D e E algarismos que formam o número ABCDE.

Formato da Entrada

5 números inteiros (algarismos), todos entre 0 e 9 (inclusivo), representando os algarismos do sinal. Ex.: caso a entrada seja 1 3 4 0 5, o valor do sinal é 13405.

Formato da Saída

Deve ser printada a frase "Feldspato, X", sendo X a instrução dada por Corneana.
- Se o sinal for uma pegadinha do Gabro, deve ser printada a frase "Feldspato, de um esporro no Gabro por mim"
- Se o sinal vier do Gêmeo Cálido, deve ser printada a frase "Feldspato, va para Gemeo Calido"
- Se o sinal vier do Gêmeo Cinzento, deve ser printada a frase "Feldspato, va para Gemeo Cinzento"
- Se o sinal vier do Vale Incerto, deve ser printada a frase "Feldspato, va para Vale Incerto"
- Se o sinal vier do Profundezas do Gigante, deve ser printada a frase "Feldspato, va para Profundezas do Gigante"
- Se o sinal vier do Abrolho Sombrio, deve ser printada a frase"Feldspato, va para Abrolho Sombrio"
- Se o sinal for um erro de leitura, deve ser printada a frase "Feldspato, foi um erro de leitura"
