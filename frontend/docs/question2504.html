Tempo limite: 1 segundo(s)

Hércules, o príncipe Adam (a Fera), Aladim, John Smith e Tarzan, afastados do set de gravação, decidiram juntos fazer uma viagem para o Japão com finalidade de aprender um pouco mais sobre a cultura do país.

Ao chegar lá, em um dos dias de sua estadia eles aprenderam uma técnica japonesa para criptografar senhas e mensagens. A técnica era muito simples, mas os viajantes acharam incrível, uma vez que não conheciam nenhuma outra. O professor que havia lhes ensinado pediu para que eles criassem um programa que pudesse agilizar esse processo de tradução, mas como eles não entendiam de programação, pediram sua ajuda para que conseguissem completar a tarefa.

Dada uma frase de X caracteres, para ser codificada bastaria trocar N caracteres pelos novos caracteres fornecidos ao programa.

Sua tarefa é construir um programa que receba a frase original junto a uma lista de caracteres e imprima a frase criptografada com os novos caracteres a serem alterados.

  • Proibido usar replace()/replaceAll()
  • Proibido usar arrays
  • É obrigatório o uso de laços de repetição para resolução desta questão.
  • É obrigatório o uso de charAt()    


Exemplo:
String str = "Vou tirar dez nessa prova!"
char c = str.charAt(4); // atribui o caracter t à variável c
in.next().charAt(0); // Lê um caracter do teclado

Formato da Entrada

S
N


S - uma string representando a frase original

N - um inteiro que indica o número de caracteres a serem trocados


Seguem N linhas no formato:

I

C


I - um inteiro que indica a posição na frase do caractere a ser trocado

C - o novo caractere a ser colocado na posição I da frase


OBS: para cada I é garantido que estes serão fornecidos em ordem crescente

Formato da Saída

SC - uma string representando a nova frase criptografada
