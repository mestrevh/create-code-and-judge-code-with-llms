Aeroportos

Tempo limite: 10 segundo(s)

O governo de um certo país em desenvolvimento quer melhorar o serviço de transporte em uma de suas áreas menos acessíveis, em uma tentativa de atrair investimento. A região consiste de vários locais importantes que devem ter acesso a um aeroporto.
Obviamente, uma opção é construir um aeroporto em cada um desses lugares, mas pode ser mais barato construir menos aeroportos e construir rodovia para conectar os outros locais. Já que são rodovia de longas distâncias conectando locais importantes no país (cidades, grandes vilas e áreas industriais, por exemplo), todas as rodovia são de duas mãos. E, existem mais de uma rodovia possível entre duas áreas. Isso porquê existem várias formas de conectar duas áreas (um túnel através de uma montanha, uma rodovia ao redor da montanha, e etc) com custos possivelmente diferentes.
Considera-se que um local acessível, se ele tem acesso a um aeroporto se ou ele tem um aeroporto or se é possível viajar para outro local que tenha aeroporto.
Você receberá o custo de construir um aeroporto e uma lista com as possíveis ruas entre pares de locais and seus respectivos custos.
O governo precisa da sua ajuda para decidir a forma mais barata de fazer com que todos os locais sejam acessíveis. O objetivo é fazer com que aeroporto sejam o mais acessível possível, então, se existem várias formas de conseguir um custo mínimo, escolha aquele que tive a maior quantidade de aeroportos.

Formato da Entrada

A primeira linha da entrada contém um inteiro T (T < 25), o número de casos de teste. O resto da entrada consiste de T casos de teste.
Cada caso de teste inicia com três inteiros N, M e A (0 < N <= 10.000, 0 <= M <= 100.000, 0 < A <= 10.000) separados por espaço, onde N é número de locais, M é o número de possíveis rodovias, e A é custo de construir um aeroporto.
As próximas M linhas conterão três inteiros X, Y e C (1 <= X, Y <= N, 0 < C <= 10.000), separados por um espaço. X e Y são dois locais, e C é o custo de construir uma rodovia entre X e Y.


Formato da Saída

Seu programa deve imprimir exatamente T linhas, uma para cada caso de teste. Cada linha deve ser da forma: "Case #X: Y Z", onde X é o número do caso de teste, Y é o custo mínimo para construir as rodovias e aeroportos, de forma que todos os locais tenham acesso a pelo menos um aeroporto, e Z é o número de aeroportos que serão construídos. Como mencionado anteriormente, se existe mais de uma resposta com o menor custo, escolha o que maximize a quantidade de aeroportos.
