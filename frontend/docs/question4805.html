Formato da Entrada

A primeira linha contém uma letra maiúscula (A, B, C ou D), indicando a equipe que recebeu a mensagem.
A = Alpha
B = Bravo
C = Charlie
D = Delta


A segunda linha contém um número N ( 1 <= N <= 100 ), o qual indica a quantidade de letras da mensagem.


As demais N linhas contêm 7 números inteiros separados por espaços, ou seja, 7 x N números. Cada número representa um bit e pode ser 0 ou 1.
Os bits chegam na ordem do mais significativo (MSB) para o menos significativo (LSB).

Formato da Saída

Para cada caractere decodificado, imprimir uma linha no formato: 

Letra decodificada: X

Onde X é o caractere correspondente ao número binário na Tabela ASCII.

Dica: Em C, é possível atribuir um número a um char, por exemplo: printf("%c", 65) = A


Ao final, deve-se imprimir uma mensagem indicando a equipe que recebeu o texto, no formato:

Equipe Alpha recebeu a mensagem!

(ou Bravo, Charlie, Delta, conforme o caso).
