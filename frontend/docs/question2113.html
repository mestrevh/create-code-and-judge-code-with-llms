Critério de Sassenfeld

Tempo limite: 1 segundo(s)

Quando se utiliza um algoritmo iterativo para solução de sistemas lineares, é importante verificar se a convergência será garantida independentemente dos valores adotados no chute inicial. Um dos critérios verificados é o Critério de Sassenfeld.

Este critério consiste em calcular um fator alfa para cada linha da matriz do sistema de equações lineares. Se o máximo valor em módulo, dentre os alfas calculados, for menor que 1 (um) a matriz passou no critério e o sistema linear converge independente do chute inicial.

Para calcular o alfa da primeira linha você deve somar os elementos da linha 1 (em módulo), exceto o elemento da diagonal principal, e dividir a soma assim obtida pelo valor do elemento da diagonal principal (em módulo).

Neste problema, dada uma matriz quadrada, você deve informar se ela satisfaz ou não esse critério e imprimir o conjunto de alfas calculados para a matriz dada.

Formato da Entrada

Um valor inteiro que representa a dimensão da matriz quadrada, seguido pelos elementos da matriz.

Formato da Saída

Você deve imprimir SIM se o critério for satisfeito e NAO caso contrário. Em qualquer um dos casos o string é seguido de um final de linha.

Logo abaixo do SIM/NAO você deve imprimir os alfas calculados para a matriz dada, com duas casas decimais, separados por espaço. Não incluir espaço após o último alfa, no lugar de espaço, inclua uma nova linha.
