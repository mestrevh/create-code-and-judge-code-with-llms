L4Q3 - Campeonato de batalha naval 3000

Tempo limite: 1 segundo(s)

Depois de conseguir recriar o jogo Batalha Naval 3000, você resolveu chamar seus colegas alunos de IP para uma competição. Porém, preocupados com o tempo gasto para definir os três primeiros lugares, já que depois do campeonato vocês iriam estudar para a miniprova de AVLC, seus colegas definiram uma condição para participar, que você criasse um programa que definiria os 3 primeiros colocados dessa competição.

Você, como uma pessoa competitiva, aceitou o desafio e além de criar esse programa, resolveu definir uma regra para deixar as coisas mais divertidas. Se o jogador de número X terminar na posição X no campeonato de número X, esse jogador está automaticamente eliminado do campeonato, ou seja, o jogador não deve ser considerado para o cálculo dos 3 primeiros lugares. Além disso, como a cada partida os jogadores estão mais acostumados com o jogo, a cada competição a pontuação dada para cada jogador aumenta.

OBS1: Se o jogador número 1 ganhar o primeiro torneio, ele não deve ser eliminado.

OBS2: Sempre existirá um possível top 3 nos casos testes, ou seja, nunca a resposta terá menos de 3 jogadores.

OBS3: Deve ser usado array bidimensional para a resolução da questão.

OBS4: A fórmula para o cálculo da pontuação é a seguinte: 

Sendo número da competição a quantidade de competições que já ocorreram antes + 1.

OBS5: Os nomes dos jogadores são representados por números, em que o jogador 1 é o número 1, jogador 2 é o número 2 e assim sucessivamente, até o jogador n, que é o número n.

Formato da Entrada

É dado um número inteiro N, que representa a quantidade de jogadores do campeonato.

Um outro número inteiro M, que representa quantos campeonatos houveram.

Uma matriz MxN:

  • cada uma das M linhas representa um campeonato
  • as N colunas são usadas para registrar a posição de cada jogador no campeonato, ou seja, se o 3 aparece na 1a coluna é porque o primeiro jogador ficou em 3o colocado.
  • portanto, a posição I x J contém o jogador que, na partida I, ficou na J-ésima posição

Formato da Saída

Imprimir:

"O primeiro lugar foi: X"

Em que X representa o número do jogador que foi primeiro lugar.

"O segundo lugar foi: Y"

Em que Y representa o número do jogador que foi segundo lugar.

"O terceiro lugar foi: Z"

Em que Z representa o número do jogador que foi terceiro lugar.


OBS: Cada frase é seguida de uma quebra de linha
