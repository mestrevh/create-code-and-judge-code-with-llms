Decompondo cubos

Tempo limite: 1 segundo(s)

Na primeira semana de aula da turma de Álgebra Vetorial e Linear para Computação, o professor pediu para que os alunos provassem a seguinte "sequência":


1³ = 1
2³ = 3 + 5
3³ = 7 + 9 + 11
4³ = 13 + 15 + 17 + 19
5³ = 21 + 23 + 25+ 27 + 29


... Ou seja, N³ (N ao cubo) é igual a soma de N números ímpares consecutivos, de forma que cada número ímpar seja usado apenas uma vez e na ordem.
A sua tarefa é mais simples: dado um inteiro N você deve dizer quais são os N números ímpares cuja soma é N³.

Formato da Entrada

A primeira linha da entrada contém T, o número de casos de teste. Em seguida, temos T linhas, cada uma correpondendo a um caso de teste. Cada caso de teste consiste de uma única linha contendo o número N.

Limites:

1 <= T <= 100

1 <= N <= 100

Formato da Saída

Para cada caso de teste imprima uma linha na forma

caso C: d1 d2 ... dN

onde

C é o número do caso, começando em 1

d1 d2 ... dN é a lista dos números ímpares que somam N^3, em ordem crescente e separados por um espaço.
