P1Q1 - Who's that Pokémon

Tempo limite: 1 segundo(s)

Após a Rebelião Greyjoy, depois de ter orquestrado um ousado ataque à frota Lannister, Euron Greyjoy, irmão mais novo de lorde Balon Greyjoy foi exilado por seus crimes cometidos contra a coroa.


Euron viajou por todos os continentes e descobriu coisas incríveis. Porém, em uma de suas viagens, uma forte tempestade desviou seu navio e o afastou de sua frota. Após semanas de navegação sem rumo, sua tripulação avistou terra e ancorou o navio.


Nesta terra desconhecida, Euron encontrou várias criaturas muito estranhas e, após dias e muita ajuda dos moradores locais, ele começou a identificá-las através de certas características e montou uma tabela para ajudá-lo a identificar as criaturas.

Visando acelerar o trabalho de identificação, Euron sequestrou você e, sob ameaça de entregá-lo ao Deus Afogado, você teria que escrever um programa que o ajudasse a identificar os pokémons baseado nas características disponíveis. Atenção, por conta da distância, nem sempre será possível identificar a cor do pokémon.


AVISO: Apenas os conceitos vistos até a última aula devem ser utilizados na solução. Caso use outros conceitos a solução será desprezada.



Formato da Entrada

M

L

C


  • M - string ( { “aquatico”, “terrestre”, “voador” } )

  • L - string ( { “bipede”, “outro”, “quadrupede” } )

  • C - string ( { “amarelo”, “azul”, “branco”, “laranja”, “lilas”, “marrom”, “verde”, “vermelho”, “nao identificada” } )


A primeira linha da entrada é composta pela característica do tipo de movimentação.

A segunda linha pela característica do tipo de locomoção em terra.

A terceira linha pela cor predominante do pokémon, que também pode não ter sido identificada, nesse caso, será recebida a string: “nao identificada”.


Formato da Saída

P

  • P - string


A linha única de saída deve ser composta por:


  • Nome do pokémon, caso apenas um pokémon apresente as características descritas;

  • Quantidade de pokémons, caso mais de um pokémon apresente as características descritas;

  • “Pokemon desconhecido”, caso nenhum pokémon tenha as características informadas.



Caso haja mais de um pokémon com as características descritas a linha única deverá estar no formato "X pokemons", onde X é quantidade de pokemon com as características descritas.
