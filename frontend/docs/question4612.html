Lista de contatos

Tempo limite: 1 segundo(s)

Aurora não está satisfeita com o aplicativo de gerenciamento de contatos de seu celular, o aplicativo original parece ter sido desenvolvido na idade da pedra, ele permite que você guarde o endereço residencial do contato (quem anota endereços hoje em dia? Basta enviar a localização!) mas não permite que você anote o nome de usuário nas redes sociais (como eu disse, da idade da pedra!).

Por causa dessas questões, Aurora decidiu implementar seu próprio aplicativo de gerenciamento de contatos. Neste novo aplicativo ela pensou em guardar as seguintes informações: nome, apelido, telefone e usuário nas redes sociais.

Uma vez cadastrados os contatos, uma pesquisa por qualquer valor que possa estar cadastrado em um dos campos deve retornar todas as informações do contato.

Formato da Entrada

A entrada é composta por várias sequências de 4 linhas onde a primeira contém uma cadeia de caracteres minúsculos representando o nome, a segunda contém uma cadeia de caracteres minúsculos representando o apelido, a terceira contém uma cadeia iniciada pelo símbolo '+' seguido por uma sequência de números representando o número do telefone e a quarta contém uma cadeia iniciada pelo símbolo '/' seguida por uma cadeia de letras minúsculas, números e caracteres especiais (sem espaços) representando o nome de usuário nas redes sociais.

Os dados do último usuário são seguidos por uma linha contendo a palavra "FIM".

Após esta linha, segue uma sequência de linhas contendo um nome, apelido, número de telefone ou nome de usuário, um por linha, indicando os dados que se quer pesquisar no registro de contatos.

O fim dos dados de pesquisa é indicado por uma linha contendo a palavra "FIM".

Formato da Saída

Para cada linha pesquisada deve-se imprimir todos os dados dos contatos para os quais algum dos campos corresponde ao termo pesquisado como mostrado nos exemplos.

Uma linha contendo cinco asteriscos seguida por uma linha em branco deve ser impressa após o resultado de cada pesquisa. Após a impressão dos dados de cada contato também deve ser impressa uma linha em branco.

Caso uma pesquisa retorne mais de um contato, eles devem ser apresentados na ordem em que foram cadastrados.

Obs.: o nome do usuário pode ser pesquisado com ou sem o '/'.
