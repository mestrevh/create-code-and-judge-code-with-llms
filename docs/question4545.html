Descrição do Problema (problem.txt)

Título: Igual à soma de dois elementos Topicos do problema: busca; Tempo limite de execução: 1 segundo(s) Descrição:

Dado um array de n números reais e outro número s, determine se o array contem dois elementos cuja soma seja s. Por exemplo, para o array 5,9, 1, 3 e s = 6, a resposta é sim, mas para o mesmo array e s = 7 a resposta é não.

Projete um algoritmo para este problema que tenha eficiência em tempo melhor do que quadrático.

Dica: Reduza o problema para o problema similar de verificar se há dois números cuja soma é zero.

Formato da entrada:

A quantidade n de números da sequência.

A sequência, um elemento por linha.

O número s.

Formato da saída:

O string "SIM" quando há dois elementos da sequência que somam s. "NAO", caso contrário.

Gerado automaticamente.
