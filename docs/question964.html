Descrição do Problema (problem.txt)

Título: Critério das linhas Topicos do problema: array; métodos numéricos; repetição; array bidimensional; Tempo limite de execução: 1 segundo(s) Descrição:

Quando se utiliza um algoritmo iterativo para solução de sistemas lineares, é importante verificar se a convergência será garantida independentemente dos valores adotados no chute inicial. Um dos critérios verificados é o Critério das Linhas.

Este critério consiste em calcular um fator alfa para cada linha da matriz do sistema de equações lineares. Se o máximo valor em módulo, dentre os alfas calculados, for menor que 1 (um) a matriz passou no critério e o sistema linear converge independente do chute inicial.

Para calcular cada alfa você deve somar os elementos de cada linha (em módulo), exceto o elemento da diagonal principal, e dividir a soma assim obtida pelo valor do elemento da diagonal principal (em módulo).

Neste problema, dada uma matriz quadrada, você deve informar se ela satisfaz ou não esse critério.

Formato da entrada:

Um valor inteiro que representa a dimensão da matriz quadrada, seguido pelos elementos da matriz.

Formato da saída:

Você deve imprimir SIM se o critério for satisfeito e NAO caso contrário. Em qualquer um dos casos o string é seguido de um final de linha.
