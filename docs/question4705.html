Título: Q09 - É um palíndromo? Topicos do problema: função; string; Tempo limite de execução: 1 segundo(s) Descrição:

Implemente uma função chamada "inverter_string2" que recebe uma string e informe se ela é um palíndromo

palíndromos são palavras que lidas de trás para frente, são a mesma palavra

Desenvolva uma função que recebe uma palavra ou frase e retorna se ela é um palíndromo (ignorar espaços e maiúsculas/minúsculas).

Formato da entrada:

Uma string

Formato da saída:

"Eh um palíndromo" caso seja e "Nao eh um palindromo" caso não seja

Gerado automaticamente.
