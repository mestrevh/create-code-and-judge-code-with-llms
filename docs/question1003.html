Título: Loteria Topicos do problema: heap; multiplas entradas; Tempo limite de execução: 1 segundo(s) Descrição:

Uma empresa de loteria está lhe contratando para desenvolver um sistema para apuração eficiente dos resultados dos concursos realizados, os apostadores podem escolher 15 números dentre os valores de 1 até 50, onde serão sorteados 10 valores dessa faixa, sendo igualmente premiados as apostas com maior e menor número de acertos, impedindo a acumulação do prêmio. Atendendo a requisitos de modularização, o sistema de apuração já recebe os dados processados com os códigos e os valores escolhidos de cada aposta. Deste modo, você precisará identificar os ganhadores.Entrada:A primeira linha contém os 10 valores sorteados, na segunda linha possui um inteiro N, que indica a quantidade de apostas realizadas, e as próximas N linhas são formadas pelo código e os valores escolhidos para cada aposta.Saída:São exibidas as duas faixas de acerto, a listagem dos códigos das apostas, e quantidade de apostas premiadas para cada faixa.
