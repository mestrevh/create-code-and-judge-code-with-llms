Título: Divide dois inteiros Topicos do problema: obi; Tempo limite de execução: 1 segundo(s) Descrição:

Dados dois números inteiros, efetue a divisão.

Antes do exibir a saída, use o comando cout.precision(2). Isso fará o programa mostrar o resultado com duas casas decimais.

Exemplo:

int mani(){
...;
cout.precision(2);
cout<<resultado;
}



Formato da entrada:

6

3

Formato da saída:

2

Gerado automaticamente.
