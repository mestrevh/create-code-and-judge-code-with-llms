Sua tarefa é, dada várias sequências de números, dizer qual é o menor número, o maior número, a média, a moda e a mediana de cada sequência.

Formato da entrada:

Uma linha contendo um inteiro C representando o número de casos de teste a entrada possui, seguida pelos C casos de teste.

Cada caso de teste começa com uma linha com um inteiro N representando a quantidade de números na sequência dessa rodada.

Seguida por uma linha com N inteiros, que é justamente a sequência a ser testada naquele caso.

OBS:

1 <= C <= 20

1 <= N <= 1000

Cada da sequência varia de -100 até 100

Formato da saída:

Para cada caso deve ser impressas 6 linhas:

  1. qual é aquela sequência
  2. os números daquela sequência ordenados em ordem crescente
  3. o menor número e o maior número da sequência
  4. a média da sequência
  5. a(s) moda(s) da sequência
  6. a mediana da sequência

OBS: nas linhas 4 e 6 quando houver números com casas decimais, só deve ser impresso a quantidade de casas decimais necessárias, sem zeros a direita desnecessariamente.
