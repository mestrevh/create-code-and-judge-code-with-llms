Descrição do Problema (problem.txt)

Título: Área e Volume de Corpos Sólidos Topicos do problema: ponto flutuante; decisão; lógica matemática; Tempo limite de execução: 1 segundo(s) Descrição:

Construa um código que calcula a área e o volume de um corpo sólido com base nas entradas do programa. 

Adote pi = 3.1415

Formato da entrada:

Uma variável do tipo char

  • Caso seja 'c', deve calcular os dados de um cilindro.
  • Caso seja 'e', deve calcular os dados de uma esfera.
Uma segunda variavel do tipo char
  • Caso seja 'a', deve calcular a área do corpo
  • Caso seja 'v', deve calcular o volume do corpo
Caso seja uma esfera, uma variável do tipo float indicando o valor do raio.

Caso seja um cilindro, duas variáveis do tipo float, indicando raio e altura, respectivamente.

Formato da saída:

Caso as entradas sejam válidas, o padrão da saída é: 

  • "A area da FORMA GEOMETRICA de raio X e altura Y eh: Z"
  • "O volume da FORMA GEOMETRICA de raio X e altura Y eh: Z"
Quando na primeira entrada for informado um caracter inválido, deve imprimir "Entrada invalida! Voce deve usar 'c' (cilindro) ou 'e' (esfera)."
E quando a segunda entrada for inválida, deve imprimir "Entrada invalida! Entre com 'a' (area) ou 'v' (volume)."
