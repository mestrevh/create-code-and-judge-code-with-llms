Descrição do Problema (problem.txt)

Título: Exponenciação de matrizes Topicos do problema: array bidimensional; requer solução eficiente; Tempo limite de execução: 1 segundo(s) Descrição:

Sua missão é receber uma matriz quadrada n X n e elevar essa matriz a k-ésima potência.

Considere:

  • 2 <= n <= 10
  • 1 <= k <= 1.000.000.000
Formato da entrada:

Você receberá vários casos de teste.

Cada caso de teste é descrito como a seguir:

Na primeira linha você receberá dois inteiros separados por um espaço: n e k, respectivamente.

Depois, você receberá n linhas, cada linha contendo n números de ponto flutuante separados por espaço.

Formato da saída:

Para cada caso de teste, imprima a matriz com o resultado da exponenciação. Cada número deve ser formatado com 3 casas decimais.

Veja o exemplo de saída para mais detalhes sobre o formato.


Não há nenhuma linha em branco entre os casos de teste.

Gerado automaticamente.
