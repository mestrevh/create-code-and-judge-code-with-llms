Descrição do Problema (problem.txt)

Título: Contagem do insertion sort Topicos do problema: ordenação; Tempo limite de execução: 1 segundo(s) Descrição:

Modifique a implementação do algoritmo Insertion Sort para contabilizar o número de deslocamentos que ele realiza durante a ordenação. A única coisa que você deve imprimir é o número de deslocamentos feitos pelo algoritmo para ordenar completamente o array. Um deslocamento ocorre quando a posição de um elemento muda no array. Não desloque um elemento se não for necessário.



Formato da entrada:

A primeira linha deve conter um inteiro n para o número de elementos a serem ordenados.

A próxima linha contem n inteiros do array.

Exemplo:

STDIN       Function
-----       --------
5           arr[] size n =5
2 1 3 1 2   arr = [2, 1, 3, 1, 2]
Formato da saída:

Retorna apenas um inteiro correspondendo pela quantidade de trocas realizadas pelo algoritmo.

Exemplo:

4

Iteration   Array      Shifts
0           2 1 3 1 2
1           1 2 3 1 2     1
2           1 2 3 1 2     0
3           1 1 2 3 2     2
4           1 1 2 2 3     1

Total                     4

Gerado automaticamente.
