Descrição do Problema (problem.txt)

Título: Preço do ingresso Topicos do problema: repetição; lógica matemática; Tempo limite de execução: 2 segundo(s) Descrição:

Uma companhia de teatro deseja dar uma série de espetáculos. a direção calcula que a R$ 5,00 o ingresso, serão vendidos 120 ingressos. As despesas são fixas em R$ 200,00. A cada R$ 0,50 diminuídos no preço do ingresso espera-se que as vendas aumentem em 26 ingressos.

Faça um programa que escreva uma tabela de valores de lucros esperados em função do preço do ingresso, fazendo variar o preço R$ 0,50 por vez no intervalo de R$ 5,00 a R$ 1,00 (R$ 5,00; R$ 4,50; ...R$ 1,50; R$ 1,00). O programa ainda deve mostrar o público esperado para cada valor do ingresso.

Formato da entrada:

Não há entradas no algoritmo

Formato da saída:

O programa deve exibir a tabela com o preço do ingresso, o público esperado e lucro esperado (utilize espaço, barra, espaço entre as informações), imprimindo na tela os valores abaixo:


preco do ingresso: R$ 5 / publico esperado: n / lucro esperado: R$ n

preco do ingresso: R$ 4.5 / publico esperado: n / lucro esperado: R$ n

preco do ingresso: R$ 4 / publico esperado: n / lucro esperado: R$ n

preco do ingresso: R$ 3.5 / publico esperado: n / lucro esperado: R$ n

preco do ingresso: R$ 3 / publico esperado: n / lucro esperado: R$ n

preco do ingresso: R$ 2.5 / publico esperado: n / lucro esperado: R$ n

preco do ingresso: R$ 2 / publico esperado: n / lucro esperado: R$ n

preco do ingresso: R$ 1.5 / publico esperado: n / lucro esperado: R$ n

preco do ingresso: R$ 1 / publico esperado: n / lucro esperado: R$ n

Gerado automaticamente.
