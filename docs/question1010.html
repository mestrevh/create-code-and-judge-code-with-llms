Descrição do Problema (problem.txt)

Título: Pi Leibniz Topicos do problema: acumulador; ciclos; Tempo limite de execução: 1 segundo(s) Descrição:

Escreva um programa paraimplementar uma série convergente, que calcula o número Pi de Leibniz. 

Você deve ler um inteiroN, maior ou igual a 1, e imprimir o somatório dos N primeiros termos da sériemultiplicado por 4.

A série utiliza o padrãoapresentado abaixo:

Pi/4 = 1/1 - 1/3 + 1/5 -1/7 + 1/9 - ...

Dicas:

- utilize o number format para formatar sua saída. Veja um exemplo para formatar o número num com 3 dígitos:

import java.text.NumberFormat;
NumberFormat nf;
nf = NumberFormat.getNumberInstance( );
nf.setMaximumFractionDigits(3);
resposta = nf.format(num)

- o numberformat utiliza as configurações regionais do seu computador. Portanto, mesmo que as configurações regionais do seu computador faça com que sua resposta seja impressa usando vírgula para separar os decimais, submeta-a, pois o Huxley vai gerar respostas com ponto e aceitará o resultado, caso o valor esteja correto;

- cuidado com a divisãopor zero. O resultado será infinito, se não tratar esse caso;

- lembre-se de alternaros sinais;

- a saída deve conterexatamente 5 dígitos após a vírgula. Por exemplo: 3.14159

Formato da entrada:

Um número inteiro.

Formato da saída:

Um número do tipo double,com exatamente 5 dígitos depois da vírgula. Por exemplo: 3.14159

Gerado automaticamente.
