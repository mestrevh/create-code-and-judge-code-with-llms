Descrição do Problema (problem.txt)

Título: Ambrósio e a diferença Topicos do problema: números binários; multiplas entradas; Tempo limite de execução: 2 segundo(s) Descrição:

Professor Ambrósio continua ensinando binários para sua filha Ambrosinita como na questão : https://www.thehuxley.com/problem/599

Ele criou um novo jogo, Ambrósio escolherá um inteito A e Ambrosinita escolherá um inteiro B. O objetivo do jogo é que Ambrosinita responda qual a diferença entre esses 2 números. A diferença entre os dois números nesse jogo é igual a quantidade de bits que são diferentes entre os dois números em suas representações binárias.

Ex:  A = 1, B = 4;

A = (001) base 2.

B = (100) base 2.

a diferença é 2 (primeiro e ultimo bit).

Ajude a Ambrosinita a responder corretamente.


Formato da entrada:

Primeira linha contém um T indicando a quantidade de casos de teste.

As próximas T linhas contém A e B, indicando a escolha de cada um.

1 ≤ T ≤ 104
0 ≤ A, B ≤ 1016

Formato da saída:

Para cada caso de teste imprima a quantidade de bits diferentes entre os 2 números em suas representações em binário.

Gerado automaticamente.
