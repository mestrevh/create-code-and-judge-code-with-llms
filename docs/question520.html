Descrição do Problema (problem.txt)

Título: BoB Topicos do problema: busca; Tempo limite de execução: 1 segundo(s) Descrição: 
Bob estava estudando algoritmos quando se deparou com o seguinte problema:



Dado um array ordenado A, com N elementos, e um número X: Quantas vezes o número X aparece em A?



Bob prontamente implementou um algoritmo de força-bruta que respondia essa pergunta.



Quando ele rodou uma vez se surpreendeu com a velocidade do programa. Mas quando ele tentou responder a pergunta uma grande quantidade Q de vezes, o programa demorou muito para acabar, tanto que ele nem teve paciência de esperar.



A sua tarefa é resolver esse problema.



Dado um array ordenado A, com N elementos, responda Q perguntas do tipo: "Quantas vezes Xi aparece em A?"
Formato da entrada: 
A primeira linha contém um inteiro T, o número de casos de teste.

Cada caso de teste terá o seguinte formato:



N Q

A1 A2 ... AN

X1 X2 ... XQ



onde

N inteiro positvo indicando a quantidade de elementos no array ordenado A,

Q inteiro indicando o número de elementos a serem procurados

Ai inteiro indicando o i-ésimo elemento de A,

Xi inteiro indicando o número referente à i-ésima pergunta.



Limites:

2^0 < T < 2^7

2^0  < N < 2^18

2^0 < Q < 2^15

-2^31 < Xi < 2^31, para todo i entre 1 e Q

-2^31 < A1 <= A2 <= ... <= AN < 2^31
Formato da saída: 
Para cada caso imprima:



caso C:

R1

R2

R3

...

RQ



onde

C é o número do caso atual, começando em 1

Ri é a resposta para a i-ésima pergunta.



ATENÇÃO: Não há linhas em branco após cada caso
