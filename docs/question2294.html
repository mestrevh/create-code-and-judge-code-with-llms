Descrição do Problema (problem.txt)

Título: Remoção de caracteres repetidos (usando fila) Topicos do problema: estrutura de dados; repetição; função; Tempo limite de execução: 1 segundo(s) Descrição:

Usando o tipo TFilha visto em aula e as funções básicas de fila, construa um programa que leia uma string s e passe como parâmetro para a função void removeRepetidosSeguidos(char s[]) que deverá remover os caracteres repetidos em sequência de s. Depois da execução da função, o programa principal deverá exibir s já modificado.

Obs.: (1) Deve ser usada uma filha na função; (2) Cuidado com o '\0'; (3) Além das funções básicas de fila, criar somente a removeRepetidosSeguidos(s); (4) não usar recursão.

Formato da entrada:

Uma string s (1 <= tamanho(s) <= 50).

Formato da saída:

A string da entrada sem caracteres repetidos em sequência. Finalize com uma quebra de linha.
