Descrição do Problema (problem.txt)

Título: Substituição de caracteres em uma string Topicos do problema: array; substrings; caractere; Tempo limite de execução: 1 segundo(s) Descrição:

Faça um programa que leia uma string e dois caracteres. Troque todas as ocorrências do primeiro caracter pelo segundo. Exemplo: Seja a string "maracatu" e os caracteres 'a' e 'o', então a string ficará "morocotu".

Formato da entrada:

Você receberá 03 linhas. 

 

  • Primeira linha: a string
  • Segunda linha: o primeiro caracter
  • Terceira linha: o segundo caracter
A string possui no máximo 100 caracteres.

 

Formato da saída:

Imprima a palavra resultante da substituição dos caracteres na string original. Imprima um final de linha depois da palavra.

Gerado automaticamente.
