Descrição do Problema (problem.txt)

Título: L2Q10 - The Long Night Topicos do problema: repetição; caractere; Tempo limite de execução: 1 segundo(s) Descrição:

Durante suas viagens você chega em Atalaialeste. Nesse momento, você descobre que, após a muralha, uma figura importante está em perigo além. Faz-se necessário enviar com urgência um corvo para a Pedra do Dragão. Para que os Lannisters não interceptarem, a mensagem tem que ser criptografada. Como não há tempo para criptografar a mensagem manualmente, você se oferece para escrever um programa que criptografe a mensagem rapidamente.

A criptografia usada é uma cifra de substituição, onde cada letra é substituída pela letra que está 6 posições depois no alfabeto, mantendo inalteradas a característica de maiúscula ou minúscula de cada letra.

Obs: Se a letra passar do Z, ela volta para o início do alfabeto.

Por exemplo: "Gendry" é criptografado para "Mktjxe"

Formato da entrada:

N linhas que precisam ser criptografadas, terminadas por fim de arquivo

As linhas só contém espaços e letras maiúsculas e minusculas.

Formato da saída:

N linhas que representam a mensagem criptografada

Gerado automaticamente.
