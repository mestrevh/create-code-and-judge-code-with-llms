Descrição do Problema (problem.txt)

Título: Ordenação I Topicos do problema: ordenação; Tempo limite de execução: 1 segundo(s) Descrição:

Implemente o algoritmo de ordenação que possua as seguintes propriedades:

  • Usa a estratégia dividir e conquistar

  • Algoritmo estável

  • Algoritmo “not in place”

  • Complexidade de tempo O(nlogn)

  • Complexidade de espaço O(n)

  • Comumente conhecido pela sua eficácia para ordenar listas encadeadas


Fica absolutamente proibido o uso de métodos de ordenação nativos de quaisquer linguagens de programação. Você deve implementar seu próprio algoritmo para resolver os problemas.

Formato da entrada:

A primeira linha contém um número inteiro N que representa o número de vetores a serem ordenados.

Em seguida há uma novo número de N novas entradas em formato de string. Estas entradas são os vetores a serem ordenados, estando os números do vetor separados por espaços.

A string "1, 7, 3, 4, 8, 2, 9" representaria o array de inteiros [1, 2, 3, 4, 7, 8, 9].

Formato da saída:

A saída deve retornar linha a linha todos os arrays ordenados.

Gerado automaticamente.
