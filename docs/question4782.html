Descrição do Problema (problem.txt)

Título: Duracell Topicos do problema: pilha; caractere; string; Tempo limite de execução: 1 segundo(s) Descrição:

Dada uma expressão composta por parênteses (), colchetes [] e chaves {}, verifique se ela está corretamente balanceada e bem aninhada.

Uma expressão é considerada válida se:

  • Cada parêntese, colchete ou chave de abertura possui um correspondente de fechamento.

  • O fechamento ocorre na ordem correta, respeitando o aninhamento. Por exemplo, a expressão ([)] é inválida, pois fecha os parênteses de forma incorreta.

Formato da entrada:
  • Um número inteiro N (1 <= N <= 20), representando a quantidade de expressões que serão verificadas.
  • Em seguida, N linhas, cada uma contendo uma expressão composta exclusivamente pelos seguintes caracteres: (, ), [, ], {, }.
  • Não haverá espaços na expressão e seu tamanho máximo será de 100 caracteres.
Formato da saída:

Para cada expressão, imprima:

  • "True" se ela for válida (balanceada e corretamente aninhada).
  • "False" caso contrário.

Gerado automaticamente.
