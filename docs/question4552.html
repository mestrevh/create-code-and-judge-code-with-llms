Descrição do Problema (problem.txt)

Título: I-ésimo Maior Topicos do problema: array; Tempo limite de execução: 1 segundo(s) Descrição:

Você vai receber uma lista com n valores e deve retornar o i-essímo maior valor.

Formato da entrada:

Uma linha contendo um valor n que corresponde a quantidade de inteiros da lista e um i (1 <= i <= n) que corresponde ao valor.

Uma linha contendo n valores inteiros.

Formato da saída:

O valor do i-essímo maior.

Gerado automaticamente.
