Descrição do Problema (problem.txt)

Título: Equação do segundo grau Topicos do problema: ponto flutuante; lógica matemática; ponteiro; Tempo limite de execução: 1 segundo(s) Descrição:

Fulaninho tem preguiça de resolver equações do segundo grau e queria criar um programa queria fizesse isso por ele.
Para resolvermos uma equação do segundo grau usamos os coeficientes da equação na fórmula de Bhaskara.


Faça um programa com a função int raizes(float A, float B, float C, float * x1,float * x2) que receba os coeficiente A, B e C e retorne a quantidade de soluções que a equação tem. No fim o programa deve imprimir as raízes e quantidade de raízes.

Formato da entrada:

O seu programa deve receber os três coeficientes A, B e C nessa ordem, separados por espaços.

Lembrando que a equação é escrita no seguinte formato: Ax² + Bx + C = 0

Formato da saída:

Na saída devem estar presentes as raízes existentes, escritas com 2 dígitos, e a quantidade de soluções (em linhas diferentes).

Se não houver raízes reais, imprimir apenas o número de soluções, no caso 0.

Gerado automaticamente.
