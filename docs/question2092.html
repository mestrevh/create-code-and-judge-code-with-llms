L3Q6 - Sentimentos São

Tempo limite: 1 segundo(s)

Depois de o encanto ser quebrado, e todos os móveis voltarem a ser pessoas, Maurice, o pai de Bela, teve um aumento muito grande na demanda na loja onde vendia suas invenções, desesperado, ele correu até Bela para que ela o ajudasse a organizar o estoque de seus produtos, para que ele pudesse ter tempo para conseguir montar todas as encomendas.

Você então deve criar um programa que ajude Bela a controlar o estoque da loja. Seu programa deve registrar todas as alterações feitas no estoque a partir dos comandos:
  • Registro: É seguido pelo nome do produto a ser registrado. (reportar: "Produto ja registrado" se o produto já tiver sido registrado anteriormente e "Produto registrado com sucesso", se o registro tiver ocorrido!)
  • Venda: É seguido pelo nome do produto e o número de peças vendidas. (reportar: "Produto nao registrado" se o produto nao tiver sido registrado, "Sem estoque para a Venda" se não houver estoque suficiente e "Venda realizada com sucesso", se ela tiver sido completa!)
  • Recarga: É seguido pelo nome do produto a ser recarregado e o número de peças da recarga. (reportar: "Produto nao registrado" se o produto não tiver sido registrado e "Recarga realizada com sucesso").


Formato da Entrada

A primeira linha da entrada é composta por um inteiro N, referente a quantidade de comandos.

Seguem os N próximos comandos, no seguinte formato:
  • Uma linha com uma string representando o comando (Registro, Venda ou Recarga);
  • Uma linha com uma string referente ao nome do produto, que pode conter espaços (Apenas se o comando for válido);
  • Uma linha com um número inteiro (pode ser negativo, neste caso o comando é inválido) referente a quantidade de produtos (Apenas em comandos que precisam de quantidade e se o produto estiver cadastrado).

Formato da Saída

Para cada comando, se o mesmo for inválido, é necessário imprimir a string "Comando invalido". Caso o comando exista, imprimir uma linha com o resultado do comando, de acordo com a descrição acima.

Após as linhas dos resultados dos comandos, imprimir uma linha para cada produto registrado com o nome e a quantidade do mesmo, separados por um ": " (dois pontos e espaço).
