Descrição do Problema (problem.txt)

Título: Média dos elementos pares Topicos do problema: array; acumulador; repetição; Tempo limite de execução: 1 segundo(s) Descrição:

Dado um vetor de números reais, calcule a média dos elementos pares positivos.

Formato da entrada:

Uma linha contendo uma sequência de números reais.

Formato da saída:

Um valor real com duas casas decimais (incluir um final de linha após o valor)

Se não houver elementos pares positivos, imprimir o inteiro -1

Gerado automaticamente.
