Título: Função para conversão de graus para radianos Topicos do problema: geometria computacional; array; função; Tempo limite de execução: 1 segundo(s) Descrição: 
Criar uma função para ler um vetor cujo número de elementos (dimensão) é passado como argumento. Crie uma outra função que receba um vetor e faça a conversão de graus para radianos de todos os elementos desse vetor. Essa função deve retornar um vetor com os valores convertidos. Escrever também o módulo principal. Observação: use o valor de pi com a maior precisão possível.
Formato da entrada: 
Um número inteiro N na primeira linha correspondente ao número de elementos do vetor.
N números reais em coluna correspondentes aos elementos do vetor de ângulos em graus.
Formato da saída: 
N números reais em coluna formatados com 5 casas decimais correspondentes aos elementos do vetor de ângulos em radianos.
