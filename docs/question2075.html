Título: Soma de somas Topicos do problema: repetição; Tempo limite de execução: 1 segundo(s) Descrição:

Uma existência adora matemática, recentemente ela está se divertindo com somatórios e se deparou com um somatório de somatórios:


Ela está certa de que encontrará propriedades interessantes nesse somatório, mas ela gostaria de continuar bricando com os outros, então, ela pediu para que você implementasse uma calculadora para essa expressão, assim ela poderá achar padrões mais facilmente.

Somatórios são somas em um intervalo, no exemplo abaixo, está-se somando de 0 até n, daí, todo "i" que estiver depois do símbolo do somatório é substituído e somado da forma como é mostrada na imagem.


Exemplo:
n = 2, m = 2
( (sin(0) + cos(0)) + (sin(1) + cos(1)) + (sin(2) + cos(2)) ) + ( (sin(1) + cos(1)) + (sin(2) + cos(2)) ) + ( (sin(2) + cos(2)) ) = 5.242998

Formato da entrada:

A entrada consistirá de dois inteiros n e m.

Formato da saída:

A saída deverá conter a resposta da expressão com 6 casas decimais.
