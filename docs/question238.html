Descrição do Problema (problem.txt)

Título: Caminho no tabuleiro Topicos do problema: estrutura de dados; busca completa; backtracking; Tempo limite de execução: 1 segundo(s) Descrição:

Considere um tabuleiro com n X n posições, uma função de custo c(i,j) que retorna o custo associado a posição i,j (onde i é a linha e j é a coluna).
Por exemplo:
O tabuleiro abaixo possui n igual a 5:

 

Por exemplo, nesse tabuleiro, c(0,2) = 5.

O seu objetivo é escolher o menor caminho entre a primeira linha (linha 0) e a última (linha n-1). Um caminho é determinado pela soma dos valores das posições que foram visitadas (função c) até chegar em uma determinada posição. É possível começar em qualquer posição da primeira linha. Ao escolher uma posição, só é possível se mover nas diagonais ou para baixo. Por exemplo, se a posição escolhida for a (0,2), é possível se mover para as posições (1,1) , (1, 2) ou (1,3).

Formato da entrada:

Você receberá um número inteiro n (1<=n<=1000), indicando o tamanho da matriz quadrada n X n. Depois você receberá os valores do tabuleiro, representando a função de custo de cada posição. Desta forma, você receberá n X n valores. Os valores serão entregues no formato de matriz. Por exemplo, a entrada para o tabuleiro acima seria:

5
10 20 5 50 60
80 6 7 0 80
3 5 7 8 2
7 6 1 1 4
6 7 4 7 8

Formato da saída:

Você deve imprimir o custo do menor caminho em uma linha. Na linha seguinte, você deve imprimir o caminho seguido para chegar à última linha do tabuleiro. O caminho deve ser impresso de trás para frente. Por exemplo, para o tabuleiro usando como exemplo, a saída seria:

12
[4,2] <- [3,3] <- [2,4] <- [1,3] <- [0,2]

O caminho representa os índices do tabuleiro. Nesse caso, o caminho começa na posição [0,2], segue para a [1,3], depois a [2,4] e assim por diante.
