Descrição do Problema (problem.txt)

Título: Protótipo de Ctrl+F Topicos do problema: multiplas entradas; eof; array bidimensional; string; busca; Tempo limite de execução: 1 segundo(s) Descrição:

Você está na rota em direção a academia e decide fazer um protótipo do Ctrl+F, que é uma ferramenta que permite pesquisar uma palavra (ou parte dela) num texto. Ao digitar a palavra que deseja buscar, o comando irá buscar todas as ocorrências do mesmo num texto e destacá-las de alguma forma.


Basicamente, você irá receber uma matriz de strings (um texto), e seu dever é receber as palavras de interesse e indicar em qual índice da matriz essa palavra ocorreu.


OBS:

  • Mesmo que uma dada palavra apareça mais de uma vez numa string, você deve indicar apenas que ela está presente naquela posição uma vez.
  • Você deve indicar todas as posições de uma dada palvra antes de pesquisar pela próxima.
  • Considere letras maiúsculas diferentes das minúsculas


Dica: Tente usar um array tridimensional.

Formato da entrada:

n -> tamanho da matriz n x n (n>1)

n^2 strings de até 25 letras

X palavras de interesse, onde X é um número indefinido.

Formato da saída:

A posição de cada ocorrência na matriz com o seguinte formato:


(i,j)

(a,b)

.

.
