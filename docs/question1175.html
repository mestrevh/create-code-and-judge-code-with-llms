Descrição do Problema (problem.txt)

Título: L1Q3 - Falha nos códigos Topicos do problema: decisão; string; Tempo limite de execução: 1 segundo(s) Descrição:

Antes de sair do laboratório do Professor Carvalho, Biu percebe que o sistema de cadastro de treinadores do professor tem uma falha muito grave de privacidade. Quando um certo treinador vai pela primeira vez em seu laboratório, é gerado um código de 10 caracteres que o identifica para futuros contatos.

Porém, esse código tem uma peculiaridade. O sétimo caractere do código identifica qual Pokémon foi escolhido pelo jogador, o que compromete a segurança do código, que deveria ser apenas uma identificação do treinador e não de sua escolha. Os Pokémons que podem ser identificados pelo código são:

  • Bulbassauro
  • Charmander
  • Squirtle


Sua tarefa é escrever um programa que, dado um código único de 10 caracteres, mostra qual Pokémon está relacionado a ele, sabendo que a identificação é feita a partir da letra inicial de seu nome (Bulbassauro - b; Charmander - c; Squirtle - s).



Formato da entrada:

C

  • C - string


A única linha é composta de pelo código identificador do Pokémon.


obs: Considere que todos os códigos que serão testados têm, exatamente, 10 caracteres.

Formato da saída:

P

  • P - string


A linha única da saída é composta pelo nome do Pokémon identificado. Caso não seja um código válido, é preciso que seu programa apresente a mensagem "Codigo Invalido".

Gerado automaticamente.
