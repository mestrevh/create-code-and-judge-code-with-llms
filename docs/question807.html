Tradução de Texto

Tempo limite: 1 segundo(s)

Escreva um programa que leia os dados de um dicionário qualquer e faça a tradução de um texto de uma língua para a outra de acordo com o dicionário fornecido.


Formato da Entrada

Uma linha contendo um número inteiro (N) de palavras do dicionário. Em seguida, as N linhas que representam o conteúdo do dicionário. Em cada uma destas linhas, há uma palavra seguida dos caracteres ‘ =>’ (espaço, igual, maior e espaço) e da palavra para a qual ela deve ser traduzida.

Após carregar os dados do dicionário, o programa deve ler uma sequência de linhas de qualquer tamanho. Estas linhas devem ser traduzidas conforme os dados do dicionário. O programa acaba quando fo rencontrada uma linha que contém apenas o caractere ‘*’(asterisco).


Formato da Saída

Para cada linha (frase a ser traduzida) da sequência de linha(s)fornecida(s) na entrada, deve ser impressa uma linha com a tradução correspondente.
