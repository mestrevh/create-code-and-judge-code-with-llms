Descrição do Problema (problem.txt)

Título: Counting Pairs Topicos do problema: lógica matemática; requer solução eficiente; Tempo limite de execução: 1 segundo(s) Descrição:

Ayalla tem um array A com N inteiros A[0], A[1], ..., A[N - 1]. Em um certo dia, ele estava brincando com esse array e fez várias perguntas a si mesmo. Cada pergunta pode ser descrita da seguinte maneira:

Ele pensou em um número inteiro K e se perguntou quantos pares (i, j) existem no array A, tal que, (A[i] + A[j]) é divisível por K e (i < j).

Entretanto, Ayalla não fazia a mínima ideia de como responder essas perguntas. Sabendo disso, você pode responder as perguntas para Ayalla?

Formato da entrada:

A primeira linha da entrada terá um inteiro N indicando o tamanho do array A.

A próxima linha vai N inteiros A[0], A[1], ..., A[N - 1], onde A[i] é o i-ésimo elemento do array.

Em seguida, vai ter um inteiro Q, indicando quantas perguntas Ayalla fez.

As próximas Q linhas vão ter um inteiro K.

1 <= N <= 10^5

1 <= A[i] <= 10^5

1 <= K <= 2 * 10^5

Formato da saída:

Q linhas, em cada uma delas, você deve imprimir a resposta para a i-ésima pergunta.
