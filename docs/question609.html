Descrição do Problema (problem.txt)

Título: Grupos sanguíneos Topicos do problema: maratona; Tempo limite de execução: 2 segundo(s) Descrição:

Existem 4 possíveis tipos sanguíneos em humanos: AB, A, B e O. Eles significam que as células vermelhas possuem, respectivamente, antígenos dos tipos A e B, somente A, somente B e nenhum antígeno.

Nosso grupo sanguíneo é determinado por dois alelos no nosso DNA. Cada alelo é do tipo A, B ou O. A seguinte tabela lista todas possíveis combinações de alelos que alguém pode ter para cada grupo sanguíneo.


Nós herdamos exatamente um alelo de cada um dos nossos dois pais. Logo, dado o grupo sanguíneo dos nossos pais, é possível afirmar com certeza se algum grupo sanguíneo é possível para qualquer filho. Por exemplo, se o grupo sanguíneo do pai é AB e a mãe B, então as possíveis combinações de alelos são, respectivamente, {AB} e {OB, BB}. Como a ordem dos alelos não importa, as possíveis combinações dos alelos dos filhos dessa casal são: {OA, AB, OB, BB}. Isso significa que os grupos AB, A e B são possíveis para os descendentes, mas o grupo O não é possível.

Pois bem, mas e se a vida na terra evoluísse de tal forma que uma pessoa pudesse ter 3 pais, 3 alelos e 3 tipos diferentes de antígenos? As combinações de alelos iriam ser como na tabela:

Se o grupo sanguíneo dos 3 pais são A, BC e O, então todos os grupos sanguíneos seriam possíveis em seus descendentes, com exceção dos grupos BC e ABC.

O universo é vasto! Pode ser que exista em algum lugar no cosmos alguma forma de vida cujo indivíduos possuam N pais, N alelos e N diferentes tipos de antígenos.

Dado o grupo sanguíneo de N pais e uma lista de Q grupos sanguíneos, seu programa deve determinar, em relação aos descendentes desses pais, quais dos grupos são possíveis e quais não são.

Formato da entrada:

A primeira linha contém dois inteiros N e Q, representando respectivamente o número de pais (e consequentemente alelos e tipos de antígenos) e o número de consultas (1<= N <= 100 e 1<= Q <= 40). Cada uma das próximas N linhas descreve o grupo sanguíneo de um pai. Depois disso, as próximas Q linhas descrevem o grupo sanguíneo a ser testado. Os tipos de antígenos são identificados por inteiros distintos de 1 a N. Cada linha que descreve um grupo sanguíneo contém um inteiro B indicando o número de tipos de antígenos no grupo sanguíneo ( 0  <= B <= N), seguido de B diferentes inteiros C1, C2, ..., CB, representando os tipos de antígenos presentes no grupo sanguíneo ( 1 <= Ci <= N, para i = 1, 2, ..., B).

Formato da saída:

Para cada uma das Q consultas, imprima uma linha com a letra 'Y' se o grupo sanguíneo é possível como descendente dos pais dados. Caso contrário imprima a letra 'N'.

Escreva os resultados na mesma ordem em que as consultas aparecem na entrada.

Gerado automaticamente.
