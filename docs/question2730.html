Título: PROGCOMP - Funções - C Topicos do problema: função; Tempo limite de execução: 1 segundo(s) Descrição:

Construa um programa que use uma função, criada por você, que recebe como parâmetro um inteiro n e exibe, a partir da posição atual do cursor no console, uma linha composta por n asteriscos.

Formato da entrada:

Um número inteiro n (0 < n < 100).

Formato da saída:

n asteriscos na mesma linha.
