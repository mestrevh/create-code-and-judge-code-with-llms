Happy Number

Tempo limite: 1 segundo(s)

Willy acabou de descobrir o conceito de Happy Number e ficou fascinado com esses números, então ele pediu que você o ajudasse a dizer se um número dado é um Happy Number.

Um número é dito Happy Number quando o substituímos pela soma do quadrado dos seus dígitos repetidas vezes e acabamos chegando ao número 1, exemplo:

7: 7² = 49: 4² + 9² = 97: 9² + 7² = 130: 1² + 3² + 0² = 10: 1² + 0² = 1. Logo 7 é um Happy Number.

Uma forma fácil de descobrir que um número não é Happy Number é quando ao repetirmos o processo de substituição chegamos a um número que já tínhamos chegado antes, exemplo:

37: 3² + 7² = 58: 5² + 8² = 89: 8² + 9² = 145: 1² + 4² + 5²  = 42: 4² + 2² = 20: 2² + 0² = 4: 4² = 16: 1² + 6² = 37. Logo 37 não é um Happy Number


Agora é sua vez de ajudar Willy, dado um número N diga se ele é umHappy Number.

Formato da Entrada

Uma única linha contendo um inteiro N.

1 <= N <= 1000

Formato da Saída

Uma única linha contendo S, caso seja um HAPPY Number, ou contendo N, caso não seja um Happy Number.
