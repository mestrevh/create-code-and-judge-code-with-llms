Descrição do Problema (problem.txt)

Título: Soma - Inteiros em linha Topicos do problema: array; acumulador; string; Tempo limite de execução: 1 segundo(s) Descrição:

Seu programa deve receber uma sequência de N números inteiros separadas por espaços “ “ e mostrar na tela a soma destes valores.

Formalmente falando, seu programa deve exibir a soma de todos os elementos Ni.

Por exemplo:

   "1 2 3 4 5 6 -1 2"

Resultado:

   A soma esperada destes números é 22.


Os números serão sempre separados por exatamente 1 espaço em branco.


Formato da entrada:

Uma string S, contendo N números separados por espaços em branco.

0 > N < 10³

-10³ > Ni < 10³

Formato da saída:

Seu programa deve exibir a soma de todos os elementos da sequência N, ou seja, o somatório de Ni.

Gerado automaticamente.
