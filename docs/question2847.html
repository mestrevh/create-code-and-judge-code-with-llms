Descrição do Problema (problem.txt)

Título: Buracos de minhoca Topicos do problema: grafo; Tempo limite de execução: 3 segundo(s) Descrição:

No ano 2163, buracos de minhoca foram descobertos. Um buraco de minhoca é um túnel através do espaço-tempo que conecta dois sistemas solares. Buracos de minhoca tem algumas propriedades:

  • São de mão única.
  • O tempo que leva para atravessá-lo é desprezível.
  • Conectam dois pontos do espaço, cada ponto situado em um sistema solar.
  • Um sistema solar pode ter mais de um buraco de minhoca.
  • Para cada sentido, existe no máximo um buraco de minhoca conectando um par de sistema solares.
  • Não existem buracos de minhoca que começam e terminam no mesmo sistema solar.
Todos os buracos de minhoca possuem uma diferença de tempo constante entre a entrada e a saída. Por exemplo, algum buraco de minhoca pode fazer com que quem o atravesse saia 15 anos no futuro. Outro buraco de minhoca pode fazer com que quem o atravesse saia 42 anos no passado.


O físico brilhante da Terra quer usar os buracos de minhoca para estudar o Big Bang. Como um warp drive ainda não foi inventado, só é possível viajar entre sistemas solares utilizando os buracos de minhoca.

O cientista quer achar um ciclo de buracos de minhoca, em algum lugar do Universo, que leve-o para o passado. Dessa forma, ao viajar várias vezes por esse ciclo, ele pode voltar no tempo o suficiente para assistir o Big Bang com seus próprios olhos. Faça um programa que ajude o físico a descobrir se é possível que esses ciclos existam.

Formato da entrada:

A primeira linha da entrada terá um inteiro c, representando a quantidade de casos de teste que virão. Cada caso de teste inicia com dois inteiros n (1 <= n <= 1000) e m (0 <= m <= 2000), indicando o número de sistemas solares e o número de buracos de minhoca. Os sistemas solares são enumerados de 0 (nosso sistema solar) até n - 1. Após isso, virão m linhas, cada uma com três inteiros x, y e t, indicando que existe um buraco de minhoca no sistema solar x que leva para o sistema solar y e para t (-1000 <= t <= 1000) anos no futuro/passado.

Formato da saída:

Para cada caso de teste, deverá ser impresso "possible", se for possível o físico observar o Big Bang, ou "not possible", se não for possível.
