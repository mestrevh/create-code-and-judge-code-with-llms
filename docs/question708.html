A transportadora

Tempo limite: 1 segundo(s)

Uma empresa transportadora possui um galpão que recebe constantemente produtos a serem entregues por uma frota de N de caminhões T[0], ... ,T[N-1]. Esses produtos são carregados nos caminhões na mesma ordem em que chegam ao galpão e os caminhões são, em seguida, liberados para efetuar as entregas. Cada caminhão T[i] possui uma capacidade máxima de carga de C[i] Kg.
 
Cada vez que o caminhão vai ser abastecido com itens para entrega, o transportador deve adicionar produtos enquanto o próximo item não exceder a capacidade de carga disponível ou, naturalmente, até que o galpão esteja totalmente vazio. No primeiro caso, mesmo que hajam outros produtos que ainda caibam no caminhão, o transportador deve parar de carregar.

Os produtos são entregues aos seus destinatários na ordem inversa a que foram carregados no caminhão (o último item carregado é o que está mais próximo à porta e, logo, o primeiro a ser entregue). Eventualmente, um caminhão pode retornar ao depósito para ser recarregado, mesmo que ainda possua itens a serem entregues. Nesse caso, o transportador deve continuar o carregamento da mesma forma descrita acima.
 
Para simular o funcionamento da empresa, são necessárias as seguintes operações:
 
1) ADD w: adiciona ao galpão um produto de peso w (w inteiro).
 
2) LOA i: carrega o caminhão T[i] de acordo com o procedimento acima.
 
3) DEL i: entrega(remove) um item do caminhão T[i].
 
4) INF i: indica a quantidade e o peso total dos itens presentes no caminhão T[i].

Formato da Entrada

O arquivo de entrada é constituído por X casos.
Cada caso possui, inicialmente, uma sequência de inteiros
 
N
C[0]
...
C[N-1]
 
onde

N = número de caminhões
C[i] = capacidade de carga (peso máximo suportado) do caminhão T[i].
 
Seguem-se várias linhas, cada uma correspondendo a uma operação numa das seguintes formas

ADD w
LOA i
DEL i
INF i

com os significados descritos acima.
Ao final de cada caso, há uma linha em branco.

IMPORTANTE: Os caminhões são supostos vazios no início de cada caso.

LIMITES:
0 < X < 1000
0 < N < 1000
10000 < w < 100000
100000 < C[i] < 1000000, para i=0,...,N-1

Formato da Saída

Para cada operação no arquivo de entrada, imprimir uma linha com a saída correspondente
 
ADD w)
imprime
M S
 
onde
M = número de itens no galpão
S = soma do peso de todos os itens do galpão.
 
LOA i)
imprime
i L
 
onde
i = número do caminhão
L = quantidade de itens no caminhão T[i] após o carregamento.
 
DEL i)
imprime
i L

onde
i = número do caminhão
L = quantidade de itens no caminhão T[i] após a entrega.
 
INF i)
imprime
i L K

onde
i = número do caminhão
L = quantidade de itens presentes no caminhão T[i]
K = peso total dos itens presentes no caminhão T[i]

Ao final de cada caso, deve ser impressa uma linha em branco.
