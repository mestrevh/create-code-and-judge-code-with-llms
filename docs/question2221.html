Descrição do Problema (problem.txt)

Título: Entrando pelo Cano Topicos do problema: decisão; string; Tempo limite de execução: 1 segundo(s) Descrição:

No começo de sua jornada, seguindo pelos canos, Mário e Luigi se depararam com uma trifurcação logo ao sair do Reino dos Cogumelos. O problema é que as descrições das placas de cada cano estão meio apagadas e eles não sabem exatamente por qual deles eles devem seguir para chegar a fortaleza do Bowser.




Mesmo com as placas pouco legíveis, Mário consegue identificar algumas letras em cada uma delas. Cabe a você descobrir o cano que eles devem seguir, fazendo um programa que, dado as letras legíveis de cada placa, verifique com essas letras se podemos formar a palavra "bowser". Aí eles saberão que é por esse cano que devem seguir.

Formato da entrada:

Três entradas correspondentes a cada um dos canos ( Esquerda, Meio e Direita ), nessa ordem.

S

  • S - String correspondente as letras legíveis.

Obs: As letras legíveis são sempre minusculas.  

Formato da saída:

C

  • C - String com a direção que Mário deve seguir.

Obs: É garantido que só uma das direções satisfaz a condição.

Gerado automaticamente.
