Descrição do Problema (problem.txt)

Título: Quociente de uma divisão inteira Topicos do problema: multiplas entradas; lógica matemática; Tempo limite de execução: 1 segundo(s) Descrição:

Crie um programa que determine o quociente de uma divisão inteira. Por exemplo, se a entrada for os valores inteiros 30 e 7, o programa deve fornecer como saída o valor iteiro 4.


obs: não é permitido utilizar comandos diretos para o cálculo desse quociente, a exemplo do comando // do python.

Formato da entrada:

Dois números inteiros representando o dividendo e divisor, respectivamente, da divisão inteira.

Formato da saída:

O número inteiro representando o quociente da divisão inteira.

Gerado automaticamente.
