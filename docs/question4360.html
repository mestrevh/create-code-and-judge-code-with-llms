Descrição do Problema (problem.txt)

Título: É uma nota válida? Topicos do problema: decisão; Tempo limite de execução: 1 segundo(s) Descrição:

Alunos precisam ter notas nas disciplinas que estão cursando. Mas, as notas precisam ser válidas, isto é, estar no intervalo possível.

Faça um programa para verificar se a nota digitada pelo professor está no intervalo entre 0 e 10, inclusive.

Caso não esteja, apresente uma mensagem de erro.

Formato da entrada:

Leia o valor de uma nota (tipo real) e apresente uma mensagem de “erro” caso ela não seja uma nota válida. A nota deve ser menor ou igual a dez e maior ou igual a zero.

Formato da saída:

Mensagem 1: "A nota digitada foi: <nota>"

Mensagem de erro caso a nota não seja válida: 'A nota nao esta no intervalo [0,10]'

Gerado automaticamente.
