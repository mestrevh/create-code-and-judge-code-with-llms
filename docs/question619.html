Descrição do Problema (problem.txt)

Título: Ambrósio e os números especiais Topicos do problema: segment tree; números binários; binary indexed tree; requer solução eficiente; Tempo limite de execução: 1 segundo(s) Descrição:

Ambrósio gosta muito de números especiais. Um número é tido especial se é não negativo, e seu primeiro digito é igual ao ultimo.

Ex: 0, 3, 5005, 5345, 111, são números especiais , já o número 1000 não é especial.

Ambrósio tem um array A1, A2, A3 .. AN. Ele quer que você execute as seguintes Q operações:

1 L R :  encontre o número de números especiais no intervalo [L, R].

2 i K : substitua o i-ésimo elemento do array A por K. ou seja Ai = K;

Formato da entrada:

A primeira linha é composta por dois inteiros N e Q, que indicam o tamanho do array e a quantidade de operações respectivamente.

Cada operação consiste de 3 inteiros como diz a descrição. O primeiro desses 3 inteiros indica qual operação deve ser feita.

1 ≤ N , Q ≤ 2*105
-1018 ≤ Ai ≤ 1018 
1 ≤ L,R ≤ N
-1018 ≤ K ≤ 1018

Formato da saída:

Para cada operação do tipo 1, imprima a quantidade de números especiais contidos no intervalo definido , um por linha.

Gerado automaticamente.
