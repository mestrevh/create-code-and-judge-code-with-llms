Descrição do Problema (problem.txt)

Título: Formatando número real como moeda Topicos do problema: ponto flutuante; string; formatação; Tempo limite de execução: 1 segundo(s) Descrição:

Escreva um programa que leia um número real (ponto flutuante) não negativo e imprima na linha seguinte o mesmo formatado como um valor monetário, usando ponto de milhar e vírgula para separar a parte decimal. Utilize 2 casas decimais.Para resolução da questão, você deve utilizar uma função chamada formatar_dinheiro que recebe como único parâmetro o valor real. Esta função deve retornar uma string com o valor no formato esperado para a impressão.

Formato da entrada:

Uma linha contendo um número real (ponto flutuante) que representa o valor monetário.

Formato da saída:

O valor monetário no formato abaixo (com um espaço em branco entre o $ e o primeiro dígito):R$ 999.999.999,99 

Gerado automaticamente.
