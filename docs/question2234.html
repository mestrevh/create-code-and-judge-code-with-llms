Descrição do Problema (problem.txt)

Título: Inversion Count (iniciante) Topicos do problema: array; repetição; Tempo limite de execução: 1 segundo(s) Descrição:

Seja A um array de inteiros positivos, e i e j posições desse array, é dito que um par (i, j) é inversível caso i < j e A[i] > A[j]. Você deve descobrir quantos pares inversíveis existem em A.


Neste problema você não deve se preocupar com a eficiência do algoritmo. 
Caso queira um desafio maior, procure a questão 2045

Formato da entrada:

Um inteiro N representando o tamanho do array, seguido de N inteiros M contendo seus elementos.

0 < N < 500
0 <= M < 10^6


Formato da saída:

O número de inversões no array.

Gerado automaticamente.
