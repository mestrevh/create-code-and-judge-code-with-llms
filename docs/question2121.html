P2 2017.2 - Trivialus

Tempo limite: 1 segundo(s)

Na palestra, após dar uma breve introdução sobre a teoria de matrizes, Sílvio começou com um exemplo prático: “Na agricultura, podemos utilizar a magia Trivialus para dividir uma área de solo em porções quadradas de tamanhos iguais e identificar quais porções estão boas para cultivo.” Cada quadrado pode ter 3 (três) cores diferentes, de acordo com a qualidade do solo, sendo:

  • Verde: excelente;

  • Amarelo: danificado;

  • Vermelho: danificado e infestado;


Como no exemplo abaixo:



Para recuperação do terreno, deve-se primeiro aplicar um repelente de lesmas, que resolve o problema de infestação dos terrenos danificados e infestados, tornando-os apenas danificados.


Após a aplicação do repelente, o bruxo deve dizer a palavra Trivialus e ter em mente o tamanho do terreno. Nesse momento, o terreno é dividido em quadrados e tem início a recuperação das áreas danificadas, que se tornam excelentes.


Hagrid precisa recuperar alguns terrenos. Para isso, ele foi à Travessa do Tranco comprar repelentes. Após resolver o problema de infestação, ele pediu ajuda para a Profa. Sprout para aplicar o feitiço Trivialus. Como a professora estava com muita raiva de Hagrid, a qualidade da magia foi comprometida e apenas algumas porções foram atingidas pelo feitiço.


Sua tarefa é escrever um programa que ajude Hagrid a computar o lucro que irá obter após sua colheita, considerando todos os seus gastos com repelentes e ganhos com áreas cultivadas.


OBS:

  • o repelente de lesmas custa 3 Galeões por área infestada;

  • Hagrid só pode plantar em áreas que estão excelentes;

  • ao final da colheita, ele consegue 4 Galeões por área cultivada.



É obrigatório o uso de array bidimensional para a solução da questão.

Formato da Entrada

L C

Qij

S

X Y


L - número inteiro positivo (2 <= N <= 1000)

C - número inteiro positivo (2 <= M <= 1000)

A primeira linha é composta pelo número de linhas (L) e número de colunas (C) da matriz que representa as áreas do terreno da plantação de Hagrid.


Qij - string { “verde”, “amarelo”, “vermelho” }

cada uma das próximas L*C linhas contém uma cor, representando a qualidade de cada área do terreno;


S - número inteiro positivo (1 <= S <= L * C)

A próxima linha contém a quantidade de áreas atingidas pelo feitiço da professora Sprout.


X - número inteiro positivo (1 <= X <= L)

Y - número inteiro positivo (1 <= Y<= C)

As próximas S linhas são compostas pelas coordenadas X e Y de cada área atingida pelo feitiço.


OBS:

  • só vai haver uma tentativa de recuperação em cada área;

  • o feitiço pode atingir uma área que já é excelente, não tendo efeito;

  • a cor informada será sempre “verde”, “amarelo” ou “vermelho".

Formato da Saída

V


V - número inteiro

A linha única é composta pelo lucro em Galeões que Hagrid irá obter, após a ajuda da professora Sprout, com a colheita de todas as áreas excelentes e gastos feitos com repelentes.  


OBS: Caso Hagrid tenha prejuízo, é necessário imprimir apenas o texto: “prejuizo”.
