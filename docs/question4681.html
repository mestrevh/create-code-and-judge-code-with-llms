Descrição do Problema (problem.txt)

Título: Dominó IV Topicos do problema: array; substrings; string; Tempo limite de execução: 5 segundo(s) Descrição:

Este problema envolve a decodificação de uma mensagem relacionada ao jogo de dominó entre dois amigos, João Pedro e Vinícius, conhecidos no IF como os maiores "ladrões" de dominó. A tarefa é verificar se Vinícius (você) pode ajudar João Pedro na próxima jogada com base nas peças que ele possui e nas necessidades apresentadas por João Pedro.

Ações de João Pedro:

  • Fecha as duas pontas no valor 'x' -> (código: `manbr` seguido de números). 
Se de um lado do jogo possui o número 'x' e do outro lado o número 'y' ('y' pode ser igual a 'x'), essa jogada deixa os dois lados iguais para um dos números (a,a ou b,b)
  • Levanta o número 'x' -> (código: `rpf`). 
Levantar se refere a colocar um número específico na mesa (em qualquer lado)
  • Come o número 'x' para valor diferente -> (código: `rrs`).
Comer se refere a tirar o número 'x' da mesa e colocar qualquer outro número no lugar
  •  Deixa a exata combinação que está -> (código: `dsfs`). PS: Observe que esta ação não necessita de um número para ser realizada 
Manter os números que já estão abertos no jogo


OBS: As jogadas ignoram os efeitos para o próximo jogador (joão pedro já calculou tudo)

Formato da entrada:
  1. Um número inteiro `n` que representa a quantidade de peças de dominó de Vinícius.
  2. Uma lista de `n` inteiros representando os valores das peças de dominó que Vinícius possui, organizados em ordem crescente.
  3. Dois números inteiros, representando os valores das pontas abertas do dominó para a próxima jogada, também dados em ordem crescente.
  4. Uma string contendo uma sequência de letras seguida de seis dígitos numéricos. O formato da string é sempre algo como "manbr201201", onde:
  •  As letras representam o código de uma ação desejada. 
  • Os seis dígitos numéricos são usados para calcular o valor necessário para a ação de João Pedro.


  • O número 'x' é dado por uma divisão entre dois números: 
`num1` é o número formado pelos três primeiros dígitos da sequência numérica.
`num2` é o número formado pelos três últimos dígitos da sequência numérica.    -
O valor de `x` é o resultado da divisão de `num1` por `num2` (ou seja, `x = num1 / num2`).
Formato da saída:

Se a jogada for possível, imprima a mensagem: `

"ai maezinha\n"

Caso contrário, imprima: `

"cade o douze\n"

Gerado automaticamente.
