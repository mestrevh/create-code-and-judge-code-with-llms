Descrição do Problema (problem.txt)

Título: Masculino e Feminino Topicos do problema: decisão; Tempo limite de execução: 20 segundo(s) Descrição:

Escreva um programa que verifique se uma letra digitada é "F" ou "M".

Conforme a letra escrever: F - Feminino, M - Masculino, Sexo nao definido.

Formato da entrada:

Será inserida uma String ("F" ou "M")

Formato da saída:

Será retornado Feminino para entrada F, Masculino para entrada M e "Sexo nao definido" caso contrário.

Gerado automaticamente.
