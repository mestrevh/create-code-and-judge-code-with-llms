Descrição do Problema (problem.txt)

Título: Beiju Text Topicos do problema: estrutura de dados; caractere; ponteiro; Tempo limite de execução: 1 segundo(s) Descrição:

Você está digitando um texto longo com um teclado quebrado. Bem, ele não está tão quebrado assim. O único problema é que às vezes a tecla "Home" ou "End" é pressionada automaticamente (internamente).
Você não percebe esse problema, já que está concentrado na digitação e nem sequer ligou o monitor! Depois que você termina de digitar, é possível ver o texto na tela (caso ligue o monitor). Em chinês, podemos chamar isso de Beiju.
Sua tarefa é descobrir qual é o texto Beiju.


ATENÇÃO: É IMPRESCINDÍVEL A UTILIZAÇÃO DA ESTRUTURA DE DADOS LINKED LIST

Formato da entrada:

Haverá uma única string de input (sem espaço e máximo de 500 caracteres).

Os caracteres podem ser letras, sublinhados ('_') e dois símbolos especiais:

  • '[' representa que a tecla Home (início do texto) foi pressionada internamente
  • ']' representa que a tecla End (fim do texto) foi pressionada internamente
Formato da saída:

Imprima a string final. (Beiju Text)

Gerado automaticamente.
