Descrição do Problema (problem.txt)

Título: Sequência de figuras Topicos do problema: decisão; lógica matemática; Tempo limite de execução: 1 segundo(s) Descrição:

Observe a sequência de figuras a seguir.


Pense: Qual a configuração da figura 4? Existe uma quinta figura diferente das 4 anteriores?

Joãozinho quer que você o ajude a escrever um programa que dado um número inteiro (n) que corresponda à posição de uma figura, apresente a configuração da figura.

Por exemplo, se n for 3, significa que a figura resultante é a 3. Se n for 5, significa que a figura resultante é a figura 1. 


Formato da entrada:

n, um número inteiro >= 1

Formato da saída:

  • 4 mensagens, se o n for válido, indicando a cor em cada uma das posições: esquerda superior, direita superior, esquerda inferior e direita inferior.
  • Caso o n seja inválido, a mensagem: O número informado é menor que 1.

Gerado automaticamente.
