Descrição do Problema (problem.txt)

Título: Os Ímpares de Josué Topicos do problema: eof; ordenação; Tempo limite de execução: 1 segundo(s) Descrição:

Josué, O Carteiro, foi pego de bobeira brincando com as etiquetas das encomendas pelo seu chefe, então foi transferido para outro setor. No setor que ele está trabalhando agora, ele recebe várias cartas para diferentes casas, mas percebeu que coincidentemente que em todas as remessas de cartas, as casas recebiam duas cartas. Porém, de vez em quando, em uma grande remessa apenas uma casa recebia uma quantidade impar de cartas. Então ele resolveu pedir para você fazer um programa que lê todos os números das casas e diz qual é o número que aparece uma quantidade ímpar de vezes nesse amontoado de números, caso apareça. 

Formato da entrada:

Entre 1 ≤ n ≤ 100000 números inteiros que variam entre 1 ≤ n ≤ 100000.

Obs: Os números podem estar em qualquer ordem.

Formato da saída:

Em uma linha, a lista dos números lidos ordenados e na outra linha o número que apareceu uma quantidade ímpar de vezes, caso tenha um número que apareceu uma quantidade ímpar de vezes, ou 0 caso não tenha aparecido.

Obs: Após o ultimo número na lista ordenada, coloque um espaço em branco antes da quebra de linha.

Gerado automaticamente.
