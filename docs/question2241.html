Descrição do Problema (problem.txt)

Título: L2Q2 - Só quer vrau Topicos do problema: repetição; Tempo limite de execução: 1 segundo(s) Descrição:

Pepeu estava ouvindo sua playlist de funk até que começou a tocar o hino atemporal SÓ QUER VRAU. Neste mesmo instante ele precisava fazer uma chamada. Quando ele clicava num número N do teclado, acontecia algum tipo de bug e a palavra "vrau" era repetida N vezes.

Sua tarefa é, dada a entrada N, escreva como a música foi tocada.

 

Formato da entrada:

N

  • N - Número natural

1 <= N 

Formato da saída:

U

  • U - String

O único valor de saída é composto pela frase "Essas malandra, assanhadinha, que so quer " seguindo por N "vrau"

Gerado automaticamente.
