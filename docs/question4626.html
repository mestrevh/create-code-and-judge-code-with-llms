Descrição do Problema (problem.txt)

Título: Q7 - Computadores Defeituosos Topicos do problema: array; decisão; repetição; Tempo limite de execução: 1 segundo(s) Descrição:

O IFPB Campus Campina Grande é uma instituição que possui diversos laboratórios e, dentre eles, os de Programação que são numerados de 1 a 4. Esses laboratórios possuem computadores para que os alunos possam utilizar durante as aulas dos professores quando essas ocorrem no laboratório. Contudo, alguns computadores apresentam problemas e/ou defeitos e muitas vezes frustam os alunos. Por isso, você foi encarregado de criar um programa que vai receber o laboratório em específico e uma sequência de 1 e 0 que indicará se o computador funciona bem ou não, junto ao seu id e exibir quantos computadores estão disponíveis para uso e quais os ID's desses computadores em questão. Cada laboratório possui uma quantidade diferente de computadores, sendo:


Laboratório 1: 28 máquinas

Labortaório 2: 26 máquinas

Laboratório 3: 15 máquinas

Laboratório 4: 20 máquinas


dado o número do laboratório, leia x entradas (onde x é igual ao número de máquinas do laboratório em questão) onde cada máquina irá ter um indentificador de funcionamento dado por 0 indicando que funciona e 1 indicando que não funciona, e o id desse computador.

OBS:


1. Caso o identificador de funcionamento seja distinto de 1 ou 0, exiba a mensagem "Identificador inválido" e leia novamente até que seja um identificador de funcionamento válido. 

2. Caso seja dado o ID de uma máquina que já foi cadastrada, exiba a mensagem "ID invalido. Nao pode haver pc's iguais" e leia novamente até que seja um ID válido.

3. Caso haja 0 máquinas funcionando, exiba a mensagem "Sem computadores disponiveis nesse laboratorio."

Formato da entrada:

Um número i que representa o laboratório, podendo ser 1, 2, 3 ou 4.

x entradas onde x será a quantidade de máquinas disponíveis no laboratório. Cada leitura receberá o identificador de funcionamento (1 ou 0) e o ID da máquina.


Formato da saída:

Uma mensagem "Quantidade de pessoas que podem utilizar os computadores: y", onde y é o número de computadores disponíveis para uso.

uma mensagem "Lista de id's:" seguido de uma lista de números que serão os ID's das máquinas que funcionam. Caso não haja máquinas funcionando, exiba a mensagem "Sem computadores disponiveis nesse laboratório."

Gerado automaticamente.
