Descrição do Problema (problem.txt)

Título: Soma de grupos divisíveis por D Topicos do problema: programação dinâmica; Tempo limite de execução: 1 segundo(s) Descrição:

Dada uma lista de N números, você pode escolher quaisquer M deles. Desta forma você pode escolhê-los de C(N,M) maneiras. Onde C(N, M) é a combinação matemática sem repetições.Você precisa determinar quantos desses grupos escolhidos possuem uma soma que é divisível por D.

Formato da entrada:

A entrada consiste de no máximo 10 casos de testes.

Cada caso de teste é descrito como a seguir.

A primeira linha de cada conjunto contém 2 inteiros N (0<N<=200) e Q (0<Q<=10). N indica quantos números existem e Q é o total de consultas. As próximas N linhas contém um inteiro. As consultas devem ser respondidas com base nesses números.

As próximas Q linhas contém Q consultas. Cada consulta contém dois inteiros D (0 <D<=20) e M (0<M<=10) cujo significado está explicado na descrição desse problema.

A entrada termina quando N=0 e Q=0.

Formato da saída:

Para cada caso de teste, imprima o "SET X:" , onde X é número do caso de teste que começa em 1.

Depois, para cada consulta imprima: "QUERY K: G", onde K é o número da consulta e G é o número de grupos encontrados.

Gerado automaticamente.
