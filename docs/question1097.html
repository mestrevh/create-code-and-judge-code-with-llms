Descrição do Problema (problem.txt)

Título: Peso ideal. Topicos do problema: decisão; lógica matemática; Tempo limite de execução: 1 segundo(s) Descrição:

Faça um programa que receba a altura e o sexo de uma pessoa e calcule e mostre seu peso ideal, utilizando as seguintes equações (onde h corresponde a altura):

  • Homens (m): (72.7 x h) - 58
  • Mulheres (f): (62,1 x h) - 44,7

Caso o usuário informe um sexo diferente de "m" ou "f", emita uma mensagem de erro ("O sexo informado eh invalido.").

Caso o usuário informe uma altura 0 ou negativa, emita uma mensagem de erro ("Altura invalida.")


Formato da entrada:
  • Um caractere que representa o sexo de uma pessoa
    • "m" para sexo masculino
    • "f" para sexo feminino
  • A altura de uma pessoa em metros (ponto flutuante (double))
Formato da saída:

Um valor do tipo float que representa o peso ideal de pessoa com precisão de 2 casas decimais

Gerado automaticamente.
