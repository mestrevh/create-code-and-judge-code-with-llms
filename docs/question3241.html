Descrição do Problema (problem.txt)

Título: Flipping Game Topicos do problema: lógica matemática; programação dinâmica; Tempo limite de execução: 1 segundo(s) Descrição:

Iahub got bored, so he invented a game to be played on paper.

He writes n integers a 1, a 2, ..., a n. Each of those integers can be either 0 or 1. He's allowed to do exactly one move: he chooses two indices i and j (1 ≤ i ≤ j ≤ n) and flips all values a k for which their positions are in range [i, j] (that is i ≤ k ≤ j). Flip the value of x means to apply operation x = 1 - x.

The goal of the game is that after exactly one move to obtain the maximum number of ones. Write a program to solve the little game of Iahub.


Formato da entrada:

The first line of the input contains an integer n (1 ≤ n ≤ 5000). In the second line of the input there are n integers: a 1, a 2, ..., a n. It is guaranteed that each of those n values is either 0 or 1.

Formato da saída:

Print an integer — the maximal number of 1s that can be obtained after exactly one move.

Gerado automaticamente.
