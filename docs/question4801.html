Descrição do Problema (problem.txt)

Título: Explorando a Math.h Topicos do problema: decisão; lógica matemática; Tempo limite de execução: 1 segundo(s) Descrição:

A biblioteca math.h é uma das mais completas e úteis da linguagem C, oferecendo diversas funções matemáticas prontas para uso. Vamos explorá-la colocando a mão na massa!

Escreva um programa que receba um número inteiro N para indicar qual operação matemática será realizada sobre um número real A:

1 – Potência de 2
2 – Raiz quadrada
3 – Módulo
4 – Logaritmo na base 2

Formato da entrada:

Dois valores:

  • Um número inteiro N (1 ≤ N ≤ 4)
  • Um número real A (1 ≤ A ≤ 1000)
Formato da saída:

Exiba o resultado da operação escolhida com duas casas decimais.

Ex: 1.23

Gerado automaticamente.
