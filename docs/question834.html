Descrição do Problema (problem.txt)

Título: Porta Criptografada Topicos do problema: ad-hoc; Tempo limite de execução: 3 segundo(s) Descrição:

Os pais de Joãozinho colocaram os brinquedos dele em um quarto e colocaram uma porta que abre com uma senha. Então, os pais de Joãozinho lançaram o desafio para ele de descobrir o mistério da senha dessa porta. Portanto, Joãozinho chamou você para descobrir o algoritmo para quebrar a senha da porta e os pais dele deixaram as dicas, que foram os exemplos de entrada e saída. Faça um programa que lido 4 números, tenha como resultado a senha seguido de uma quebra de linha. 

Formato da entrada:

O programa irá ler 4 números :Ni > 1.

Sendo N>=1.

Formato da saída:

Uma única linha contendo a senha seguido de uma quebra de linha.

Gerado automaticamente.
