Descrição do Problema (problem.txt)

Título: Carteira Habilitacao Topicos do problema: decisão; Tempo limite de execução: 1 segundo(s) Descrição:

Faça um programa que verifique se o usuário pode tirar a carteira de habilitação.

Caso não possa, o programa deve informar quantos anos faltam para que ele possa tirá-la.

Formato da entrada:

Um número inteiro que representa a idade do usuário em anos.

Formato da saída:

O programa deve exibir uma das seguintes saídas quando:

  • O usuário está habilitado:
    • "Voce ja pode tirar habilitacao!"
    • "Voce tem direito a habilitacao ha x ano(s)"
  • Quando o usuário não está habilitado:
    • "Você precisa esperar mais x ano(s)!"

Gerado automaticamente.
