Descrição do Problema (problem.txt)

Título: Gerenciador de user/senha Topicos do problema: array; substrings; repetição; string; dicionário; Tempo limite de execução: 20 segundo(s) Descrição:
Você foi contratado por uma empresa de segurança para desenvolver um gerenciador de usuários e senhas super seguro, a aplicação deve gerar uma senha com base em um trecho de música escolhida pelo usuário. A senha deve conter 15 caracteres. O usuário pode gerar quantas senhas quiser com base nesse trecho de música, além de dizer se aceita aquela senha ou não. Uma vez que o usuário aceitou uma senha ele deve dizer qual é o username para aquela senha. Discutindo com a sua equipe, foram decididas as seguintes regras:

  1. A música inserida não pode ter quebras de linha 

  2. O usuário deve dizer quantas senhas quer gerar

  3. Toda letra "a", “E”, 'e', 'i', 'o', 'u'  deve ser substituída por @, %, !, @, #, $  respectivamente

  4. Todo o espaço deve ser removido 

  5. Apenas a primeira letra de cada palavra deve ser maiúscula 

  6. O usuário deve dizer um número entre 6 e o tamanho da música - 6 (numero mágico)

  7. A senha é gerada com base:

    1. 5 primeiras letras a partir do número de senhas geradas + 5 últimas letras + 5 próximas letras após o numero mágico pelo usuário.
      Dica: Se é a 5 senha que está sendo gerada, as 5 letras iniciais devem começar em 5.

  8. A senha deve ser invertida

  9. O usuário deve receber a senha e aceitar (S), ou pedir para gerar outra (N)

  10. Por fim o gerador de senhas deve mostrar um dicionário com os usuários e senhas
