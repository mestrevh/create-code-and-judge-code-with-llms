Descrição do Problema (problem.txt)

Título: Descriptografia Topicos do problema: recursão; string; Tempo limite de execução: 1 segundo(s) Descrição:

Dado uma mensagem S criptografada somente com { '*' , '-'  }. Você tem que descriptografar a mensagem, você percebeu o padrão e agora vai desenvolver um algoritmo para desvendar a mensagem.

Definições: 

S-1, indica a remoção do primeiro carácter da string.  

+, concatenação de string.

Pi, a posição do carácter na mensagem inicial. 

|S|, o tamanho da string restante

A função está descrita assim:

D ( S ) = 

Se |S| = 0 , D( S ) = "";

Se o primeiro elemento for '*', D( S ) = (Pi+D(S-1));

Se o primeiro elemento for '-', D( S ) = Pi+D(S-1).


Formato da entrada:

Uma string S somente com { '*' , '-' }

1 <= |S| <= 100

Formato da saída:

Uma linha com a mensagem descriptografada.

Gerado automaticamente.
