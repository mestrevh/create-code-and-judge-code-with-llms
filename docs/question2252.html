Descrição do Problema (problem.txt)

Título: Média Final com Prova de Reposição Topicos do problema: entrada formatada; substrings; repetição; ordenação; formatação; Tempo limite de execução: 1 segundo(s) Descrição:
Um professor avalia os alunos de sua turma através de 3 provas. Porém, ele decidiu que daria a opção dos alunos fazerem mais uma prova, ou seja, uma prova de reposição, para que pudessem substituir a sua pior nota, caso a de reposição seja maior.

Seu programa deve ler o nome completo do aluno e suas notas, sendo que podem ser fornecidas de 0 a 4 notas por aluno. Como são obrigatórias 3 (três) provas para calcular a média do aluno, quando forem fornecidas menos de 3 (três) notas, as demais serão consideradas como 0 (zero), sendo a média calculada em função de 3 notas.

Após calcular todas as médias dos alunos, o seu programa deve mostrar o nome e a média de cada aluno ordenados de forma crescente pela média. Se houver empate na nota, ordenar pelo nome
Formato da entrada:

A primeira linha contém um inteiro N que informa o número de alunos. Nas N linhas seguintes são fornecidos os dados de cada aluno, sendo um por linha. As informações dos alunos são Nome Completo e Notas, sendo que podem ser fornecidas de 0 a 4 notas. O Nome Completo e as Notas são separados por vírgula, sem espaços em branco entre os valores.

Formato da saída:

A saída será composta por N linhas contendo em cada uma o Nome Completo e a Média do aluno com 2 casas decimais separados entre si por um espaço em branco. Os alunos devem ser classificados pela ordem crescente das médias.
