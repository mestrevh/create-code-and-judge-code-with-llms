Título: Verificar Ordem em Lista Topicos do problema: array; decisão; repetição; array bidimensional; ordenação; Tempo limite de execução: 1 segundo(s) Descrição:

Verifique se os números lidos estão ordenados.  Você receberá uma lista como entrada e deverá imprimir se os elementos presentes estão ordenados ou não. 


Utilize a função eval() da linguagem Python para interpretar a entrada como lista da linguagem.

lista = eval(input())

Exemplo:

[1,4,5,7,10] - Ordenado


[1,0,1,4,10] - Nao ordenado


Formato da entrada:

[1,4,5,7,10]

Formato da saída:

Ordenado
