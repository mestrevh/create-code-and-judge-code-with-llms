Título: Q4 - Fora, repetidos! Topicos do problema: array; decisão; repetição; Tempo limite de execução: 1 segundo(s) Descrição:

Mateus é uma pessoa muito esquecida e perfeccionista, então ele te chamou para ajudá-lo com uma tarefa. Ele precisa guardar alguns números em duas listas separadas, mas como ele é esquecido, ele pode acabar falando números repetidos. Mateus quer uma lista final com todos os números que ele falou, mas sem os números repetidos. Ajude ele com isso e seja uma pessoa legal!


1 - Mateus te dirá quantos números serão alocados para a primeira lista, que é representado por n.

2 - Em seguida, serão dados n números que serão atribuídos para a primeira lista de números de Mateus.

3 - Em seguida, Mateus te dirá quantos números serão alocadods para a segunda lista, que é representado por n2.

4 - Faça o mesmo que fez para a lista 1, mas adiciionando esses novos números na segunda lista.

5 - por fim, mostre uma nova lista com todos os números que mateus forneceu na lista 1 e 2, mas sem números repetidos.

Formato da entrada:

Um número n que representa a quantidade de elementos da primeira lista.

n entradas que serão os números alocados para a primeira lista.

Um núimero n2 que representa a quantidade de elementos da segunda lista.

n2 entradas que serão os números alocados para a segunda lista.j

Formato da saída:

Uma lista com todos os números que Mateus forneceu sem que haja números repetidos.
