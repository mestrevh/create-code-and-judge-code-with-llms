Descrição do Problema (problem.txt)

Título: Inversões Topicos do problema: dividir e conquistar; Tempo limite de execução: 1 segundo(s) Descrição:

Dada uma sequência de elementos r, uma inversão é um par (i, j) tal que  i<j e r[i] > r[j]. Ou seja, uma inversão é nada mais do que um par de elementos que está fora da ordem na lista r.

O objetivo deste problema é projetar um algoritmo com complexidade O(n log n) para computar o número de inversões de uma sequência.

Formato da entrada:

A quantidade n de elementos da sequência, em uma linha.

Todos os n elementos, um em cada linha.

Formato da saída:

A quantidade de inversões na sequência de entrada.

Gerado automaticamente.
