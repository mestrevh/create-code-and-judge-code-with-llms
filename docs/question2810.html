Descrição do Problema (problem.txt)

Título: Cédulas Topicos do problema: lógica matemática; Tempo limite de execução: 1 segundo(s) Descrição:

Escreva um programa que leia um valor inteiro, referente a um valor monetário. A seguir, o programa deve calcular o menor número de notas possíveis (cédulas) no qual o valor pode ser decomposto. As notas consideradas são de 100, 50, 20, 10, 5, 2 e 1. A seguir mostre o valor lido e a relação de notas necessárias.

Formato da entrada:

O usuário deve digitar um valor inteiro N.

Formato da saída:

O programa deve exibir a quantidade mínima de notas de cada tipo necessárias, conforme o exemplo fornecido, imprimindo na tela os valores abaixo:

Digite o valor inteiro:

n nota(s) de R$ 100,00

n nota(s) de R$ 50,00

n nota(s) de R$ 20,00

n nota(s) de R$ 10,00

n nota(s) de R$ 5,00

n nota(s) de R$ 2,00

n moeda(s) de R$ 1,00

Gerado automaticamente.
