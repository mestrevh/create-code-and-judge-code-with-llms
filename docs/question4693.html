Descrição do Problema (problem.txt)

Título: Merge Sort Passo a Passo Topicos do problema: estrutura de dados; eof; array; alocação dinâmica; dividir e conquistar; recursão; ponteiro; ordenação; Tempo limite de execução: 3 segundo(s) Descrição:

Crie um programa que cumpre os seguintes requisitos:

  1. Recebe um array númerico como entrada, que pode ter qualquer tamanho.
  2. Ordena esse array usando um algoritmo Merge Sort.
  3. Exibe cada parte do array conforme ordenar essa parte.


Formato da entrada:

Um sequencia de números na mesma linha, separados apenas por um espaçamento simples.

Formato da saída:

Cada linha da saída deve exibir uma parte ordenada do array, a ordenação deve ser feita da esquerda para direita, e em caso de assimetria o lado esquerdo deve ficar com mais valores.

O print deve ser feito a cada operação de merge, e deve exibir apenas os elementos que foram ordenados nessa operação.
