Descrição do Problema (problem.txt)

Título: Os retângulos estão nos cercando Topicos do problema: ad-hoc; repetição; maratona; Tempo limite de execução: 1 segundo(s) Descrição:

Você vai receber uma lista limites de retângulos. Esses limites de retângulo são definidos por pontos em coordenadas inteiras. Os retângulos cercam quadrados no grid de coordenadas. Faça um programa que determine o total de quadrados do grid que são cercados por algum retângulo.


Formato da entrada:

A entrada terá entre 1 e 10 casos de teste. Cada caso de teste inicial com um número n, indicando o número retângulos. Em seguida, virão n linhas com 4 inteiros a, b, c e d, onde (a, b) é ponto inferior esquerdo e (c, d) é o ponto superior direito do retângulo. A entrada para quando n for 0.

0 < n <= 50
a < c, b < d
0 <= a, b, c e d <= 500

Formato da saída:

Para cada caso de teste, imprima o total de quadrados do grid que são cercados por algum retângulo.
