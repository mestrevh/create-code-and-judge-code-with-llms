Descrição do Problema (problem.txt)

Título: Selecionando os titulares do ataque Topicos do problema: decisão; repetição; função; Tempo limite de execução: 1 segundo(s) Descrição:

Faça uma função que escolhe o ataque titular da seleção brasileira baseado nas características de cada jogador. Cada característica será usada para compor um índice técnico que será usado para definir os titulares.

Seu programa deve receber:

  • O nome do jogador
  • Seu desempenho no treino (bom, mediano, ruim)
  • Quantos gols ele fez no último jogo
  • Qual o seu nível de cansaço (muito cansado, pouco cansado, descansado)
  • Seu nível de entrosamento com os companheiros (alto, médio ou baixo)

No final, o programa deve imprimir os índices técnicos dos três jogadores que devem ser os titulares do ataque do Brasil.

O índice técnico deve ser calculado da seguinte forma:

Um jogador não pode ter 2 características no nível mínimo seu índice será zero. Caso contrário, o índice será calculado da seguinte forma:

índice = desempenho_no_treino*2 + numero_de_gols*3.5 + cansaço*1.5 + entrosamento*2

O programa pára quando receber 5 jogadores.

Formato da entrada:
  • O nome do jogador
  • Seu desempenho no treino
    • 2 para bom
    • 1 para mediano
    • 0 para ruim
  • Quantos gols ele fez no último jogo
    • O mínimo é zero
  • Qual o seu nível de cansaço
    • 0 para muito cansado
    • 1 para pouco cansado
    • 2 para descansado
  • Seu nível de entrosamento com os companheiros
    • 2 para alto
    • 1 para médio
    • 0 para baixo
Formato da saída:

Os índices dos 3 jogadores que serão so titulares na escalação.
