Descrição do Problema (problem.txt)

Título: Invertor de frases Topicos do problema: pilha; Tempo limite de execução: 1 segundo(s) Descrição:

Faça um programa, utilizando pilha, que inverta a ordem das letras de uma string, deixando as palavras na mesma ordem incial.

Exemplo (Entrada -> Sainda): 

"Arthur Marsaro" -> "ruthrA orasraM"

Atenção: mantenha os espaços da frase no local correto.


Formato da entrada:

Uma string de até 50 characteres

Formato da saída:

A frase com as suas palavras invertidas

Gerado automaticamente.
