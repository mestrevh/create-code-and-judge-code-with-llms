Descrição do Problema (problem.txt)

Título: Q9 - Fora, repetidos! v2 Topicos do problema: array; repetição; Tempo limite de execução: 1 segundo(s) Descrição:

À essa altura, você já deve ter feito a "Q4 - Fora, repetidos!", certo? Senão, tudo bem! 


Sua missão é receber n números que serão alocados para uma primeira lista; n2 números alocados para a segunda lista e, por fim, mostrar:


1 - Uma lista apenas com os números que se repetiram, sem repetidos (por exemplo, se 6 se repetiu 3 vezes, essa lista de repetidos terá apenas um 6.)

2 - Uma lista com todos os números sem números repetidos

3 - Uma lista com todos os números, excetos os números repetidos que estão na lista de repetidos. (Por exemplo, se há 6 na lista com todos os números e há 6 na lista de repetidos, essa lista não deve conter o número 6.)

Formato da entrada:

Primeiramente, um número n que representa a quantidade de elementos da primeira lista.

Segundamente, serão lidos n números que serão alocados para a primeira lista.

Terceiramente, n2 números que representa a quantidade de elementos da segunda lista.

Quartamente, serão lidos n2 números que serão alocados para a segunda lista.

Formato da saída:

3 LIstas:

Uma apenas com os números que se repetiram

Uma com todos os números, incluindo os repetidos, mas sem repetí-los

Uma com todos os números, exceto os que estão na lista de repetidos.

Gerado automaticamente.
