Descrição do Problema (problem.txt)

Título: Acidez de uma solução Topicos do problema: decisão; Tempo limite de execução: 1 segundo(s) Descrição:

Escreva um programa que leia o valor do índice de acidez (pH) de uma solução e informe se ela é ácida, básica ou neutra, sendo que o pH deve estar entre 0 e 14. Caso o usuário digite um valor fora dessa faixa, deve ser mostrada um mensagem de erro.

A solução é ácida quando o pH é menor que 7

A solução é básica quando o pH é maior que 7

Caso contrário a solução é neutra


Formato da entrada:

O usuario deve digitar o valor do pH (entre 0.0 e 14.0)

Formato da saída:

O programa deve exibir se a solução é ácida, básica ou netra, imprimindo na tela os valores abaixo:


Digite o pH da solucao:

Solucao basica/acida/neutra / Valor do pH deve estar entre 0 e 14

Gerado automaticamente.
