Descrição do Problema (problem.txt)

Título: P1 - Média das idades Topicos do problema: decisão; repetição; lógica matemática; Tempo limite de execução: 1 segundo(s) Descrição:

Faça um programa que leia idades até que a idade digitada seja 0 (zero). No final, mostre a média das idades e a classificação dessa média de idades de acordo com a seguinte tabela:

de 0 a 25: jovem

de 26 a 60: adulta

acima de 60: idosa


OBS.: Mostre a média arredondada para casos de números quebrados.

Formato da entrada:

serão lidas n idades até que a idade lida seja 0.

Formato da saída:

a média das idades e a classifcação dessa média de idades.
