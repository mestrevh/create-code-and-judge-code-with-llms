Descrição do Problema (problem.txt)

Título: Remoção de strings Topicos do problema: programação dinâmica; Tempo limite de execução: 1 segundo(s) Descrição:

Dada uma string s de 2 caracteres 'a' e 'b'. Seja um grupo a maior substring de um mesmo caractere consecutivo.

Qualquer grupo g de tamanho s de tamanho de pelo menos 2 pode ser removido e uma nova string é construída concatenando o que resta do lado esquerdo com o que resta do lado direito de s.

Esse processo é repetido até a string se tornar vazia ou não existir mais grupos de tamanho pelo menos 2.

Por exemplo, a string s = babbbaaabb tem 5 grupos: b, a, bbb, aaa e bb. A string s pode ser transformada em uma string vazia removendo os grupos da seguinte sequência:

babbbaaabb ----- (removendo 'bbb') ----> baaaabb
baaaabb ----- (removendo 'aaaa') ----> bbb
bbb ----- (removendo 'bbb') ----> string vazia

Entretanto, dependendo da sequência de remoções, você não conseguirá gerar uma string vazia. Lembre-se que você só poderá remover grupos, de acordo com a definição de grupo acima. Por exemplo, se as remoções forem:

babbbaaabb ----- (removendo 'bb') ----> babbbaaa
babbbaaa ----- (removendo 'bbb') ----> baaaa
baaaa ----- (removendo 'aaaa') ----> b

No exemplo acima, 'b' não é um grupo, pois possui somente 1 elemento.

Dada uma string, escreva um programa para decidir se a string pode ser transformada em uma string fazia através de uma sequência de operações de remoções de grupos.




Formato da entrada:

Você receberá uma linha contendo o inteiro T indicando a quantidade de casos de testes.

Cada caso de teste consiste de uma única linha contendo N caracteres 'a' e 'b' (1<= N <=25).

Formato da saída:

Para cada caso de teste imprima uma única linha contendo 0 ou 1.

1 caso a string pode ser transformada em uma string vazia. 0 caso contrário.

Gerado automaticamente.
